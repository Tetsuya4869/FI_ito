1. 統計アンサンブル

📖 ミクロカノニカルアンサンブル（孤立系）

エネルギー $E$，粒子数 $N$，体積 $V$ が一定。状態数 $\Omega(E)$ に対して:

$$S = k_B\ln\Omega(E) \quad \text{(Boltzmann の公式)}$$

温度: $\frac{1}{T} = \left(\frac{\partial S}{\partial E}\right)_{N,V}$。全ての微視的状態は等確率（等重率の原理）。

📖 カノニカルアンサンブル（熱浴と接触）

温度 $T$（$\beta = 1/(k_BT)$），$N$，$V$ 一定。微視的状態 $s$ の確率:

$$P(s) = \frac{e^{-\beta E(s)}}{Z}, \quad Z = \sum_s e^{-\beta E(s)} \quad \text{(分配関数)}$$

Boltzmann 因子 $e^{-\beta E}$。Helmholtz 自由エネルギー: $F = -k_BT\ln Z$。

📖 大カノニカルアンサンブル（熱浴＋粒子浴）

$T$，化学ポテンシャル $\mu$，$V$ 一定。状態 $(N,s)$ の確率:

$$P(N,s) = \frac{e^{-\beta(E(N,s) - \mu N)}}{\mathcal{Z}}, \quad \mathcal{Z} = \sum_{N,s}e^{-\beta(E-\mu N)}$$

大分配関数 $\mathcal{Z}$，グランドポテンシャル: $\Omega = -k_BT\ln\mathcal{Z}$。

2. 分配関数と熱力学量

✓ 熱力学量の導出（カノニカル）
$$\langle E\rangle = -\frac{\partial\ln Z}{\partial\beta}, \quad F = -k_BT\ln Z$$ $$S = -\frac{\partial F}{\partial T} = k_B\left(\ln Z + \beta\langle E\rangle\right)$$ $$P = -\frac{\partial F}{\partial V}, \quad C_V = \frac{\partial\langle E\rangle}{\partial T} = k_B\beta^2\left(\langle E^2\rangle - \langle E\rangle^2\right)$$
📝 例 — 二準位系

エネルギー $0, \varepsilon$ の 2 状態。$Z = 1 + e^{-\beta\varepsilon}$。

$$\langle E\rangle = \frac{\varepsilon}{e^{\beta\varepsilon}+1}, \quad C = k_B\left(\frac{\varepsilon}{k_BT}\right)^2\frac{e^{\varepsilon/k_BT}}{(e^{\varepsilon/k_BT}+1)^2}$$

低温 ($T\to 0$): $\langle E\rangle\to 0$。高温 ($T\to\infty$): $\langle E\rangle\to\varepsilon/2$（等分配）。

✓ エネルギー等分配則

温度 $T$ の古典系で，Hamiltonian が $q_i$ または $p_i$ の二次形式 $a\xi^2$ として現れる各自由度に $k_BT/2$ のエネルギーが配分される。

3. 量子統計分布

✓ Bose-Einstein・Fermi-Dirac 分布

一粒子状態 $\varepsilon$ の平均占有数:

$$\langle n_\varepsilon\rangle = \frac{1}{e^{\beta(\varepsilon-\mu)} \mp 1} \quad \begin{cases}-: \text{Bose-Einstein (ボソン)}\\ +: \text{Fermi-Dirac (フェルミオン)}\end{cases}$$

古典極限 ($e^{\beta(\varepsilon-\mu)}\gg 1$): Boltzmann 分布 $\langle n\rangle \approx e^{-\beta(\varepsilon-\mu)}$。

4. 相転移と Ising モデル

📖 Ising モデル

格子点 $i$ にスピン $s_i = \pm 1$。Hamiltonian:

$$H = -J\sum_{\langle ij\rangle}s_i s_j - h\sum_i s_i$$

$J>0$: 強磁性（平行配列が安定），$h$: 外部磁場。

✓ 1次元 Ising モデルの厳密解

転送行列法（transfer matrix）により厳密に解ける:

$$Z = \lambda_+^N + \lambda_-^N, \quad \lambda_\pm = e^{\beta J}\cosh(\beta h) \pm \sqrt{e^{2\beta J}\sinh^2(\beta h) + e^{-2\beta J}}$$

1次元では $T>0$ で長距離秩序なし（相転移は $T=0$ のみ）。2次元では $T_c > 0$ で相転移（Onsager, 1944）。

📖 相転移の分類と臨界現象

1次相転移: 秩序パラメータが不連続，潜熱あり（例: 融解）。

2次相転移（連続）: 秩序パラメータが連続的に0に，相関長が発散 $\xi \sim |T-T_c|^{-\nu}$。

臨界指数（$d=3$ Ising）: $\nu\approx 0.63$，$\beta\approx 0.33$，$\gamma\approx 1.24$。

平均場理論: $M \sim (T_c-T)^{1/2}$（実際の $\beta=0.33$ とずれる）。

5. 揺動散逸定理

✓ 揺動散逸定理（Green-Kubo 関係）

平衡状態での自発的揺動と外場に対する応答（散逸）が結びつく:

$$\chi''(\omega) = \frac{\omega}{2k_BT}S(\omega)$$

$\chi''$: 応答関数の虚部（散逸），$S(\omega)$: 揺動のパワースペクトル。

拡散係数 $D$ と粘性 $\eta$ の関係（Einstein 関係）: $D = k_BT/(6\pi\eta r)$（Stokes-Einstein）。

📖 Johnson-Nyquist ノイズ

抵抗 $R$ の熱雑音: $\langle V^2\rangle = 4k_BTR\,\Delta f$（$\Delta f$: 帯域幅）。揺動散逸定理の電気的例。

6. 非平衡統計力学入門

📖 Boltzmann 方程式
$$\frac{\partial f}{\partial t} + \mathbf{v}\cdot\nabla_r f + \frac{\mathbf{F}}{m}\cdot\nabla_v f = \left(\frac{\partial f}{\partial t}\right)_{\rm coll}$$

$f(\mathbf{r},\mathbf{v},t)$: 分布関数。衝突項を適切に近似することで拡散・熱伝導・粘性などの輸送現象が導ける（Chapmann-Enskog 展開）。

✓ H 定理と不可逆性

Boltzmann の $H$ 関数 $H = \int f\ln f\,d^3v$ は時間とともに単調減少（$\dot H \le 0$）。これが熱力学第二法則（エントロピー増大）のミクロな根拠を与える。

✏️ 練習問題
  1. $N$ 個の調和振動子からなる系の分配関数を計算し，熱容量 $C_V$ を求めよ（Einstein モデル）。
  2. 2次元 Ising モデルの平均場近似を行い，臨界温度 $T_c$ を求めよ。
  3. 理想 Fermi 気体の Fermi エネルギー $E_F$ と，$T=0$ での全エネルギーを計算せよ。
  4. Fokker-Planck 方程式 $\partial_t P = \gamma\partial_x(xP) + D\partial_x^2 P$ の定常解を求めよ。
