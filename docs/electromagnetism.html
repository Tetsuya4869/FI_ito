1. Maxwell 方程式

📖 Maxwell 方程式（微分形）

電場 $\mathbf{E}$，磁束密度 $\mathbf{B}$，電荷密度 $\rho$，電流密度 $\mathbf{J}$，真空誘電率 $\varepsilon_0$，真空透磁率 $\mu_0$ として:

$$\nabla \cdot \mathbf{E} = \frac{\rho}{\varepsilon_0} \quad \text{(Gauss の法則)}$$ $$\nabla \cdot \mathbf{B} = 0 \quad \text{(磁気単極子の不存在)}$$ $$\nabla \times \mathbf{E} = -\frac{\partial \mathbf{B}}{\partial t} \quad \text{(Faraday の法則)}$$ $$\nabla \times \mathbf{B} = \mu_0 \mathbf{J} + \mu_0\varepsilon_0 \frac{\partial \mathbf{E}}{\partial t} \quad \text{(Ampère-Maxwell の法則)}$$
📖 Maxwell 方程式（積分形）
$$\oint_S \mathbf{E}\cdot d\mathbf{A} = \frac{Q_{\rm enc}}{\varepsilon_0}$$ $$\oint_S \mathbf{B}\cdot d\mathbf{A} = 0$$ $$\oint_C \mathbf{E}\cdot d\boldsymbol{\ell} = -\frac{d}{dt}\int_S \mathbf{B}\cdot d\mathbf{A}$$ $$\oint_C \mathbf{B}\cdot d\boldsymbol{\ell} = \mu_0 I_{\rm enc} + \mu_0\varepsilon_0 \frac{d}{dt}\int_S \mathbf{E}\cdot d\mathbf{A}$$
✓ 電荷保存則（連続の方程式）

Maxwell 方程式から自動的に導かれる:

$$\frac{\partial \rho}{\partial t} + \nabla \cdot \mathbf{J} = 0$$
📖 Lorentz 力
$$\mathbf{F} = q(\mathbf{E} + \mathbf{v}\times\mathbf{B})$$

Maxwell 方程式と組み合わせることで，電磁気学の完全な記述が得られる。

2. 電磁ポテンシャルとゲージ変換

📖 スカラーポテンシャルとベクトルポテンシャル

$\nabla\cdot\mathbf{B}=0$ より $\mathbf{B} = \nabla\times\mathbf{A}$（ベクトルポテンシャル $\mathbf{A}$）。

これを Faraday の法則に代入すると $\mathbf{E} = -\nabla\phi - \partial_t\mathbf{A}$（スカラーポテンシャル $\phi$）。

📖 ゲージ変換

任意の関数 $\Lambda(\mathbf{r},t)$ に対して

$$\mathbf{A} \to \mathbf{A} + \nabla\Lambda, \quad \phi \to \phi - \frac{\partial\Lambda}{\partial t}$$

は物理量 $\mathbf{E}, \mathbf{B}$ を変えない（ゲージ不変性）。

Coulomb ゲージ: $\nabla\cdot\mathbf{A} = 0$（静電気・非相対論的量子力学で有用）。

Lorenz ゲージ: $\nabla\cdot\mathbf{A} + \mu_0\varepsilon_0\partial_t\phi = 0$（相対論的共変性を持つ）。

✓ Lorenz ゲージでの波動方程式

Lorenz ゲージ下で Maxwell 方程式は波動方程式に帰着する:

$$\Box \mathbf{A} = \mu_0 \mathbf{J}, \quad \Box \phi = \frac{\rho}{\varepsilon_0}$$

ただし $\Box = \nabla^2 - \mu_0\varepsilon_0\partial_t^2$（d'Alembertian）。

3. 電磁波の伝播

✓ 真空中の電磁波

真空（$\rho=0, \mathbf{J}=0$）でのMaxwell方程式から:

$$\nabla^2\mathbf{E} = \mu_0\varepsilon_0 \frac{\partial^2\mathbf{E}}{\partial t^2}, \quad \nabla^2\mathbf{B} = \mu_0\varepsilon_0 \frac{\partial^2\mathbf{B}}{\partial t^2}$$

光速: $c = 1/\sqrt{\mu_0\varepsilon_0} \approx 3\times10^8\,\text{m/s}$。

平面波解: $\mathbf{E} = \mathbf{E}_0 e^{i(\mathbf{k}\cdot\mathbf{r} - \omega t)}$，$\mathbf{B} = \frac{1}{c}\hat k\times\mathbf{E}$。

$\mathbf{E}\perp\mathbf{B}\perp\hat k$（横波），$|\mathbf{B}| = |\mathbf{E}|/c$。

📖 エネルギーと Poynting ベクトル

電磁場のエネルギー密度:

$$u = \frac{1}{2}\left(\varepsilon_0 E^2 + \frac{1}{\mu_0}B^2\right)$$

Poynting ベクトル（エネルギー流束）:

$$\mathbf{S} = \frac{1}{\mu_0}\mathbf{E}\times\mathbf{B}$$

エネルギー保存（Poynting の定理）: $\partial_t u + \nabla\cdot\mathbf{S} = -\mathbf{J}\cdot\mathbf{E}$。

4. 境界値問題と多重極展開

📖 境界条件

誘電体・導体界面での境界条件（法線成分 $n$，接線成分 $t$）:

$$\varepsilon_1 E_{1n} - \varepsilon_2 E_{2n} = \sigma_f, \quad E_{1t} = E_{2t}$$ $$B_{1n} = B_{2n}, \quad \frac{1}{\mu_1}B_{1t} - \frac{1}{\mu_2}B_{2t} = K_f$$
📖 多重極展開

原点から遠い点 $\mathbf{r}$ でのポテンシャル ($r' \ll r$):

$$\phi(\mathbf{r}) = \frac{1}{4\pi\varepsilon_0}\left[\frac{Q}{r} + \frac{\mathbf{p}\cdot\hat r}{r^2} + \frac{1}{2}\sum_{ij}Q_{ij}\frac{\hat r_i \hat r_j}{r^3} + \cdots\right]$$

$Q = \int\rho\,dV$（全電荷），$\mathbf{p} = \int\mathbf{r}'\rho\,dV$（双極子モーメント），$Q_{ij} = \int(3r'_ir'_j - r'^2\delta_{ij})\rho\,dV$（四極子）。

5. 相対論的定式化

📖 4-ポテンシャルと電磁場テンソル

4-ポテンシャル: $A^\mu = (\phi/c, \mathbf{A})$。

電磁場テンソル: $F^{\mu\nu} = \partial^\mu A^\nu - \partial^\nu A^\mu$。成分:

$$F^{\mu\nu} = \begin{pmatrix} 0 & -E_x/c & -E_y/c & -E_z/c \\ E_x/c & 0 & -B_z & B_y \\ E_y/c & B_z & 0 & -B_x \\ E_z/c & -B_y & B_x & 0 \end{pmatrix}$$
✓ Maxwell 方程式の共変形

Maxwell 方程式は 2 本の共変方程式に凝縮される:

$$\partial_\mu F^{\mu\nu} = \mu_0 J^\nu \quad \text{(Gauss・Ampère)}$$ $$\partial_{[\mu}F_{\nu\rho]} = 0 \quad \text{(Faraday・磁気単極子なし)}$$

4-電流 $J^\nu = (c\rho, \mathbf{J})$。Lorentz 変換で $\mathbf{E}$ と $\mathbf{B}$ が混合する。

📖 電磁場の Lagrangian
$$\mathcal{L} = -\frac{1}{4\mu_0}F_{\mu\nu}F^{\mu\nu} - J^\mu A_\mu$$

この作用 $S = \int\mathcal{L}\,d^4x$ を変分すると Maxwell 方程式が得られる。
