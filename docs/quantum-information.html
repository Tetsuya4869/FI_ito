💡 前提知識

本ページは 量子力学（Hilbert 空間・bra-ket 記法・演算子形式）と 情報理論（Shannon エントロピー・相互情報量）の理解を前提とする。量子状態を「情報を担う資源」とみなし，その操作・測定・伝送・誤り訂正を体系的に扱う。

1. 量子状態と密度行列

📖 定義 — qubit（量子ビット）

2 準位系の純粋状態は 2 次元 Hilbert 空間 $\mathcal{H}=\mathbb{C}^2$ のベクトル:

$$|\psi\rangle = \alpha|0\rangle + \beta|1\rangle, \quad |\alpha|^2 + |\beta|^2 = 1, \quad \alpha,\beta\in\mathbb{C}$$

$\{|0\rangle,|1\rangle\}$ は計算基底。$n$ qubit 系は $\mathcal{H}^{\otimes n}=\mathbb{C}^{2^n}$ に住み，状態空間の次元が指数的に増大する点が古典ビットとの本質的差異である。

📖 定義 — 密度行列 $\rho$

状態のアンサンブル $\{p_i,|\psi_i\rangle\}$（確率 $p_i$ で $|\psi_i\rangle$）に対して密度行列を

$$\rho = \sum_i p_i\,|\psi_i\rangle\langle\psi_i|$$

で定める。$\rho$ は次の 3 条件で特徴づけられる: (i) Hermite $\rho=\rho^\dagger$，(ii) 半正定値 $\rho\ge 0$，(iii) トレース 1 $\mathrm{tr}\,\rho = 1$。観測量 $\hat A$ の期待値は $\langle\hat A\rangle = \mathrm{tr}(\rho\hat A)$。

✓ 定理 — 純粋状態と混合状態の判定

状態 $\rho$ が純粋状態（ある $|\psi\rangle$ で $\rho=|\psi\rangle\langle\psi|$）であることは次と同値:

$$\rho^2 = \rho \iff \mathrm{tr}(\rho^2) = 1$$

一般には純粋度 $\mathrm{tr}(\rho^2)\le 1$ であり，等号成立が純粋状態。$\mathrm{tr}(\rho^2)<1$ のとき混合状態とよぶ。最大混合状態は $\rho = I/d$（$d$ は次元）で $\mathrm{tr}(\rho^2)=1/d$。

📖 定義 — Bloch 球

1 qubit の密度行列は Pauli 行列 $\sigma_x,\sigma_y,\sigma_z$ を用いて

$$\rho = \frac{1}{2}\left(I + \mathbf{r}\cdot\boldsymbol{\sigma}\right) = \frac{1}{2}\begin{pmatrix} 1+r_z & r_x - ir_y \\ r_x + ir_y & 1-r_z\end{pmatrix}$$

と書ける。$\mathbf{r}=(r_x,r_y,r_z)\in\mathbb{R}^3$ を Bloch ベクトルとよび，$|\mathbf{r}|\le 1$。$|\mathbf{r}|=1$（球面上）が純粋状態，$|\mathbf{r}|<1$（球内部）が混合状態，原点 $\mathbf{r}=0$ が最大混合状態 $I/2$ に対応する。

📖 定義 — 部分トレース

合成系 $\mathcal{H}_A\otimes\mathcal{H}_B$ の状態 $\rho_{AB}$ から，部分系 $A$ の縮約密度行列を部分トレースで得る:

$$\rho_A = \mathrm{tr}_B(\rho_{AB}) = \sum_k (I_A\otimes\langle k|_B)\,\rho_{AB}\,(I_A\otimes|k\rangle_B)$$

$\{|k\rangle_B\}$ は $\mathcal{H}_B$ の正規直交基底。$\rho_A$ は $A$ のみを観測する者にとっての全情報を与える。たとえ $\rho_{AB}$ が純粋でも $\rho_A$ は一般に混合状態となる——これがもつれの兆候である。

📝 例 — Bell 状態の縮約

$|\Phi^+\rangle = \frac{1}{\sqrt2}(|00\rangle+|11\rangle)$ に対し $\rho_{AB}=|\Phi^+\rangle\langle\Phi^+|$ は純粋。だが部分トレースをとると

$$\rho_A = \mathrm{tr}_B(\rho_{AB}) = \frac{1}{2}\big(|0\rangle\langle0| + |1\rangle\langle1|\big) = \frac{I}{2}$$

となり，$A$ 単独では最大混合状態。$\mathrm{tr}(\rho_A^2)=1/2<1$ であり，全体の純粋さが部分系には現れない。

2. 量子もつれ

📖 定義 — テンソル積と積状態・もつれ状態

合成系の状態空間は $\mathcal{H}_{AB}=\mathcal{H}_A\otimes\mathcal{H}_B$。純粋状態 $|\psi\rangle_{AB}$ が積状態（分離可能）とは，ある $|\phi\rangle_A,|\chi\rangle_B$ で

$$|\psi\rangle_{AB} = |\phi\rangle_A\otimes|\chi\rangle_B$$

と書けることをいう。そう書けないとき，状態はもつれている（entangled）という。混合状態 $\rho_{AB}$ については，分離可能とは $\rho_{AB}=\sum_i p_i\,\rho_A^{(i)}\otimes\rho_B^{(i)}$ と表せることを指す。

📖 定義 — Bell 状態（最大もつれ状態）

2 qubit の完全正規直交基底をなす 4 つの最大もつれ状態:

$$|\Phi^\pm\rangle = \tfrac{1}{\sqrt2}(|00\rangle\pm|11\rangle), \quad |\Psi^\pm\rangle = \tfrac{1}{\sqrt2}(|01\rangle\pm|10\rangle)$$

これらは縮約密度行列がいずれも $I/2$ となり，もつれの度合いが最大である。量子テレポーテーションや超密度符号化の資源となる。

✓ 定理 — Schmidt 分解

任意の純粋状態 $|\psi\rangle_{AB}\in\mathcal{H}_A\otimes\mathcal{H}_B$ は，各系の適切な正規直交基底 $\{|i\rangle_A\},\{|i\rangle_B\}$ を選ぶことで

$$|\psi\rangle_{AB} = \sum_{i=1}^{r}\sqrt{\lambda_i}\,|i\rangle_A\otimes|i\rangle_B, \quad \lambda_i>0,\ \sum_i\lambda_i=1$$

と一意に表せる。$\{\lambda_i\}$ は Schmidt 係数，非ゼロ項の個数 $r$ を Schmidt 数という。$r=1$ が積状態，$r\ge2$ がもつれ状態。$\lambda_i$ は両縮約密度行列 $\rho_A,\rho_B$ の共通の固有値である。

✓ 定理 — もつれの判定（PPT 条件）

分離可能な状態 $\rho_{AB}$ は部分転置 $\rho_{AB}^{T_B}$ が半正定値（PPT: Positive Partial Transpose）である。よって $\rho_{AB}^{T_B}$ が負の固有値をもてば $\rho_{AB}$ はもつれている（Peres-Horodecki 判定）。$2\times2$ および $2\times3$ 系では PPT が分離可能性と同値となり，必要十分条件である。

📖 定義 — もつれエントロピー

純粋状態 $|\psi\rangle_{AB}$ のもつれの定量化として，縮約密度行列の von Neumann エントロピーを用いる:

$$E(|\psi\rangle_{AB}) = S(\rho_A) = -\sum_i \lambda_i\log\lambda_i$$

$\lambda_i$ は Schmidt 係数。積状態で $E=0$，最大もつれ状態（Bell 状態）で $E=\log 2 = 1$ bit（$\log_2$ のとき）。$S(\rho_A)=S(\rho_B)$ が常に成立する点に注意。

📝 例 — 一般 2 qubit 純粋状態のもつれ

$|\psi\rangle = \cos\theta\,|00\rangle + \sin\theta\,|11\rangle$ は既に Schmidt 形。Schmidt 係数は $\lambda_1=\cos^2\theta,\ \lambda_2=\sin^2\theta$。もつれエントロピーは

$$E = -\cos^2\theta\log\cos^2\theta - \sin^2\theta\log\sin^2\theta$$

$\theta=\pi/4$ で最大（Bell 状態），$\theta=0,\pi/2$ で $0$（積状態）となる。

3. 量子測定とチャネル

📖 定義 — 射影測定（von Neumann 測定）

観測量 $\hat A = \sum_m a_m P_m$（$P_m$ は固有値 $a_m$ の固有空間への射影，$\sum_m P_m = I$，$P_mP_n=\delta_{mn}P_m$）に対する測定。結果 $m$ を得る確率と測定後状態は

$$p(m) = \mathrm{tr}(P_m\rho), \quad \rho \to \frac{P_m\rho P_m}{p(m)}$$

純粋状態 $|\psi\rangle$ なら $p(m)=\langle\psi|P_m|\psi\rangle$（Born 則）。

📖 定義 — POVM（一般化測定）

正値演算子値測度 $\{E_m\}$ は次を満たす演算子の集合: $E_m\ge0$（半正定値）かつ $\sum_m E_m = I$。結果 $m$ の確率は

$$p(m) = \mathrm{tr}(E_m\rho)$$

射影測定は $E_m=P_m$（直交射影）の特別な場合。POVM は測定後状態を一意に定めないが，確率分布のみが必要な状況で最も一般的な測定の記述を与える。要素数は次元を超えてよい。

📖 定義 — 量子チャネル（CPTP 写像）

物理的に許される最も一般的な状態変換は，密度行列の集合上の線形写像 $\mathcal{E}$ で次を満たすもの:

(i) 完全正値性（CP）: 任意の補助系 $R$ に対し $\mathcal{E}\otimes\mathrm{id}_R$ が正値を保つ，(ii) トレース保存（TP）: $\mathrm{tr}(\mathcal{E}(\rho))=\mathrm{tr}(\rho)$。この CPTP 写像を量子チャネルとよぶ。単なる正値性ではなく完全正値性が必要なのは，もつれた補助系と合わせた変換も物理的でなければならないため。

✓ 定理 — Kraus 表現（演算子和表現）

$\mathcal{E}$ が量子チャネルであることは，ある演算子の組 $\{K_k\}$（Kraus 演算子）が存在して

$$\mathcal{E}(\rho) = \sum_k K_k\,\rho\,K_k^\dagger, \quad \sum_k K_k^\dagger K_k = I$$

と書けることと同値である。完全正値性は和の形から，トレース保存は完全性条件 $\sum_k K_k^\dagger K_k = I$ から従う。Kraus 表現はユニタリでない量子過程（エネルギー散逸や脱位相）も統一的に記述する。

📝 例 — 代表的ノイズモデル

ビット反転チャネル: $\mathcal{E}(\rho) = (1-p)\rho + p\,X\rho X$（$X=\sigma_x$）。

位相反転（位相緩和）チャネル: $\mathcal{E}(\rho) = (1-p)\rho + p\,Z\rho Z$（$Z=\sigma_z$，非対角成分を減衰）。

脱分極チャネル: $\mathcal{E}(\rho) = (1-p)\rho + \dfrac{p}{3}(X\rho X + Y\rho Y + Z\rho Z)$。確率 $p$ で状態を最大混合 $I/2$ 方向へ崩す。

振幅減衰チャネル: Kraus 演算子 $K_0=\begin{pmatrix}1&0\\0&\sqrt{1-\gamma}\end{pmatrix},\ K_1=\begin{pmatrix}0&\sqrt\gamma\\0&0\end{pmatrix}$ で励起状態 $|1\rangle$ から $|0\rangle$ へのエネルギー散逸（$T_1$ 緩和）を表す。

4. Bell 不等式と非局所性

📖 定義 — EPR パラドックスと局所実在論

Einstein-Podolsky-Rosen（1935）はもつれ状態の片方を測定すると遠方のもう片方が瞬時に確定することを指摘し，量子力学が不完全で「局所的な隠れた変数」が真の状態を定めると主張した。局所実在論は (i) 実在性（測定前に値が定まっている）と (ii) 局所性（遠方の操作は瞬時に影響しない）を仮定する。Bell はこの仮定が検証可能な不等式を生むことを示した。

✓ 定理 — CHSH 不等式

Alice は観測量 $A_0,A_1$（固有値 $\pm1$），Bob は $B_0,B_1$（固有値 $\pm1$）を測定する。CHSH 量を

$$S = \langle A_0 B_0\rangle + \langle A_0 B_1\rangle + \langle A_1 B_0\rangle - \langle A_1 B_1\rangle$$

と定める。任意の局所実在論的（局所隠れ変数）理論では

$$|S| \le 2$$

が成り立つ。これが破れれば局所実在論は棄却される。

✓ 定理 — Tsirelson 限界

量子力学では CHSH 量は古典限界 $2$ を超えうるが，無限には大きくなれず

$$|S| \le 2\sqrt{2} \approx 2.828$$

に抑えられる（Tsirelson 限界）。最大値 $2\sqrt2$ は Bell 状態 $|\Phi^+\rangle$ と適切な測定方向（相対角 $45^\circ$）で達成される。$2<|S|\le 2\sqrt2$ の領域が量子非局所性の領域である。

📝 例 — Bell 状態での CHSH の最大化

$|\Phi^+\rangle$ に対し Alice が角度 $0,\ \pi/2$，Bob が角度 $\pi/4,\ -\pi/4$ で各々 $\boldsymbol{\sigma}\cdot\hat n$ を測ると，各相関は $\langle A_iB_j\rangle = \pm\cos(\pi/4)=\pm1/\sqrt2$ となり

$$S = 4\times\frac{1}{\sqrt2} = 2\sqrt2$$

古典限界 $2$ を明確に超える。Aspect らの実験以降，抜け穴を塞いだ実験（2015）でこの破れが確証された。

💡 非局所性と因果律

Bell 不等式の破れは「瞬時の情報伝達」を意味しない。測定結果は局所的にはランダムで，2 者が結果を比較するには古典通信が要る。よって非局所相関は相対論的因果律（信号は光速以下）と矛盾しない（no-signaling 原理）。量子相関は局所的でも完全に古典的でもない，第三の性質である。

5. 量子プロトコル

✓ 定理 — 複製不可能定理（no-cloning theorem）

未知の量子状態を複製するユニタリ $U$，すなわち任意の $|\psi\rangle$ に対し $U(|\psi\rangle\otimes|0\rangle)=|\psi\rangle\otimes|\psi\rangle$ を満たすものは存在しない。

証明（概略）: 2 状態 $|\psi\rangle,|\phi\rangle$ に適用し内積をとると，ユニタリ性から $\langle\psi|\phi\rangle = \langle\psi|\phi\rangle^2$。よって $\langle\psi|\phi\rangle\in\{0,1\}$ となり，直交または同一の状態しか複製できず，任意状態の複製は不可能。□ これは量子暗号の安全性の根拠であり，かつ量子誤り訂正で単純な冗長化が使えない理由でもある。

📖 定義 — 量子テレポーテーション

Alice の未知状態 $|\psi\rangle=\alpha|0\rangle+\beta|1\rangle$ を，共有した Bell 状態 $|\Phi^+\rangle_{AB}$ と 2 ビットの古典通信のみで Bob に転送する。全系は

$$|\psi\rangle\otimes|\Phi^+\rangle = \tfrac12\sum_{i} |\beta_i\rangle_{A'A}\otimes (\sigma_i|\psi\rangle)_B$$

と Bell 基底 $\{|\beta_i\rangle\}$ で展開できる。Alice が自分の 2 qubit を Bell 基底測定し，結果 2 bit を Bob に送る。Bob は対応する Pauli 補正 $\sigma_i\in\{I,X,Z,XZ\}$ を施すと $|\psi\rangle$ を復元する。状態自体は送らず（複製不可能定理と整合），Alice 側の元状態は測定で破壊される。

📖 定義 — 超密度符号化（superdense coding）

共有 Bell 状態 $|\Phi^+\rangle$ を用い，Alice が 1 qubit を送るだけで 2 古典ビットを伝送する。Alice は 2 bit $(a,b)$ に応じて自分の qubit に Pauli 操作 $Z^a X^b$ を施し，これにより全体が 4 つの直交 Bell 状態のいずれかになる:

$$\{I,\,X,\,Z,\,ZX\}\otimes I\,|\Phi^+\rangle = \{|\Phi^+\rangle,\,|\Psi^+\rangle,\,|\Phi^-\rangle,\,|\Psi^-\rangle\}$$

Bob は 2 qubit を Bell 基底測定して $(a,b)$ を完全に読み取る。もつれを資源として通信容量を倍化する点でテレポーテーションと双対をなす。

6. 量子エントロピー

📖 定義 — von Neumann エントロピー

密度行列 $\rho$ の von Neumann エントロピーは Shannon エントロピーの量子版:

$$S(\rho) = -\mathrm{tr}(\rho\log\rho) = -\sum_i \lambda_i\log\lambda_i$$

$\{\lambda_i\}$ は $\rho$ の固有値。純粋状態で $S=0$，$d$ 次元最大混合状態 $I/d$ で最大値 $S=\log d$。基底は文脈に応じ $\log_2$（bit）または自然対数（nat）をとる。

✓ 定理 — von Neumann エントロピーの性質

劣加法性: $S(\rho_{AB}) \le S(\rho_A) + S(\rho_B)$（等号は積状態）。

三角不等式（Araki-Lieb）: $|S(\rho_A) - S(\rho_B)| \le S(\rho_{AB})$。

強劣加法性: $S(\rho_{ABC}) + S(\rho_B) \le S(\rho_{AB}) + S(\rho_{BC})$。

古典との重要な相違: 純粋もつれ状態では全体 $S(\rho_{AB})=0$ なのに部分系 $S(\rho_A)>0$ となりうる（条件付きエントロピーが負になる）。

📖 定義 — 量子相互情報

合成系 $\rho_{AB}$ における全相関（古典＋量子）の尺度:

$$I(A\!:\!B) = S(\rho_A) + S(\rho_B) - S(\rho_{AB}) \ge 0$$

劣加法性より非負。最大もつれ Bell 状態では $I(A\!:\!B)=2\log2$ となり，古典系の上限 $\log2$ の 2 倍に達する——量子相関の超越性を示す。

✓ 定理 — Holevo 限界

Alice が確率 $\{p_i\}$ で状態 $\{\rho_i\}$ を符号化し $\rho=\sum_i p_i\rho_i$ を Bob に送るとき，Bob が任意の測定で取り出せる古典情報（Bob の結果 $Y$ と Alice の入力 $X$ の相互情報量）は Holevo 量 $\chi$ で抑えられる:

$$I(X\!:\!Y) \le \chi = S(\rho) - \sum_i p_i\,S(\rho_i)$$

$\chi\le\log d$ より，$d$ 次元系（$\log d$ qubit）から取り出せる古典情報は高々 $\log d$ bit。すなわち 1 qubit には 1 bit 分の古典情報しか保持できない（重ね合わせの自由度は測定で完全には汲み取れない）。

📝 例 — 非直交状態の識別限界

$|0\rangle$ と $|+\rangle=\tfrac1{\sqrt2}(|0\rangle+|1\rangle)$ を各 $1/2$ で送る。これらは非直交（$\langle0|+\rangle=1/\sqrt2$）ゆえ完全識別は不可能。$\rho=\tfrac12(|0\rangle\langle0|+|+\rangle\langle+|)$ の固有値から $S(\rho)\approx0.60$ bit，各 $\rho_i$ は純粋で $S(\rho_i)=0$ なので $\chi\approx0.60$ bit $<1$ bit。非直交性が伝送可能な情報を制限する。

7. 量子誤り訂正

📖 定義 — 量子誤りの離散化

連続的なノイズも，1 qubit 誤りは Pauli 演算子 $\{I,X,Z,Y=iXZ\}$ の線形結合で表せる（$X$: ビット反転，$Z$: 位相反転，$Y$: 両方）。任意の Kraus 演算子は $K_k=\sum c_{k,P}\,P$ と展開でき，符号が $X,Z$ 誤りを訂正できれば射影測定により連続誤りも離散誤りへ「畳み込んで」訂正できる。これが量子誤り訂正を可能にする鍵である。

✓ 定理 — Shor の 9 量子ビット符号

1 論理 qubit を 9 物理 qubit に符号化し，任意の 1 qubit 誤りを訂正する最初の符号。位相反転符号（$|0\rangle\to|+\!+\!+\rangle$ 型）とビット反転符号（$|0\rangle\to|000\rangle$ 型）の連結:

$$|0_L\rangle = \tfrac1{2\sqrt2}(|000\rangle+|111\rangle)^{\otimes3}, \quad |1_L\rangle = \tfrac1{2\sqrt2}(|000\rangle-|111\rangle)^{\otimes3}$$

各 3 qubit ブロック内の多数決でビット反転を，ブロック間の符号でビット位相反転を訂正する。誤り症候群（syndrome）を測定するため論理状態を壊さずに誤り位置を特定できる。

📖 定義 — スタビライザー符号

$n$ qubit の Pauli 群の可換部分群 $\mathcal{S}$（$-I\notin\mathcal{S}$）を安定化群とし，符号空間を全生成元の $+1$ 固有空間

$$\mathcal{C} = \{\,|\psi\rangle : g|\psi\rangle = |\psi\rangle,\ \forall g\in\mathcal{S}\,\}$$

として定める。$n-k$ 個の独立生成元で $k$ 論理 qubit を符号化する。誤り $E$ は安定化生成元との反交換／交換のパターン（症候群）を生み，状態を測定せずに $E$ を同定できる。Shor 符号・Steane 符号・表面符号（surface code）すべてがこの枠組みに収まり，現代の量子誤り訂正の標準言語である。

✓ 定理 — しきい値定理（fault-tolerant threshold）

物理 qubit あたりのゲート・測定の誤り率 $p$ がある臨界値 $p_{\mathrm{th}}$ を下回るなら，符号を再帰的に連結することで論理誤り率を任意に小さくできる:

$$p_L \sim p_{\mathrm{th}}\left(\frac{p}{p_{\mathrm{th}}}\right)^{2^\ell} \xrightarrow{\ell\to\infty} 0 \quad (p$\ell$ は連結のレベル。表面符号では $p_{\mathrm{th}}\sim 1\%$ 程度と比較的高く実現的である。任意精度の計算が多項式的オーバーヘッドで可能になることを保証し，スケーラブルな量子計算の理論的基盤を与える。
