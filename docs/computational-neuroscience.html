💡 前提知識

本ページは微分方程式と力学系の言葉でニューロンの動態を記述する。常微分方程式（膜電位の時間発展）、力学系（固定点・分岐・アトラクター）、機械学習の数理（学習則・最適化）を前提とする。

1. ニューロンの生物物理

📖 定義 — 膜電位と膜の等価回路

ニューロンの細胞膜はキャパシタとイオンチャネル（抵抗）の並列回路としてモデル化される。膜電位 $V$ は膜内外の電位差で，静止状態では約 $-65\,\mathrm{mV}$。膜容量 $C_m$ を用いて電流保存則より

$$C_m \frac{dV}{dt} = -\sum_i I_i + I_{\rm ext}$$

$I_i$ は各イオン種を通じた電流，$I_{\rm ext}$ は外部入力電流。各イオン電流は $I_i = g_i (V - E_i)$（$g_i$: コンダクタンス，$E_i$: 平衡電位）で与えられる。

📖 定義 — Nernst の平衡電位

イオン $X$ の平衡電位（逆転電位）は熱力学的平衡から Nernst の式で決まる：

$$E_X = \frac{RT}{zF}\ln\frac{[X]_{\rm out}}{[X]_{\rm in}}$$

$R$: 気体定数，$T$: 絶対温度，$z$: 価数，$F$: Faraday 定数。$\mathrm{Na^+}$ は正（$\approx +50\,\mathrm{mV}$），$\mathrm{K^+}$ は負（$\approx -77\,\mathrm{mV}$）。

✓ 定理 — Hodgkin-Huxley モデル

イカの巨大軸索の活動電位を記述する4変数の連立 ODE。電位依存性の $\mathrm{Na^+}$ チャネル（活性化 $m$，不活性化 $h$）と $\mathrm{K^+}$ チャネル（活性化 $n$）を含む：

$$C_m\frac{dV}{dt} = -\bar g_{\rm Na} m^3 h (V - E_{\rm Na}) - \bar g_{\rm K} n^4 (V - E_{\rm K}) - g_L(V - E_L) + I_{\rm ext}$$

ゲート変数 $x \in \{m, h, n\}$ は一次の緩和方程式に従う：

$$\frac{dx}{dt} = \alpha_x(V)(1 - x) - \beta_x(V)\,x = \frac{x_\infty(V) - x}{\tau_x(V)}$$

$\alpha_x, \beta_x$ は電位依存の開閉率，$x_\infty = \alpha_x/(\alpha_x+\beta_x)$ は定常開確率。

📝 例 — 活動電位（スパイク）の生成

脱分極が閾値（$\approx -55\,\mathrm{mV}$）を超えると，$\mathrm{Na^+}$ チャネルが急速に開いて $V$ が立ち上がる（脱分極相）。続いて $\mathrm{Na^+}$ チャネルが不活性化（$h\to 0$）し，遅い $\mathrm{K^+}$ チャネルが開いて再分極する。この正帰還→負帰還の連鎖が 全か無か のスパイクを生む。再分極後に過分極（アンダーシュート）が起こり，不応期 となる。

💡 時間スケールの分離

$m$ の時定数 $\tau_m$ は $h, n$ より格段に小さい。この高速・低速変数の分離が力学系的な解析（後述の簡約モデル）を可能にする。準静的近似 $m \approx m_\infty(V)$ により変数を消去できる。

2. 簡約ニューロンモデル

📖 定義 — Leaky Integrate-and-Fire（LIF）モデル

スパイク波形の詳細を捨象し，膜電位の積分と閾値発火だけを残した最小モデル：

$$\tau_m \frac{dV}{dt} = -(V - V_{\rm rest}) + R\,I(t)$$

$V$ が閾値 $V_{\rm th}$ に達した瞬間にスパイクを発し，$V$ を $V_{\rm reset}$ にリセットする（必要なら不応期 $\tau_{\rm ref}$ を挿入）。$\tau_m = R C_m$ は膜時定数。

✓ 定理 — LIF の f-I 曲線

定常入力電流 $I$ に対する発火率（周波数）は閉形式で求まる。$RI > V_{\rm th}-V_{\rm rest}$ のとき，不応期込みの発火間隔から

$$f(I) = \left[\tau_{\rm ref} + \tau_m \ln\frac{RI - (V_{\rm reset}-V_{\rm rest})}{RI - (V_{\rm th}-V_{\rm rest})}\right]^{-1}$$

閾値直上で発火率がゼロから連続的に立ち上がる（Type I 的挙動）。

📖 定義 — FitzHugh-Nagumo モデル

HH モデルを2変数（速い電位 $v$，遅い回復変数 $w$）に簡約した興奮性系：

$$\dot v = v - \frac{v^3}{3} - w + I, \qquad \dot w = \varepsilon (v + a - b w)$$

$0 < \varepsilon \ll 1$ により時間スケールが分離する。相平面上で $v$-ナルクライン（3次曲線）と $w$-ナルクライン（直線）の交点が固定点を与え，Hopf 分岐を経てリミットサイクル（周期発火）が現れる。力学系の典型的応用例。

📝 例 — 発火率モデル（rate model）

個々のスパイクを平均化し，ニューロン集団の発火率 $r(t)$ の動態を扱う粗視化モデル：

$$\tau \frac{dr}{dt} = -r + \phi\!\left(\sum_j w_j r_j + I_{\rm ext}\right)$$

$\phi(\cdot)$ は入出力関数（シグモイドや ReLU 型のf-I曲線），$w_j$ はシナプス結合強度。ネットワークレベルの解析に便利で，深層学習の人工ニューロンと直接対応する。

3. シナプスと可塑性

📖 定義 — シナプス伝達とコンダクタンスモデル

プレシナプスのスパイク到来によりポストシナプスのコンダクタンスが瞬間的に増え，指数的に減衰する：

$$I_{\rm syn}(t) = g_{\rm syn}(t)\,(V - E_{\rm syn}), \qquad g_{\rm syn}(t) = \bar g\sum_{t_f} e^{-(t - t_f)/\tau_s}\,\Theta(t - t_f)$$

$E_{\rm syn}$ が静止電位より高ければ興奮性（AMPA・NMDA），低ければ抑制性（$\mathrm{GABA}$）。$t_f$ はスパイク到来時刻，$\Theta$ はステップ関数。

📖 定義 — Hebb 則

「同時に発火する細胞は結合を強める」という基本原理。発火率 $r_i$（ポスト），$r_j$（プレ）に対し

$$\frac{dw_{ij}}{dt} = \eta\, r_i\, r_j$$

$\eta$ は学習率。素朴な Hebb 則は無制限に増大するため，正規化（Oja 則 $\dot w = \eta\,r_i(r_j - r_i w)$）や恒常性機構で安定化する。Oja 則は重みベクトルを主成分方向へ収束させる。

✓ 定理 — STDP（スパイクタイミング依存可塑性）

結合変化はプレ・ポストのスパイク時刻差 $\Delta t = t_{\rm post} - t_{\rm pre}$ に依存する。典型的な窓関数は

$$\Delta w = \begin{cases} A_+\, e^{-\Delta t/\tau_+} & (\Delta t > 0,\ \text{LTP}) \\[4pt] -A_-\, e^{\Delta t/\tau_-} & (\Delta t < 0,\ \text{LTD}) \end{cases}$$

プレがポストに先行すれば（因果的，$\Delta t>0$）結合は増強（LTP），逆順なら減弱（LTD）。Hebb 則に因果的な時間構造を与えたものと解釈できる。

💡 LTP と LTD の分子機構

NMDA 受容体は脱分極とグルタミン酸の同時到来でのみ $\mathrm{Ca^{2+}}$ を通す「同時検出器」として働く。大きく速い $\mathrm{Ca^{2+}}$ 流入は長期増強（LTP）を，小さく持続的な流入は長期抑圧（LTD）を誘導する。この双方向性が STDP の生物物理的基盤を与える。

4. ネットワークダイナミクス

📖 定義 — リカレントネットワーク

ニューロン同士が相互結合した系。発火率変数 $\mathbf{r}$ と結合行列 $W$ に対し

$$\tau \frac{d\mathbf{r}}{dt} = -\mathbf{r} + \phi(W\mathbf{r} + \mathbf{I})$$

$W$ の固有値スペクトルが安定性を決める。フィードバックが信号を保持・増幅し，短期記憶や持続的活動（working memory）の基盤となる。

✓ 定理 — Hopfield モデルとエネルギー関数

対称結合 $w_{ij} = w_{ji}$，$w_{ii}=0$ をもつ離散ニューロン $s_i \in \{-1, +1\}$ の非同期更新 $s_i \leftarrow \mathrm{sgn}(\sum_j w_{ij} s_j)$ を考える。このとき

$$E(\mathbf{s}) = -\frac{1}{2}\sum_{i,j} w_{ij}\, s_i s_j$$

は更新ごとに単調非増加であり，力学は $E$ の局所最小（アトラクター）へ必ず収束する。$E$ は Lyapunov 関数の役割を果たす。

📝 例 — 連想記憶

$P$ 個の記憶パターン $\boldsymbol{\xi}^{\mu}$ を Hebb 則で埋め込む：

$$w_{ij} = \frac{1}{N}\sum_{\mu=1}^{P} \xi_i^{\mu}\,\xi_j^{\mu}$$

各パターンがエネルギー地形の谷（アトラクター）となり，ノイズや部分入力から完全なパターンを想起できる（パターン補完）。記憶容量は $P_{\max} \approx 0.138\,N$ で，これを超えると干渉によりスピングラス状態に崩壊する。

5. 神経符号化

📖 定義 — 発火率符号と時間符号

発火率符号: 時間窓内のスパイク数（平均発火率）が情報を担う。時間符号: 個々のスパイクの精密なタイミングや相対位相が情報を担う。同じスパイク列から，平均化の時間スケールによって異なる情報を読み出せる。

✓ 定理 — ポアソン過程としてのスパイク列

不規則発火を瞬時発火率 $r(t)$ をもつ非斉次ポアソン過程で近似する。区間 $[0,T]$ に $n$ 個のスパイクが生じる確率は

$$P(n) = \frac{(\bar r T)^n}{n!}\, e^{-\bar r T}, \qquad \bar r = \frac{1}{T}\int_0^T r(t)\,dt$$

このときスパイク数の分散が平均に等しく，変動係数 $\mathrm{CV}=1$ となる（ポアソン的不規則性の指標）。

📝 例 — 情報理論的解析

刺激 $S$ と神経応答 $R$（発火率やスパイクパターン）の間の相互情報量で符号化効率を定量化する：

$$I(S;R) = \sum_{s, r} P(s, r)\,\log_2\frac{P(s, r)}{P(s)\,P(r)} = H(R) - H(R\mid S)$$

$H(R)$ は応答の総エントロピー（表現容量），$H(R\mid S)$ はノイズエントロピー。詳細は情報理論を参照。スパイク列のエントロピーから単一ニューロンの情報伝達率（bits/spike）が見積もられる。

💡 集団符号化と次元削減

個々のニューロンは粗く曖昧でも，多数のニューロンの集団活動として刺激を精密に表現できる（population coding）。神経状態は低次元の多様体上にしばしば拘束され，PCA や多様体学習でその構造を抽出できる。

6. 学習理論

📖 定義 — 教師あり・教師なし・強化学習

教師あり学習: 目標出力との誤差を最小化（小脳の運動学習など）。教師なし学習: 入力の統計構造を抽出（Hebb 則・主成分・スパース符号化）。強化学習: スカラー報酬を最大化する行動方策を学習（大脳基底核）。

✓ 定理 — 報酬予測誤差と TD 学習

強化学習における価値関数 $V(s)$ は時間差（TD）誤差で更新される：

$$\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t), \qquad V(s_t) \leftarrow V(s_t) + \alpha\,\delta_t$$

$\gamma$ は割引率，$\alpha$ は学習率。中脳ドーパミンニューロンの位相性発火がこの $\delta_t$（報酬予測誤差）を符号化することが実験的に示されており，脳における誤差信号の代表例である。

📝 例 — 誤差逆伝播と生物学的妥当性

深層ネットワークの誤差逆伝播法は重み行列の転置 $W^\top$ を逆方向に使うが，実際のシナプスは一方向であり生物学的に非現実的（weight transport problem）。これに対し，ランダムなフィードバック結合で誤差を伝える feedback alignment や，局所的な活動差で勾配を近似する predictive coding 等が脳に近い代替学習則として研究されている。

7. 応用と人工知能との関係

📖 定義 — ニューロモルフィック計算

スパイクを基本単位とするイベント駆動のハードウェア（SpiNNaker，Loihi，TrueNorth など）。アナログ回路で膜方程式を直接実装し，イベントがあるときだけ計算するため，超低消費電力での並列処理を実現する。スパイキングニューラルネット（SNN）がそのソフトウェア対応物。

✏️ 練習問題
  1. LIF モデル $\tau_m \dot V = -(V-V_{\rm rest}) + RI$ について，定常電流 $I$ 下で $V$ が $V_{\rm th}$ に達するまでの時間を解析的に求め，f-I 曲線（不応期なし）を導け。
  2. FitzHugh-Nagumo モデルの固定点を $I=0$ で求め，Jacobi 行列の固有値からその安定性を分類せよ。$\varepsilon$ の役割を述べよ。
  3. STDP 窓関数 $\Delta w(\Delta t)$ の下で，プレ・ポストがともにポアソン的に独立発火するとき，重みの平均的ドリフト $\langle \dot w\rangle$ を $A_\pm, \tau_\pm$ と発火率で表せ。LTP/LTD のバランス条件を求めよ。
  4. 2パターン（直交 $\boldsymbol\xi^1\cdot\boldsymbol\xi^2=0$）を埋め込んだ Hopfield ネットで，各記憶パターンがエネルギー $E$ の局所最小であり安定であることを示せ。
  5. 瞬時発火率 $r(t)=r_0(1+\cos\omega t)$ をもつ非斉次ポアソン過程について，時間窓 $T$ あたりのスパイク数の平均と分散を求め，変動係数 CV を評価せよ。
