1. フーリエ級数と直交系

📖 定義 — 三角級数とフーリエ係数

周期 $2\pi$ の関数 $f \in L^1(\mathbb{T})$ に対し，フーリエ係数を

$$\hat f(n) = \frac{1}{2\pi}\int_{-\pi}^{\pi} f(x)\,e^{-inx}\,dx \quad (n \in \mathbb{Z})$$

と定め，形式的なフーリエ級数を $\displaystyle f(x) \sim \sum_{n\in\mathbb{Z}} \hat f(n)\,e^{inx}$ とする。

✓ 定理 — 直交性と完全性

系 $\{e^{inx}\}_{n\in\mathbb{Z}}$ は $L^2(\mathbb{T})$ の完全正規直交系をなす：

$$\frac{1}{2\pi}\int_{-\pi}^{\pi} e^{inx}\,\overline{e^{imx}}\,dx = \delta_{nm}$$

したがって任意の $f\in L^2(\mathbb{T})$ に対し部分和 $S_N f = \sum_{|n|\le N}\hat f(n)e^{inx}$ は $L^2$ ノルムで $f$ に収束し，Parseval の等式 $\|f\|_2^2 = \sum_n |\hat f(n)|^2$ が成り立つ。

📝 例 — 点収束と Dirichlet 核

部分和は Dirichlet 核 $D_N$ とのたたみ込みで表される：

$$S_N f(x) = (f * D_N)(x), \qquad D_N(t) = \sum_{|n|\le N} e^{int} = \frac{\sin\bigl((N+\tfrac12)t\bigr)}{\sin(t/2)}$$

$\|D_N\|_{L^1} \sim \frac{4}{\pi^2}\log N \to \infty$（Lebesgue 定数の発散）ゆえ，連続関数でも点収束が保証されない。Fejér 核（Cesàro 平均）を用いると一様収束が回復する。

💡 収束の階層

$L^2$ 収束（常に成立）・点収束（Dini 条件などが必要）・一様収束（滑らかさが必要）は別物である。Carleson の定理により，$f\in L^2(\mathbb{T})$ のフーリエ級数はほとんど至る所で $f$ に収束することが知られる（深い結果）。

2. フーリエ変換

📖 定義 — $L^1$ 上のフーリエ変換

$f \in L^1(\mathbb{R}^n)$ に対し，フーリエ変換を

$$\hat f(\xi) = \int_{\mathbb{R}^n} f(x)\,e^{-2\pi i x\cdot\xi}\,dx$$

で定義する。$\hat f$ は有界連続で $\|\hat f\|_\infty \le \|f\|_1$，さらに Riemann-Lebesgue の補題より $|\xi|\to\infty$ で $\hat f(\xi)\to 0$。

✓ 定理 — 反転公式

$f\in L^1(\mathbb{R}^n)$ かつ $\hat f\in L^1(\mathbb{R}^n)$ ならば，ほとんど至る所で

$$f(x) = \int_{\mathbb{R}^n} \hat f(\xi)\,e^{2\pi i x\cdot\xi}\,d\xi$$

が成り立つ。特に Schwartz 空間 $\mathcal{S}(\mathbb{R}^n)$ 上でフーリエ変換は全単射であり，微分と乗法を交換する：$\widehat{\partial_j f}(\xi) = 2\pi i\,\xi_j\,\hat f(\xi)$。

✓ 定理 — Plancherel の定理と $L^2$ 理論

$\mathcal{S}(\mathbb{R}^n)$ 上で $\|\hat f\|_2 = \|f\|_2$ が成り立ち，フーリエ変換は $L^2(\mathbb{R}^n)$ 上のユニタリ作用素へ一意に拡張される。より一般に Parseval の等式

$$\int_{\mathbb{R}^n} f(x)\,\overline{g(x)}\,dx = \int_{\mathbb{R}^n} \hat f(\xi)\,\overline{\hat g(\xi)}\,d\xi$$

が任意の $f,g\in L^2$ に対して成立する。

📝 例 — Gauss 関数と Hausdorff-Young

Gauss 関数はフーリエ変換の固有関数：$f(x)=e^{-\pi|x|^2}$ に対し $\hat f(\xi)=e^{-\pi|\xi|^2}$。

補間により $1\le p\le 2$，$\frac1p+\frac1{p'}=1$ のとき Hausdorff-Young 不等式 $\|\hat f\|_{p'} \le \|f\|_p$ が成り立つ。

3. たたみ込みと近似恒等

📖 定義 — たたみ込み

$f,g$ のたたみ込みを

$$(f * g)(x) = \int_{\mathbb{R}^n} f(x-y)\,g(y)\,dy$$

で定義する。フーリエ変換は積を対角化する：$\widehat{f*g} = \hat f\cdot\hat g$，$\widehat{fg} = \hat f * \hat g$。

✓ 定理 — Young の不等式

$1\le p,q,r\le\infty$ が $\dfrac1p+\dfrac1q = 1+\dfrac1r$ を満たすとき，

$$\|f * g\|_r \le \|f\|_p\,\|g\|_q$$

が成り立つ。特に $g\in L^1$ ならば $\|f*g\|_p \le \|f\|_p\|g\|_1$ となり，たたみ込みは $L^p$ 上の有界作用素を与える。

📖 定義 — 近似恒等（軟化子）

$\phi\in L^1(\mathbb{R}^n)$，$\int\phi = 1$ とし，$\phi_t(x) = t^{-n}\phi(x/t)$ とおく。族 $\{\phi_t\}_{t>0}$ を近似恒等という。

✓ 定理 — 近似恒等による収束

$1\le p<\infty$ かつ $f\in L^p(\mathbb{R}^n)$ のとき，$t\to 0^+$ で

$$\|f * \phi_t - f\|_p \to 0$$

が成り立つ。これにより $C_c^\infty(\mathbb{R}^n)$ が $L^p$ で稠密であること，および軟化による滑らかな近似が正当化される。

💡 総和核との関係

フーリエ級数の Fejér 核，フーリエ変換の Gauss 核・Poisson 核はいずれも近似恒等の具体例であり，発散する級数・積分の「総和法」を与える。

4. 超関数

📖 定義 — テスト関数と超関数

テスト関数の空間 $\mathcal{D}(\Omega) = C_c^\infty(\Omega)$ 上の連続線形汎関数を超関数（distribution）といい，その全体を $\mathcal{D}'(\Omega)$ と書く。局所可積分関数 $f$ は $\langle T_f, \varphi\rangle = \int f\varphi$ により超関数を定める。

📝 例 — Dirac のデルタと弱微分

$\langle \delta, \varphi\rangle = \varphi(0)$ で定まる $\delta$ は超関数であるが，どの関数からも来ない。超関数の微分は $\langle T', \varphi\rangle = -\langle T, \varphi'\rangle$ で定義され，Heaviside 関数 $H$ に対し $H' = \delta$ が成り立つ。

📖 定義 — テンパード超関数 $\mathcal{S}'$

Schwartz 空間 $\mathcal{S}(\mathbb{R}^n)$（急減少 $C^\infty$ 関数）上の連続線形汎関数の全体をテンパード超関数の空間 $\mathcal{S}'(\mathbb{R}^n)$ という。多項式増大の関数や $\delta$ はこれに属する。

✓ 定理 — フーリエ変換の拡張

フーリエ変換は $\mathcal{S}(\mathbb{R}^n)$ 上の同相写像であるから，双対性 $\langle \hat T, \varphi\rangle = \langle T, \hat\varphi\rangle$ により $\mathcal{S}'(\mathbb{R}^n)$ 上へ一意に拡張される。例えば $\hat\delta = 1$，$\hat 1 = \delta$，$\widehat{e^{2\pi i a\cdot x}} = \delta_a$ が得られ，偏微分方程式の基本解の解析に用いられる。

5. Hardy-Littlewood 極大関数

📖 定義 — 極大関数

$f\in L^1_{\rm loc}(\mathbb{R}^n)$ に対し，Hardy-Littlewood 極大関数を

$$Mf(x) = \sup_{r>0} \frac{1}{|B(x,r)|}\int_{B(x,r)} |f(y)|\,dy$$

✓ 定理 — Lebesgue 微分定理

極大不等式の帰結として，$f\in L^1_{\rm loc}(\mathbb{R}^n)$ ならばほとんど至る所の $x$ で

$$\lim_{r\to 0^+} \frac{1}{|B(x,r)|}\int_{B(x,r)} |f(y)-f(x)|\,dy = 0$$

💡 弱型不等式の役割

強型不等式が $p=1$ で成り立たない（$Mf$ が $L^1$ でない）ため，弱型 $(1,1)$ 不等式が本質的である。弱型評価と稠密な関数族での収束を組み合わせて a.e. 収束を導くのが標準的な戦略。

6. 特異積分作用素

📖 定義 — Hilbert 変換

直線上の Hilbert 変換は主値積分

$$Hf(x) = \frac{1}{\pi}\,\mathrm{p.v.}\!\int_{\mathbb{R}} \frac{f(y)}{x-y}\,dy$$

で定義される。フーリエ側では乗子 $\widehat{Hf}(\xi) = -i\,\mathrm{sgn}(\xi)\,\hat f(\xi)$ を持ち，$L^2$ 上のユニタリに近い等長作用素となる。

📖 定義 — Calderón-Zygmund 核

核 $K(x)$ が (i) 大きさ条件 $|K(x)|\le A|x|^{-n}$，(ii) 平滑条件（Hörmander 条件）$\int_{|x|>2|y|}|K(x-y)-K(x)|\,dx \le B$ を満たし，対応する作用素が $L^2$ 有界であるとき，$Tf = \mathrm{p.v.}\,K * f$ をCalderón-Zygmund 作用素という。

7. 応用

📝 例 — 偏微分方程式

フーリエ変換は定数係数線形 PDE を代数方程式に変える。熱方程式 $\partial_t u = \Delta u$ は $\hat u(t,\xi) = e^{-4\pi^2|\xi|^2 t}\hat u_0(\xi)$ となり，解は熱核 Gauss 関数とのたたみ込みで表される。Sobolev 空間 $H^s = \{u : (1+|\xi|^2)^{s/2}\hat u \in L^2\}$ も周波数側で定義される。

📝 例 — 信号処理とサンプリング

帯域制限された信号（$\hat f$ が有界台）は Shannon-Nyquist の標本化定理により離散標本から完全に再構成できる。フィルタリングは周波数側の乗算，すなわちたたみ込みとして実現され，高速フーリエ変換（FFT）が計算を支える。

📝 例 — 加法的数論

Hardy-Littlewood の円周法は指数和 $\sum_n e^{2\pi i n^k\alpha}$ を主要弧と副次弧に分けて評価し，Waring の問題や Goldbach 型問題における表現数の漸近公式を導く。調和解析的評価が整数論の核心に現れる好例である。

💡 ウェーブレット

フーリエ変換は周波数の局在に優れるが時間の局在を欠く。ウェーブレット変換は平行移動とスケーリングによる基底を用い，時間・周波数の同時局在を実現する。多重解像度解析（MRA）は $L^2(\mathbb{R})$ の階層的分解を与え，画像圧縮（JPEG 2000）や特異積分の離散化に応用される。

✏️ 練習問題
  1. 矩形関数 $f = \mathbf{1}_{[-1,1]}$ のフーリエ変換を計算し，$\hat f \notin L^1(\mathbb{R})$ であることを確認せよ。反転公式がそのままでは適用できない理由を述べよ。
  2. Plancherel の定理を用いて $\displaystyle\int_{\mathbb{R}} \left(\frac{\sin \pi x}{\pi x}\right)^2 dx = 1$ を示せ。
  3. 近似恒等 $\{\phi_t\}$ に対し，$f\in L^1$ かつ $f$ が $x_0$ で連続ならば $(f*\phi_t)(x_0)\to f(x_0)$ を示せ。
  4. Hardy-Littlewood 極大不等式（弱 $(1,1)$ 型）を Vitali の被覆補題を用いて証明せよ。
  5. Hilbert 変換の乗子表示 $\widehat{Hf}(\xi) = -i\,\mathrm{sgn}(\xi)\,\hat f(\xi)$ から $H^2 = -\mathrm{Id}$（$L^2$ 上）を導け。
