💡 前提知識

本ページは古典力学（Newton 力学・運動方程式・釣り合い）と，ベクトルと行列（特にテンソル，対称行列の対角化・固有値）の基礎を前提とする。連続体の物理量はテンソル場として記述されるため，添字記法 $A_{ij}$ と和の規約に慣れておくと読みやすい。

1. 連続体と変形

📖 定義 — 連続体近似と変位場

物体を構成する原子の離散性を無視し，質量・運動量・エネルギーが空間に連続的に分布するとみなす理想化を連続体近似という。各物質点は基準配置での位置 $\mathbf{X}$ で識別され，変形後の現在位置 $\mathbf{x}$ へ移る。

変位場はこの移動量を表すベクトル場：

$$\mathbf{u}(\mathbf{X}, t) = \mathbf{x}(\mathbf{X}, t) - \mathbf{X}$$

連続体の変形・運動はすべて $\mathbf{u}$ とその空間微分から記述される。

📖 定義 — 変形勾配テンソル

基準配置から現在配置への局所的な写像 $\mathbf{x} = \mathbf{x}(\mathbf{X})$ の Jacobi 行列を変形勾配テンソルという：

$$F_{iJ} = \frac{\partial x_i}{\partial X_J} = \delta_{iJ} + \frac{\partial u_i}{\partial X_J}$$

体積変化は $J = \det F$ で測られ（$J>0$ が物理的），$J=1$ は非圧縮変形を意味する。$F$ から並進・回転（剛体運動）を除いた純粋な変形がひずみとして取り出される。

💡 Lagrange 記述と Euler 記述

Lagrange（物質）記述では物質点 $\mathbf{X}$ を独立変数とし「この粒子を追いかける」。固体力学で自然な記述で，変位 $\mathbf{u}(\mathbf{X},t)$ がそのまま未知関数となる。

Euler（空間）記述では空間点 $\mathbf{x}$ を独立変数とし「ある場所を通過する物質」を見る。流体で標準的。両者は物質微分 $\dfrac{D}{Dt}=\partial_t + \mathbf{v}\cdot\nabla$ で結ばれる。微小変形の弾性論では両記述の差は高次の微小量となり区別が不要になる。

2. ひずみテンソル

📖 定義 — 微小ひずみテンソル

変位勾配 $\partial u_i/\partial x_j$ が微小（$|\partial u| \ll 1$）のとき，変形の対称部分を微小ひずみテンソルと定義する：

$$\varepsilon_{ij} = \frac{1}{2}\left(\frac{\partial u_i}{\partial x_j} + \frac{\partial u_j}{\partial x_i}\right) = \frac{1}{2}(\partial_j u_i + \partial_i u_j)$$

反対称部分 $\omega_{ij}=\tfrac12(\partial_j u_i - \partial_i u_j)$ は微小な剛体回転を表し，変形には寄与しない。$\varepsilon_{ij}$ は対称テンソル（$\varepsilon_{ij}=\varepsilon_{ji}$）で，独立成分は 6 個。

💡 成分の意味

対角成分 $\varepsilon_{xx},\varepsilon_{yy},\varepsilon_{zz}$ は各軸方向の伸び縮み（垂直ひずみ），非対角成分 $\varepsilon_{xy}$ 等は角度の変化（せん断ひずみ）を表す。工学せん断ひずみ $\gamma_{xy}=2\varepsilon_{xy}$ が用いられることもあるので注意。有限変形では Green-Lagrange ひずみ $E_{ij}=\tfrac12(F_{ki}F_{kj}-\delta_{ij})$ を使い，二次の項を含む。

✓ 主ひずみ

$\varepsilon_{ij}$ は実対称テンソルなので直交変換で対角化でき，固有値 $\varepsilon_1,\varepsilon_2,\varepsilon_3$ を主ひずみ，対応する固有方向を主軸という。主軸方向にはせん断ひずみが 0 で，純粋な伸縮のみが生じる。

📖 定義 — 体積ひずみ（膨張率）

微小変形での相対体積変化はひずみテンソルのトレースに等しい：

$$\theta = \frac{\Delta V}{V} = \varepsilon_{xx}+\varepsilon_{yy}+\varepsilon_{zz} = \varepsilon_{kk} = \nabla\cdot\mathbf{u}$$

これを体積ひずみ（dilatation）という。非圧縮材料では $\theta=0$，すなわち $\nabla\cdot\mathbf{u}=0$。

📝 例 — 単軸引張

$x$ 方向に一様引張を受け，変位が $u_x = \alpha x$，$u_y = -\beta y$，$u_z = -\beta z$ で与えられるとき：

$$\varepsilon_{xx}=\alpha,\quad \varepsilon_{yy}=\varepsilon_{zz}=-\beta,\quad \varepsilon_{ij}=0\ (i\neq j)$$

体積ひずみは $\theta = \alpha - 2\beta$。後述のポアソン比は $\nu=\beta/\alpha$ で定義される。

3. 応力テンソル

📖 定義 — Cauchy 応力テンソルと応力ベクトル

連続体内部の仮想的な面に働く単位面積あたりの内力を応力という。法線ベクトル $\mathbf{n}$ をもつ面に作用する力（応力ベクトル，トラクション）$\mathbf{t}^{(\mathbf{n})}$ は，Cauchy 応力テンソル $\sigma_{ij}$ を用いて

$$t_i^{(\mathbf{n})} = \sigma_{ij}\, n_j$$

と線形に表される（Cauchy の関係式）。$\sigma_{ij}$ は「$j$ に垂直な面に働く力の $i$ 成分」を表す。対角成分が垂直応力，非対角成分がせん断応力。

✓ 釣り合い方程式（運動量保存）

微小体積要素についての Newton の第二法則（運動量保存則）を局所形にすると：

$$\frac{\partial \sigma_{ij}}{\partial x_j} + f_i = \rho\,\ddot u_i$$

ここで $f_i$ は体積力密度（重力など），$\rho$ は質量密度，$\ddot u_i$ は加速度。静的釣り合いでは右辺が 0 となり $\partial_j\sigma_{ij}+f_i=0$。

✓ 応力の対称性

角運動量保存則（体積要素のモーメント釣り合い）から，体積モーメントがない通常の連続体では Cauchy 応力テンソルは対称となる：

$$\sigma_{ij} = \sigma_{ji}$$

これにより独立成分は 6 個。対称性ゆえ応力テンソルも対角化可能で，固有値を主応力，固有方向を主応力軸という。

📝 例 — 静水圧

流体静止状態のように，あらゆる面に等方的な圧力 $p$ のみが働く場合，応力テンソルは

$$\sigma_{ij} = -p\,\delta_{ij}$$

せん断応力は存在しない。一般の応力は等方部分（圧力）$-p\delta_{ij}$（$p=-\tfrac13\sigma_{kk}$）と，トレースが 0 の偏差応力 $s_{ij}=\sigma_{ij}+p\delta_{ij}$ に分解できる。偏差応力が形状変化（せん断）を担う。

4. 構成則（線形弾性）

📖 定義 — 一般化 Hooke の法則

応力とひずみを結ぶ材料固有の関係を構成則という。線形弾性体では応力がひずみに線形比例する（一般化 Hooke の法則）：

$$\sigma_{ij} = C_{ijkl}\,\varepsilon_{kl}$$

$C_{ijkl}$ は弾性係数テンソル（4 階テンソル，対称性により独立成分は最大 21 個）。これは 1 次元のばねの法則 $F=kx$ を 3 次元テンソルへ一般化したものである。

✓ 等方弾性体と Lamé 定数

すべての方向が等価な等方弾性体では，弾性係数は 2 つの独立定数（Lamé 定数 $\lambda,\mu$）のみで表される：

$$\sigma_{ij} = \lambda\,\varepsilon_{kk}\,\delta_{ij} + 2\mu\,\varepsilon_{ij}$$

$\mu$ は剛性率（せん断弾性係数），$\lambda$ は第一 Lamé 定数。逆に解くと $\varepsilon_{ij}=\dfrac{1}{2\mu}\left(\sigma_{ij}-\dfrac{\lambda}{3\lambda+2\mu}\sigma_{kk}\delta_{ij}\right)$。

📖 定義 — 工学的弾性定数

実用上は次の定数が用いられ，Lamé 定数と相互変換できる：

定数意味関係式
ヤング率 $E$単軸引張での応力/ひずみ$E=\dfrac{\mu(3\lambda+2\mu)}{\lambda+\mu}$
ポアソン比 $\nu$横収縮/縦伸び$\nu=\dfrac{\lambda}{2(\lambda+\mu)}$
剛性率 $G=\mu$せん断応力/せん断ひずみ$G=\dfrac{E}{2(1+\nu)}$
体積弾性率 $K$圧力/体積ひずみ$K=\lambda+\dfrac{2}{3}\mu=\dfrac{E}{3(1-2\nu)}$

熱力学的安定性から $-1<\nu<\tfrac12$。$\nu\to\tfrac12$ は非圧縮極限（$K\to\infty$）。

📝 例 — 単軸引張でのヤング率

$x$ 方向のみに応力 $\sigma_{xx}=\sigma$（他成分 0）をかけると，等方則から

$$\varepsilon_{xx}=\frac{\sigma}{E},\quad \varepsilon_{yy}=\varepsilon_{zz}=-\nu\,\frac{\sigma}{E}$$

すなわち縦に伸びると横に縮む。$\sigma=E\varepsilon_{xx}$ が 1 次元 Hooke 則の形である。

5. 弾性体の運動と弾性波

✓ Navier-Cauchy 方程式

等方弾性体の構成則 $\sigma_{ij}=\lambda\varepsilon_{kk}\delta_{ij}+2\mu\varepsilon_{ij}$ を運動方程式 $\partial_j\sigma_{ij}+f_i=\rho\ddot u_i$ に代入すると，変位場 $\mathbf{u}$ に関する基礎方程式が得られる：

$$\rho\,\ddot{\mathbf{u}} = (\lambda+\mu)\,\nabla(\nabla\cdot\mathbf{u}) + \mu\,\nabla^2\mathbf{u} + \mathbf{f}$$

これを Navier-Cauchy 方程式という。弾性体の静的変形・振動・波動をすべて支配する。

✓ 縦波（P 波）と横波（S 波）

外力なし $\mathbf{f}=0$ のとき，Helmholtz 分解 $\mathbf{u}=\nabla\phi+\nabla\times\boldsymbol{\psi}$ により波は 2 種類に分離される。

縦波（P 波，疎密波）: $\nabla\cdot\mathbf{u}\neq0,\ \nabla\times\mathbf{u}=0$。変位が伝播方向に平行で，体積変化を伴う。波速

$$c_P = \sqrt{\frac{\lambda+2\mu}{\rho}}$$

横波（S 波，せん断波）: $\nabla\cdot\mathbf{u}=0,\ \nabla\times\mathbf{u}\neq0$。変位が伝播方向に垂直で，体積変化を伴わない。波速

$$c_S = \sqrt{\frac{\mu}{\rho}}$$
💡 P 波が先に届く

常に $c_P>c_S$ なので（$\lambda+2\mu>\mu$），地震では縦波の P 波が先に，横波の S 波が遅れて到達する。両者の到達時間差から震源距離を推定できる。流体は $\mu=0$ のためせん断波 S 波を伝えず，音波（縦波）のみが伝わる。

6. 梁の曲げと座屈

📖 定義 — Euler-Bernoulli 梁理論

細長い棒（梁）の曲げを扱う近似理論。「変形前に断面に垂直だった平面は変形後も平面のまま中立軸に垂直」と仮定する。中立軸からの距離 $y$ に比例して軸方向ひずみが生じ，たわみ $w(x)$ の曲率に関係づけられる：

$$\varepsilon_{xx} = -y\,\frac{d^2 w}{dx^2}$$
✓ 曲げモーメントとたわみの方程式

断面に働く内力モーメント（曲げモーメント）$M$ は曲率に比例する：

$$M = E I \frac{d^2 w}{dx^2},\qquad I = \int_A y^2\,dA$$

$EI$ は曲げ剛性，$I$ は断面二次モーメント。分布荷重 $q(x)$ とのたわみの基礎方程式：

$$EI\,\frac{d^4 w}{dx^4} = q(x)$$

境界条件（固定端・単純支持・自由端）を課して $w(x)$ を解く。

📝 例 — 片持ち梁の先端たわみ

長さ $L$ の片持ち梁の自由端に集中荷重 $P$ を加えると，先端のたわみは

$$w_{\max} = \frac{P L^3}{3 E I}$$

剛性 $EI$ が大きいほど，また長さ $L$ が短いほどたわみにくい（$L^3$ に効く）。

✓ Euler 座屈

軸方向の圧縮荷重 $P$ がある臨界値を超えると，梁・柱は突然横方向に座屈する。両端単純支持の柱のEuler 座屈荷重は

$$P_{\rm cr} = \frac{\pi^2 E I}{L^2}$$

これは Navier 方程式 $EI\,w''''+P\,w''=0$ の非自明解が存在する条件（固有値問題）として導かれる。端末条件により $L$ を有効長さ $L_e$ に置換する（一端固定他端自由なら $L_e=2L$ 等）。座屈は材料破壊ではなく構造の幾何学的不安定性である。

7. 塑性・粘弾性と流体への接続

📖 定義 — 降伏条件と塑性変形

応力が小さいうちは線形弾性（除荷で元に戻る）だが，降伏応力を超えると永久変形（塑性変形）が残る。降伏の開始は降伏条件で判定する。代表例の von Mises 条件は偏差応力 $s_{ij}$ を用いて

$$\sqrt{\tfrac{3}{2}\,s_{ij}s_{ij}} = \sigma_Y$$

と書け，$\sigma_Y$ は降伏応力。Tresca 条件（最大せん断応力基準）も広く使われる。塑性流動は体積を保存し，偏差応力に支配される。

📖 定義 — 粘弾性

ゴム・高分子・生体組織などは弾性（ばね）と粘性（ダッシュポット）の両方の性質をもつ。粘弾性体は応力がひずみだけでなくひずみ速度にも依存する。代表的モデル：

Maxwell モデル（直列）: $\dot\varepsilon=\dfrac{\dot\sigma}{E}+\dfrac{\sigma}{\eta}$（応力緩和を表現）。
Kelvin-Voigt モデル（並列）: $\sigma=E\varepsilon+\eta\dot\varepsilon$（クリープを表現）。

$\eta$ は粘性係数。時間依存の応答（クリープ・応力緩和・ヒステリシス）が特徴。

💡 固体から流体への連続性

応力を「弾性（変形量に依存）」と「粘性（変形速度に依存）」の組み合わせと見ると，固体力学と流体力学は連続的につながる。剛性率 $\mu\to0$ の極限では弾性応力が消え，せん断に抵抗しない＝流体になる。逆に Navier-Cauchy 方程式の構成則 $\sigma=\lambda\theta\delta+2\mu\varepsilon$ で，ひずみをひずみ速度に置き換えた $\sigma_{ij}=-p\delta_{ij}+\lambda'\,\dot\varepsilon_{kk}\delta_{ij}+2\eta\,\dot\varepsilon_{ij}$ が Newton 流体の構成則であり，これを運動方程式に入れると Navier-Stokes 方程式が得られる。応力・ひずみ・釣り合いというテンソルの枠組みは固体・流体に共通である。

✏️ 練習問題
  1. 変位場 $u_x=ky,\ u_y=u_z=0$（単純せん断）に対するひずみテンソル $\varepsilon_{ij}$ と回転テンソル $\omega_{ij}$ を求め，体積ひずみが 0 であることを確認せよ。
  2. 等方弾性体の関係 $E=\dfrac{\mu(3\lambda+2\mu)}{\lambda+\mu}$，$\nu=\dfrac{\lambda}{2(\lambda+\mu)}$ から，逆に $\lambda,\mu$ を $E,\nu$ で表せ。また $\nu\to\tfrac12$ で体積弾性率 $K$ がどうなるか論ぜよ。
  3. 静水圧 $\sigma_{ij}=-p\,\delta_{ij}$ をかけたときの体積ひずみ $\theta$ を等方則から求め，$p=-K\theta$ となることを示し $K$ を Lamé 定数で表せ。
  4. Navier-Cauchy 方程式に対し，渦なし変位 $\mathbf{u}=\nabla\phi$ と発散なし変位 $\mathbf{u}=\nabla\times\boldsymbol{\psi}$ をそれぞれ代入し，P 波・S 波の波動方程式と波速 $c_P,c_S$ を導け。
  5. 両端単純支持の柱（長さ $L$，曲げ剛性 $EI$）について，方程式 $EI\,w''''+P\,w''=0$ と境界条件 $w(0)=w(L)=0$，$w''(0)=w''(L)=0$ から Euler 座屈荷重 $P_{\rm cr}=\pi^2 EI/L^2$ を導け。
💡 次のステップ

剛性率 $\mu\to0$ の極限・構成則のひずみ速度版から自然につながる流体力学（Navier-Stokes 方程式）へ進むとよい。また本ページの基礎方程式（Navier-Cauchy・梁のたわみ・波動方程式）はすべて偏微分方程式であり，その解法（境界値問題・固有値問題・分離変数法）を学ぶと弾性体の具体問題を解けるようになる。
