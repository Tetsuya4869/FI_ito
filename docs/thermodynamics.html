💡 関連

熱力学はマクロな量（温度・圧力・体積・エントロピーなど）の間の関係を，分子の詳細によらず普遍的に記述する。これらマクロ量のミクロな対応物（分配関数・ミクロ状態数）を扱うのが 統計力学 である。本ページの法則は，統計力学から導出することもできる。

1. 熱力学の基本概念

📖 定義 — 系・外界・状態量

系（system）は注目する物体の集まり，その外部を外界（surroundings）と呼ぶ。境界を通じた物質・エネルギーのやり取りによって，孤立系・閉じた系・開いた系に分類する。

状態量（状態変数）は系の現在の状態だけで決まる量で，経路によらない。温度 $T$，圧力 $P$，体積 $V$，内部エネルギー $U$，エントロピー $S$ などが該当する。一方，熱 $Q$ と仕事 $W$ は経路に依存する過程量であり，状態量ではない。

状態量は示量変数（系の大きさに比例: $V, U, S, n$）と示強変数（大きさによらない: $T, P, \mu$）に分けられる。

📖 定義 — 熱平衡

系のマクロな状態量が時間とともに変化しなくなった状態を熱平衡という。平衡状態は少数の状態量で完全に指定できる。

✓ 第0法則 — 温度の存在

系 $A$ と $C$ が熱平衡にあり，$B$ と $C$ も熱平衡にあるならば，$A$ と $B$ も熱平衡にある（熱平衡の推移律）。

この推移律により，すべての系に共通する状態量として温度 $T$ が定義でき，温度計による温度測定が正当化される。

✓ 理想気体の状態方程式

希薄な気体（分子間相互作用と分子の大きさを無視）は，物質量 $n$ [mol] に対して

$$PV = nRT$$

に従う。ここで $R = 8.314\ \mathrm{J\,K^{-1}\,mol^{-1}}$ は気体定数。分子数 $N$ と Boltzmann 定数 $k_B = R/N_A$ を用いれば $PV = Nk_BT$ とも書ける。

📝 例 — 等温膨張での状態

$T$ 一定で理想気体を $V_1 \to V_2$ に膨張させると，$PV = nRT = $ 一定なので圧力は $P_1V_1 = P_2V_2$ に従って減少する。等温線（$PV$ 図上の双曲線）に沿って状態が移動する。

2. 第1法則

📖 定義 — 内部エネルギー

内部エネルギー $U$ は系を構成する分子の運動エネルギーと相互作用ポテンシャルの総和で，状態量である。系全体の重心運動・外場のエネルギーは含めない。

✓ 熱力学第1法則（エネルギー保存）

系に流入する微小熱量 $\delta Q$ と系が外界にする微小仕事 $\delta W$ に対して，内部エネルギーの変化は

$$dU = \delta Q - \delta W$$

$dU$ は完全微分（状態量の変化）だが，$\delta Q,\ \delta W$ は不完全微分（過程に依存）であることを記号 $\delta$ で区別する。準静的過程では体積変化による仕事が

$$\delta W = P\,dV$$

で与えられる。

📖 定義 — 準静的過程

系が常に平衡状態を保ったまま無限にゆっくり進む理想化された過程を準静的過程という。各瞬間で状態量が定義でき，可逆過程の前提となる。実際の有限速度の過程は不可逆である。

📖 定義 — 熱容量 $C_V,\ C_P$

単位温度上昇に要する熱量を熱容量という。体積一定と圧力一定で値が異なる:

$$C_V = \left(\frac{\partial U}{\partial T}\right)_V, \qquad C_P = \left(\frac{\partial H}{\partial T}\right)_P$$

ここで $H = U + PV$ はエンタルピー（第5節参照）。体積一定では $\delta W = 0$ なので $\delta Q = dU$，圧力一定では $\delta Q = dH$ となる。

✓ Mayer の関係式（理想気体）

理想気体では内部エネルギーが温度のみの関数（$U = U(T)$）であることから，

$$C_P - C_V = nR$$

が成り立つ。1 mol 当たりのモル比熱では $c_P - c_V = R$。比熱比 $\gamma = C_P/C_V$ は単原子理想気体で $5/3$，二原子分子で $7/5$ となる。

📝 例 — 理想気体の等温膨張で系がする仕事

温度 $T$ 一定で $V_1 \to V_2$ に準静的膨張させると，

$$W = \int_{V_1}^{V_2} P\,dV = \int_{V_1}^{V_2}\frac{nRT}{V}\,dV = nRT\ln\frac{V_2}{V_1}$$

等温では $dU = 0$ なので $Q = W = nRT\ln(V_2/V_1)$。膨張のために吸収した熱がすべて仕事に変換される。

3. 第2法則とエントロピー

✓ 第2法則 — Clausius の表現と Kelvin の表現

Clausius の表現: 低温の物体から高温の物体へ，他に何の変化も残さずに熱を移すことはできない。

Kelvin（Thomson）の表現: 一つの熱源から熱を取り出し，それを全部仕事に変える以外に何の変化も残さない過程は存在しない。

この2つの表現は等価であることが証明できる。いずれも「熱の自発的な流れには方向性がある」ことを述べている。

📖 定義 — 可逆過程と不可逆過程

系と外界をともに元の状態に完全に戻せる過程を可逆過程，戻せない過程を不可逆過程という。摩擦・有限温度差での熱伝導・自由膨張などはすべて不可逆である。可逆過程は準静的かつ散逸のない理想化された極限である。

📖 定義 — エントロピー

可逆過程に沿って系に流入する熱 $\delta Q_{rev}$ を温度 $T$ で割った量の積分により，状態量エントロピー $S$ を定義する:

$$dS = \frac{\delta Q_{rev}}{T}$$

$S$ が状態量であることは Clausius の不等式 $\oint \delta Q/T \le 0$（可逆なら等号）から保証される。

✓ エントロピー増大則

孤立系では，任意の自発過程に対してエントロピーは決して減少しない:

$$dS \ge 0$$

等号は可逆過程，不等号は不可逆過程で成り立つ。一般の系では，系と外界を合わせた全エントロピー $S_{\rm sys} + S_{\rm surr}$ について $dS_{\rm total} \ge 0$ となる。これが時間の矢を与える。

📝 例 — 理想気体の自由膨張

真空への自由膨張（$V_1 \to V_2$，断熱・仕事なし）では $Q = W = 0$ ゆえ $dU = 0$，温度は不変。しかし $S$ は状態量なので，同じ始終状態を結ぶ可逆等温膨張で計算でき，

$$\Delta S = nR\ln\frac{V_2}{V_1} > 0$$

外界に変化はないので全体でも $\Delta S > 0$。自由膨張が不可逆であることを示す。

4. 熱機関とCarnotサイクル

📖 定義 — 熱機関と効率

熱機関は高温熱源（温度 $T_h$）から熱 $Q_h$ を受け取り，仕事 $W$ をして，低温熱源（温度 $T_c$）に熱 $Q_c$ を捨てるサイクル装置である。エネルギー保存から $W = Q_h - Q_c$。熱効率は

$$\eta = \frac{W}{Q_h} = 1 - \frac{Q_c}{Q_h}$$

で定義される（$0 \le \eta < 1$）。

📖 定義 — Carnot サイクル

2つの等温過程と2つの断熱過程からなる可逆サイクル:

  1. 等温膨張（$T_h$）: 高温熱源から $Q_h$ を吸収。
  2. 断熱膨張: $T_h \to T_c$ に温度低下。
  3. 等温圧縮（$T_c$）: 低温熱源へ $Q_c$ を放出。
  4. 断熱圧縮: $T_c \to T_h$ に戻る。
✓ Carnot の定理と効率

同じ2つの熱源で動作するすべての熱機関のうち，可逆機関（Carnot 機関）の効率が最大であり，作業物質によらず温度のみで決まる。理想気体の Carnot サイクルを計算すると $Q_c/Q_h = T_c/T_h$ となり，

$$\eta_{\rm Carnot} = 1 - \frac{T_c}{T_h}$$

不可逆機関では $\eta < \eta_{\rm Carnot}$。これは第2法則の直接の帰結であり，熱力学温度（絶対温度）の定義の基礎にもなる。

📝 例 — Carnot 効率の数値

$T_h = 600\ \mathrm{K}$，$T_c = 300\ \mathrm{K}$ のとき，$\eta_{\rm Carnot} = 1 - 300/600 = 0.5$（50%）。すなわち吸収熱の半分しか仕事にできず，残りは必ず低温側へ捨てられる。

💡 エントロピーとの整合

Carnot サイクル1周ではエントロピーが元に戻るため $\oint dS = Q_h/T_h - Q_c/T_c = 0$，すなわち $Q_c/Q_h = T_c/T_h$。Carnot 効率の式はエントロピーが状態量であることと等価である。

5. 熱力学ポテンシャルとMaxwell関係式

📖 定義 — 熱力学ポテンシャル

内部エネルギー $U(S,V)$ の Legendre 変換により，扱いやすい独立変数に応じた4つのポテンシャルを定義する:

ポテンシャル定義自然な変数全微分
内部エネルギー $U$$U$$(S,V)$$dU = T\,dS - P\,dV$
エンタルピー $H$$H = U + PV$$(S,P)$$dH = T\,dS + V\,dP$
Helmholtz 自由エネルギー $F$$F = U - TS$$(T,V)$$dF = -S\,dT - P\,dV$
Gibbs 自由エネルギー $G$$G = U - TS + PV$$(T,P)$$dG = -S\,dT + V\,dP$
💡 ポテンシャルの物理的意味

等温過程（$T$ 一定）で系がなし得る最大仕事は $-\Delta F$，等温等圧過程での自発変化の方向は $G$ が減少する向き（$dG \le 0$）で決まる。化学反応・相平衡では $G$ が中心的役割を果たす。$H$ は定圧過程の熱の出入りに対応する。

✓ Maxwell 関係式

各ポテンシャルが状態量で，全微分の2階偏微分が交換可能（$\partial^2/\partial x\partial y = \partial^2/\partial y\partial x$）であることから，4つの関係式が得られる:

$$\left(\frac{\partial T}{\partial V}\right)_S = -\left(\frac{\partial P}{\partial S}\right)_V, \qquad \left(\frac{\partial T}{\partial P}\right)_S = \left(\frac{\partial V}{\partial S}\right)_P$$ $$\left(\frac{\partial S}{\partial V}\right)_T = \left(\frac{\partial P}{\partial T}\right)_V, \qquad \left(\frac{\partial S}{\partial P}\right)_T = -\left(\frac{\partial V}{\partial T}\right)_P$$

左の2式は $U,H$ から，下の2式は $F,G$ から導かれる。測定困難なエントロピー微分を，測定可能な $P,V,T$ の微分に置き換える強力な道具である。

📝 例 — Maxwell 関係式の応用

$F$ から得られる $\left(\partial S/\partial V\right)_T = \left(\partial P/\partial T\right)_V$ を用いると，等温膨張でのエントロピー変化や熱の出入りを，状態方程式 $P(T,V)$ だけから計算できる。理想気体では $\left(\partial P/\partial T\right)_V = nR/V$ なので $\Delta S = nR\ln(V_2/V_1)$ が再現される。

6. 相平衡と相転移

📖 定義 — 相・相図・化学ポテンシャル

相とは固体・液体・気体のように物理的性質が一様な状態をいう。$P$-$T$ 平面に各相の存在領域と境界線を描いたものが相図で，境界線上では2相が共存する。

化学ポテンシャルは1粒子（または1 mol）当たりの Gibbs 自由エネルギー $\mu = (\partial G/\partial N)_{T,P}$。多相系の平衡条件は，各相の化学ポテンシャルが等しいこと: $\mu_1 = \mu_2$。

✓ Clausius-Clapeyron の式

2相共存線 $P(T)$ の傾きは，潜熱 $L$（モル当たり）と両相のモル体積差 $\Delta v = v_2 - v_1$ を用いて

$$\frac{dP}{dT} = \frac{L}{T\,\Delta v}$$

で与えられる（共存線上で $\mu_1 = \mu_2$ を $T$ で微分して導く）。気液転移で気相を理想気体・$v_g \gg v_l$ と近似すると

$$\frac{dP}{dT} \approx \frac{LP}{RT^2} \;\Rightarrow\; \ln P \approx -\frac{L}{RT} + \text{const.}$$

となり，蒸気圧の温度依存性を説明する。

✓ Gibbs の相律

$c$ 成分・$p$ 相が平衡共存する系の自由度（独立に変えられる示強変数の数）は

$$f = c - p + 2$$

純物質（$c=1$）では，1相で $f=2$（$T,P$ を自由に選べる面），2相共存で $f=1$（共存線），3相共存で $f=0$（三重点，1点に固定）。これが相図の次元構造を決める。

📝 例 — 水の三重点

水は $c=1$ で，固・液・気の3相が共存する点では $f = 1 - 3 + 2 = 0$。すなわち三重点は唯一の温度・圧力（$T = 273.16\ \mathrm{K}$，$P \approx 611\ \mathrm{Pa}$）でのみ実現する。

7. 第3法則と低温

✓ 第3法則（Nernst の定理）

温度が絶対零度に近づくとき，あらゆる可逆等温過程に伴うエントロピー変化はゼロに近づく:

$$\lim_{T\to 0}\Delta S = 0$$

Planck の強い形では，純粋な完全結晶のエントロピーは $T \to 0$ で普遍的な定数（ふつう $0$）に近づく:

$$\lim_{T\to 0} S = 0$$

これによりエントロピーの絶対値が定まる。統計力学的には，基底状態の縮退度を $g$ として $S(0) = k_B \ln g$ に対応する。

💡 低温での熱容量

第3法則から $S(T) = \int_0^T (C/T')\,dT'$ が有限であるためには，$T\to 0$ で $C \to 0$ でなければならない。実際，固体の比熱は Debye の $T^3$ 則，金属の電子比熱は $T$ に比例して $0$ へ向かう。古典統計の Dulong-Petit 則（$C = 3R$ 一定）は低温で破綻する。

✓ 絶対零度到達不可能性

第3法則の帰結として，有限回の操作で絶対零度に到達することはできない。断熱消磁などの冷却サイクルは，$T \to 0$ で各ステップの温度降下が次第に小さくなり，無限のステップを要する。絶対零度は近づけても決して到達できない極限である。

📝 例 — エントロピーの絶対値の利用

第3法則を基準（$S(0)=0$）にとると，物質の標準エントロピー $S(T) = \int_0^T (C_P/T')\,dT'$（相転移の潜熱項を加える）を実験的熱容量から積分で求められる。これは化学反応の $\Delta G = \Delta H - T\Delta S$ を通じて反応の自発性を予測するのに使われる。

💡 次のステップ

本ページのマクロな法則をミクロな分子描像（分配関数・状態数・ゆらぎ）から基礎づけるのが 統計力学 である。エントロピー $S = k_B\ln W$ や熱力学ポテンシャルの統計的導出はそこで扱う。また，流体の熱輸送・対流・状態方程式の連続体への応用は 流体力学 へ続く。
