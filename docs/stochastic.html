1. ブラウン運動

📖 定義 — ブラウン運動（Wiener 過程）

確率過程 $W = \{W_t\}_{t \ge 0}$ が（標準）ブラウン運動であるとは：

  1. $W_0 = 0$ (a.s.)
  2. 独立増分: $0 \le s < t$，$W_t - W_s$ は $\mathcal{F}_s$ と独立
  3. 正規増分: $W_t - W_s \sim \mathcal{N}(0, t-s)$
  4. 連続経路: $t \mapsto W_t(\omega)$ は a.s. 連続
✓ ブラウン運動の性質
  • $W_t$ は $\mathcal{F}_t$-マルチンゲール
  • $W_t^2 - t$ はマルチンゲール（二次変分 $[W]_t = t$）
  • 非有界変動: 経路は至るところ微分不可能で，全変動は a.s. 無限大
  • 二次変分: $\sum_i (W_{t_{i+1}} - W_{t_i})^2 \to t$ (in $L^2$)
  • スケーリング: $\tilde W_t = cW_{t/c^2}$ もブラウン運動
💡 $dW_t$ の計算規則
$$dW_t \cdot dW_t = dt, \quad dW_t \cdot dt = 0, \quad dt \cdot dt = 0$$

2. フィルトレーションとマルチンゲール

📖 定義 — フィルトレーション・適合過程

増大するσ加法族族 $\mathbb{F} = \{\mathcal{F}_t\}_{t \ge 0}$（$s \le t \implies \mathcal{F}_s \subset \mathcal{F}_t$）をフィルトレーションという。$X_t$ が各 $t$ で $\mathcal{F}_t$-可測のとき適合過程という。

📖 定義 — マルチンゲール

適合過程 $M_t$ がマルチンゲールであるとは，$E[|M_t|] < \infty$ かつ

$$E[M_t \mid \mathcal{F}_s] = M_s \quad \forall s \le t$$

$\ge$ なら劣マルチンゲール，$\le$ なら優マルチンゲール。

📝 マルチンゲールの例
  • $W_t$（ブラウン運動）
  • $W_t^2 - t$
  • $M_t = \exp(\sigma W_t - \frac{\sigma^2}{2}t)$（指数マルチンゲール）
  • $E[X \mid \mathcal{F}_t]$（任意の可積分 $X$ に対して）

3. 伊藤積分

📖 定義 — 伊藤積分の構成

Step 1 — 単純過程: $\phi_t = \sum_k \xi_k \mathbf{1}_{(t_k,t_{k+1}]}$（$\xi_k$ は $\mathcal{F}_{t_k}$-可測）に対して，

$$\int_0^T \phi_t\,dW_t = \sum_k \xi_k (W_{t_{k+1}} - W_{t_k})$$

Step 2 — 一般の予測可能過程: $E[\int_0^T \phi_t^2\,dt] < \infty$ を満たす過程を $L^2$ の単純過程で近似して定義する。

✓ 定理 — 伊藤等長性 (Itô Isometry)
$$E\!\left[\left(\int_0^T \phi_t\,dW_t\right)^2\right] = E\!\left[\int_0^T \phi_t^2\,dt\right]$$
✓ マルチンゲール性

$\phi \in L^2$（適切な条件のもと）なら $M_t = \int_0^t \phi_s\,dW_s$ はマルチンゲール。したがって $E[M_t] = 0$。

⚠️ Itô 積分 vs. Stratonovich 積分

伊藤積分は左端点で評価（非先読み）。Stratonovich 積分は中点で評価し，通常の連鎖律が成り立つが，マルチンゲール性は失われる。変換式:

$$\int_0^T f(W_t)\,\circ\,dW_t = \int_0^T f(W_t)\,dW_t + \frac{1}{2}\int_0^T f'(W_t)\,dt$$

4. 伊藤の公式

✓ 定理 — 伊藤の公式（1次元）

$X_t$ が Itô 過程 $dX_t = \mu_t\,dt + \sigma_t\,dW_t$ で，$f \in C^{1,2}([0,\infty) \times \mathbb{R})$ ならば，

$$df(t, X_t) = \frac{\partial f}{\partial t}\,dt + \frac{\partial f}{\partial x}\,dX_t + \frac{1}{2}\frac{\partial^2 f}{\partial x^2}\,d\langle X \rangle_t$$

展開形（$d\langle X \rangle_t = \sigma_t^2\,dt$ に注意）：

$$df(t,X_t) = \left(\frac{\partial f}{\partial t} + \mu_t\frac{\partial f}{\partial x} + \frac{1}{2}\sigma_t^2\frac{\partial^2 f}{\partial x^2}\right)dt + \sigma_t\frac{\partial f}{\partial x}\,dW_t$$
💡 二次変分による直観

テイラー展開 $df \approx f_t\,dt + f_x\,dX + \frac{1}{2}f_{xx}(dX)^2$ において，$(dX)^2 = \sigma_t^2\,dt + O(dt^{3/2})$ となるため，通常の微積分に比べて $\frac{1}{2}\sigma_t^2 f_{xx}\,dt$ の項（伊藤修正項）が加わる。

📝 例 1 — $f(x) = x^2$，$X_t = W_t$
$$d(W_t^2) = 2W_t\,dW_t + dt$$ $$\therefore \int_0^t W_s\,dW_s = \frac{W_t^2 - t}{2}$$

通常の微積分なら $\int_0^t W_s\,dW_s = W_t^2/2$。差の $-t/2$ が伊藤修正項。

📝 例 2 — 幾何ブラウン運動 (GBM)

$dS_t = \mu S_t\,dt + \sigma S_t\,dW_t$ に $f(x) = \ln x$ を適用すると，

$$d(\ln S_t) = \frac{1}{S_t}dS_t - \frac{1}{2S_t^2}(dS_t)^2 = \left(\mu - \frac{\sigma^2}{2}\right)dt + \sigma\,dW_t$$ $$\therefore \quad S_t = S_0 \exp\!\left[\left(\mu - \frac{\sigma^2}{2}\right)t + \sigma W_t\right]$$
✓ 積の公式（伊藤の積公式）

$X_t, Y_t$ が Itô 過程のとき，

$$d(X_t Y_t) = X_t\,dY_t + Y_t\,dX_t + d\langle X, Y\rangle_t$$

5. 確率微分方程式

📖 定義 — SDE
$$dX_t = b(t, X_t)\,dt + \sigma(t, X_t)\,dW_t, \quad X_0 = x_0$$

$b$（ドリフト係数），$\sigma$（拡散係数）は与えられた関数。

✓ 定理 — 存在一意性（Lipschitz 条件）

$b, \sigma$ が $x$ について Lipschitz 連続かつ線形増大条件 $|b(t,x)|+|\sigma(t,x)| \le C(1+|x|)$ を満たせば，強解が a.s. 一意に存在する。

モデルSDE解析解
GBM（株価）$dS = \mu S\,dt + \sigma S\,dW$$S_0 e^{(\mu-\sigma^2/2)t + \sigma W_t}$
Ornstein-Uhlenbeck$dX = \theta(\mu-X)\,dt + \sigma\,dW$$\mu + (X_0-\mu)e^{-\theta t} + \sigma\int_0^t e^{-\theta(t-s)}dW_s$
CIR（金利）$dr = \kappa(\theta-r)\,dt + \sigma\sqrt{r}\,dW$非中心 $\chi^2$ 分布

6. Girsanov の定理

✓ 定理 — Girsanov

$\theta_t$ が適合過程で Novikov 条件 $E\!\left[e^{\frac{1}{2}\int_0^T \theta_t^2\,dt}\right] < \infty$ を満たすとする。Radon-Nikodym 導関数

$$\frac{dQ}{dP} = \exp\!\left(-\int_0^T \theta_t\,dW_t - \frac{1}{2}\int_0^T \theta_t^2\,dt\right) \equiv \mathcal{E}(\theta)_T$$

で定義される確率測度 $Q$ のもとで，

$$\tilde W_t = W_t + \int_0^t \theta_s\,ds$$

は $Q$-ブラウン運動になる。

7. 金融数学への応用

📝 ブラック・ショールズ公式の導出

設定: $dS_t = \mu S_t\,dt + \sigma S_t\,dW_t^P$（リアル測度 $P$）

Step 1 — リスク中立測度 $Q$ へ変換: Girsanov で $\theta_t = (\mu-r)/\sigma$（市場価格のリスク），$dW_t^Q = dW_t^P + \theta_t\,dt$

$$dS_t = r S_t\,dt + \sigma S_t\,dW_t^Q$$

Step 2 — オプション価格: 裁定なし価格は $Q$ のもとでの割引き期待値

$$C_0 = e^{-rT} E^Q[\max(S_T - K, 0)]$$

Step 3 — 計算: $S_T = S_0 e^{(r-\sigma^2/2)T + \sigma W_T^Q}$，$W_T^Q \sim \mathcal{N}(0,T)$ を利用すると，

$$\boxed{C_0 = S_0 N(d_1) - K e^{-rT} N(d_2)}$$ $$d_1 = \frac{\ln(S_0/K) + (r+\sigma^2/2)T}{\sigma\sqrt{T}}, \quad d_2 = d_1 - \sigma\sqrt{T}$$
📝 ブラック・ショールズ PDE

$V(t, S)$ にデルタヘッジ $\Pi = V - \Delta S$（$\Delta = \partial V/\partial S$）を適用すると，

$$\frac{\partial V}{\partial t} + rS\frac{\partial V}{\partial S} + \frac{1}{2}\sigma^2 S^2\frac{\partial^2 V}{\partial S^2} - rV = 0$$

境界条件: コール $V(T,S) = (S-K)^+$，プット $V(T,S) = (K-S)^+$

✏️ 練習問題
  1. 伊藤の公式を使って $\int_0^t W_s^2\,dW_s$ を $W_t$ と $\int_0^t W_s\,ds$ で表せ。
  2. $X_t = e^t \sin(W_t)$ の SDE を求めよ。
  3. Ornstein-Uhlenbeck 過程 $dX_t = -\theta X_t\,dt + \sigma\,dW_t$ の解析解を $Y_t = e^{\theta t}X_t$ を使って導け。
  4. プット-コール・パリティ $C - P = S_0 - Ke^{-rT}$ を導け。
解答例を見る
  1. $\int_0^t W_s^2\,dW_s$ の表示. $f(x)=x^3$ に伊藤の公式（$X_t=W_t$，$\mu=0$, $\sigma=1$）を適用する： $$d(W_t^3) = 3W_t^2\,dW_t + \tfrac{1}{2}\cdot 6W_t\,dt = 3W_t^2\,dW_t + 3W_t\,dt.$$ 両辺を $0$ から $t$ まで積分して（$W_0=0$） $$W_t^3 = 3\int_0^t W_s^2\,dW_s + 3\int_0^t W_s\,ds.$$ よって $$\int_0^t W_s^2\,dW_s = \frac{W_t^3}{3} - \int_0^t W_s\,ds.$$
  2. $X_t=e^t\sin(W_t)$ の SDE. $f(t,x)=e^t\sin x$ とおくと $f_t=e^t\sin x$，$f_x=e^t\cos x$，$f_{xx}=-e^t\sin x$。$X_t=W_t$（$\mu=0,\sigma=1$）に対する伊藤の公式より $$dX_t = \left(f_t + \tfrac12 f_{xx}\right)dt + f_x\,dW_t = \left(e^t\sin W_t - \tfrac12 e^t\sin W_t\right)dt + e^t\cos W_t\,dW_t.$$ すなわち $$dX_t = \tfrac{1}{2}e^t\sin(W_t)\,dt + e^t\cos(W_t)\,dW_t = \tfrac{1}{2}X_t\,dt + e^t\cos(W_t)\,dW_t.$$
  3. OU 過程 $dX_t=-\theta X_t\,dt+\sigma\,dW_t$ の解. $Y_t=e^{\theta t}X_t$ とおき積の公式（または $f(t,x)=e^{\theta t}x$ への伊藤の公式）を使う。$f_t=\theta e^{\theta t}x$，$f_x=e^{\theta t}$，$f_{xx}=0$ なので $$dY_t = \theta e^{\theta t}X_t\,dt + e^{\theta t}\,dX_t = \theta e^{\theta t}X_t\,dt + e^{\theta t}(-\theta X_t\,dt + \sigma\,dW_t) = \sigma e^{\theta t}\,dW_t.$$ ドリフトが消えるので積分して $$Y_t = Y_0 + \sigma\int_0^t e^{\theta s}\,dW_s,\qquad Y_0=X_0.$$ $X_t=e^{-\theta t}Y_t$ に戻すと $$X_t = X_0 e^{-\theta t} + \sigma\int_0^t e^{-\theta(t-s)}\,dW_s.$$ （表の OU 解で $\mu=0$ とした形に一致する。）
  4. プット-コール・パリティ. 満期 $T$ で，同一の行使価格 $K$ をもつコールとプットは $\max(S_T-K,0)-\max(K-S_T,0)=S_T-K$ を満たす。リスク中立測度 $Q$ のもとで割引期待値をとると $$C_0 - P_0 = e^{-rT}E^Q[S_T-K] = e^{-rT}E^Q[S_T] - Ke^{-rT}.$$ $Q$ のもとで割引株価 $e^{-rt}S_t$ はマルチンゲール（$E^Q[S_T]=S_0 e^{rT}$）なので $$C_0 - P_0 = e^{-rT}\cdot S_0 e^{rT} - Ke^{-rT} = S_0 - Ke^{-rT}.$$
