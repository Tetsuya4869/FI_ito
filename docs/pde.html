1. PDE の分類

2 階線形 PDE $au_{xx} + 2bu_{xy} + cu_{yy} + \cdots = 0$ の判別式 $\Delta = b^2 - ac$ で分類する：

型条件代表例物理的意味
楕円型$\Delta < 0$Laplace 方程式 $\Delta u=0$定常状態
放物型$\Delta = 0$熱方程式 $u_t = \kappa u_{xx}$拡散・時間発展
双曲型$\Delta > 0$波動方程式 $u_{tt} = c^2 u_{xx}$波の伝播

2. 熱方程式

📖 熱方程式
$$u_t = \kappa \Delta u \quad (x \in \mathbb{R}^n,\, t > 0), \quad u(x,0) = u_0(x)$$
✓ 基本解（熱核）

$\mathbb{R}^n$ での基本解（熱核）：

$$\Phi(x,t) = \frac{1}{(4\pi\kappa t)^{n/2}} \exp\!\left(-\frac{|x|^2}{4\kappa t}\right)$$

初期値問題の解：

$$u(x,t) = \int_{\mathbb{R}^n} \Phi(x-y, t)\,u_0(y)\,dy = (\Phi(\cdot,t) * u_0)(x)$$
📝 分離変数法（有界領域）

$u_t = \kappa u_{xx}$，$x \in [0,L]$，$u(0,t)=u(L,t)=0$，$u(x,0)=f(x)$

$u(x,t) = X(x)T(t)$ とおくと，$T'/(\kappa T) = X''/X = -\lambda$。

$$u(x,t) = \sum_{n=1}^\infty b_n e^{-\kappa n^2\pi^2 t/L^2} \sin\!\left(\frac{n\pi x}{L}\right), \quad b_n = \frac{2}{L}\int_0^L f(x)\sin\!\left(\frac{n\pi x}{L}\right)dx$$
✓ 最大値原理

$u_t - \kappa\Delta u = 0$ が $Q_T = \Omega \times (0,T]$ で成り立つとき，

$$\max_{\bar Q_T} u = \max_{\partial_p Q_T} u$$

（$\partial_p Q_T$ は放物境界 $= \{t=0\} \cup \{\partial\Omega \times [0,T]\}$）

3. 波動方程式

📖 波動方程式
$$u_{tt} = c^2 \Delta u, \quad u(x,0) = f(x),\;\; u_t(x,0) = g(x)$$
✓ d'Alembert の公式（1 次元）
$$u(x,t) = \frac{f(x+ct)+f(x-ct)}{2} + \frac{1}{2c}\int_{x-ct}^{x+ct} g(s)\,ds$$

解は 2 つの進行波 $f(x+ct)/2$ と $f(x-ct)/2$ の重ね合わせ。

✓ エネルギー保存則
$$E(t) = \frac{1}{2}\int_{\mathbb{R}} \left(u_t^2 + c^2 u_x^2\right)dx = \text{const}$$
💡 Huygens の原理（3 次元）

3 次元波動方程式では，初期データの台が $B(x_0, R)$ に含まれるとき，$x$ での解は $ct = |x-x_0|$ 付近でのみ非零（シャープな波面）。2 次元では尾を引く（Huygens の原理は奇数次元でのみ成立）。

4. ラプラス方程式・ポアソン方程式

📖 ラプラス方程式・ポアソン方程式
$$\Delta u = 0 \quad (\text{ラプラス}), \quad \Delta u = -f \quad (\text{ポアソン})$$

ラプラス方程式の解を調和関数という。

✓ 平均値性質

$u$ が $\Omega$ 上で調和ならば，$B(x,r) \Subset \Omega$ に対して，

$$u(x) = \frac{1}{|\partial B(x,r)|}\int_{\partial B(x,r)} u\,dS = \frac{1}{|B(x,r)|}\int_{B(x,r)} u\,dy$$
✓ 最大値原理（楕円型）

$\Omega$ 上の調和関数は，$\overline\Omega$ 内の最大値と最小値を $\partial\Omega$ 上で達成する。

📝 Green 関数（自由空間）
$$G(x) = \begin{cases} -\dfrac{1}{2\pi}\ln|x| & n=2 \\[6pt] \dfrac{1}{(n-2)\omega_n |x|^{n-2}} & n \ge 3 \end{cases}$$

$\omega_n$ は $\mathbb{R}^n$ の単位球の体積。$\Delta G(x) = \delta(x)$。

5. フーリエ変換法

📖 フーリエ変換
$$\hat f(\xi) = \mathcal{F}[f](\xi) = \int_{\mathbb{R}^n} f(x) e^{-i\xi\cdot x}\,dx$$ $$f(x) = \mathcal{F}^{-1}[\hat f](x) = \frac{1}{(2\pi)^n}\int_{\mathbb{R}^n} \hat f(\xi) e^{i\xi\cdot x}\,d\xi$$
✓ 基本性質
  • 微分: $\widehat{\partial^\alpha f}(\xi) = (i\xi)^\alpha \hat f(\xi)$
  • 畳み込み: $\widehat{f*g} = \hat f \cdot \hat g$
  • Parseval: $\|f\|_{L^2} = \|\hat f\|_{L^2}/(2\pi)^{n/2}$
  • Gauss 関数: $\mathcal{F}[e^{-|x|^2/2}](\xi) = (2\pi)^{n/2}e^{-|\xi|^2/2}$
📝 フーリエ変換で熱方程式を解く

$u_t = \kappa\Delta u$ に $x$ についてフーリエ変換を適用すると，

$$\partial_t \hat u(t,\xi) = -\kappa|\xi|^2 \hat u(t,\xi) \implies \hat u(t,\xi) = \hat u_0(\xi) e^{-\kappa|\xi|^2 t}$$

逆変換: $u(x,t) = \Phi(\cdot,t) * u_0$（熱核との畳み込み）。

6. 弱解と Sobolev 空間

📖 定義 — Sobolev 空間
$$W^{k,p}(\Omega) = \{u \in L^p(\Omega) \mid \partial^\alpha u \in L^p(\Omega),\; |\alpha| \le k\}$$ $$H^k(\Omega) = W^{k,2}(\Omega), \quad \|u\|_{H^k}^2 = \sum_{|\alpha|\le k}\|\partial^\alpha u\|_{L^2}^2$$
📖 定義 — 弱解

$-\Delta u = f$ の弱解とは，$u \in H^1_0(\Omega)$ で，

$$\int_\Omega \nabla u \cdot \nabla v\,dx = \int_\Omega f v\,dx \quad \forall v \in H^1_0(\Omega)$$

を満たすもの（部分積分で得られる変分形式）。

✓ 定理 — Lax-Milgram

Hilbert 空間 $H$ 上で $a(\cdot,\cdot)$ が連続かつ強圧的（coercive）なら，$a(u,v) = \langle f, v\rangle$ の解 $u \in H$ が一意に存在する。
