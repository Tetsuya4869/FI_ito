1. Schrödinger 方程式

📖 時間依存 Schrödinger 方程式
$$i\hbar\frac{\partial}{\partial t}|\Psi(t)\rangle = \hat H |\Psi(t)\rangle$$

波動関数 $\Psi(\mathbf{r},t) = \langle\mathbf{r}|\Psi(t)\rangle$，確率密度 $\rho = |\Psi|^2$，$\int|\Psi|^2\,d^3r = 1$。

$\hat H = -\dfrac{\hbar^2}{2m}\nabla^2 + V(\mathbf{r})$（位置表示）。

✓ 時間非依存 Schrödinger 方程式（定常状態）
$$\hat H\psi_n = E_n\psi_n$$

一般解: $\Psi(\mathbf{r},t) = \sum_n c_n \psi_n(\mathbf{r})e^{-iE_n t/\hbar}$，$c_n = \langle\psi_n|\Psi(0)\rangle$。

📖 確率流と連続の方程式
$$\mathbf{J} = \frac{\hbar}{2mi}(\Psi^*\nabla\Psi - \Psi\nabla\Psi^*), \quad \frac{\partial\rho}{\partial t} + \nabla\cdot\mathbf{J} = 0$$

2. Hilbert 空間と演算子形式

📖 Dirac の bra-ket 記法

状態ベクトル: $|\psi\rangle \in \mathcal{H}$（Hilbert 空間）。内積: $\langle\phi|\psi\rangle \in \mathbb{C}$。

完全性: $\sum_n |n\rangle\langle n| = \hat I$（離散），$\int |x\rangle\langle x|\,dx = \hat I$（連続）。

観測量 = Hermite 演算子 $\hat A = \hat A^\dagger$。期待値: $\langle\hat A\rangle = \langle\psi|\hat A|\psi\rangle$。

✓ スペクトル定理

Hermite 演算子 $\hat A$ は実固有値をもち，異なる固有値に対応する固有ベクトルは直交する。完全正規直交基底が存在する（スペクトル分解 $\hat A = \sum_n a_n |n\rangle\langle n|$）。

📖 正準交換関係
$$[\hat x_i, \hat p_j] = i\hbar\delta_{ij}, \quad [\hat x_i,\hat x_j] = [\hat p_i,\hat p_j] = 0$$

位置表示: $\hat x = x$，$\hat p = -i\hbar\nabla$。運動量表示: $\hat p = p$，$\hat x = i\hbar\nabla_p$。

✓ Ehrenfest の定理

期待値は古典力学の方程式に従う:

$$m\frac{d^2}{dt^2}\langle\mathbf{r}\rangle = -\langle\nabla V\rangle$$

3. 不確定性原理

✓ Robertson の不確定性関係

任意の Hermite 演算子 $\hat A, \hat B$ について:

$$\Delta A \cdot \Delta B \ge \frac{1}{2}|\langle[\hat A,\hat B]\rangle|$$

位置・運動量: $\Delta x \cdot \Delta p \ge \hbar/2$（$[\hat x, \hat p] = i\hbar$）。

エネルギー・時間: $\Delta E \cdot \Delta t \ge \hbar/2$（$\Delta E$: エネルギー幅，$\Delta t$: 状態の寿命）。

💡 不確定性の本質

位置と運動量の不確定性は測定の技術的限界ではなく，量子状態自体の性質。両演算子が同時に確定値を持つ（同時固有状態に在る）ことは $[\hat x,\hat p]\ne 0$ ゆえ不可能。

4. 調和振動子

📖 昇降演算子（生成・消滅演算子）

$\hat H = \hat p^2/(2m) + m\omega^2\hat x^2/2$ に対して:

$$\hat a = \sqrt{\frac{m\omega}{2\hbar}}\left(\hat x + \frac{i\hat p}{m\omega}\right), \quad \hat a^\dagger = \sqrt{\frac{m\omega}{2\hbar}}\left(\hat x - \frac{i\hat p}{m\omega}\right)$$ $$[\hat a, \hat a^\dagger] = 1, \quad \hat H = \hbar\omega\left(\hat a^\dagger\hat a + \frac{1}{2}\right)$$
✓ エネルギー固有値と固有状態
$$E_n = \hbar\omega\left(n + \frac{1}{2}\right), \quad n = 0, 1, 2, \ldots$$

$\hat a^\dagger|n\rangle = \sqrt{n+1}|n+1\rangle$，$\hat a|n\rangle = \sqrt{n}|n-1\rangle$，$\hat a|0\rangle = 0$。

基底状態波動関数: $\psi_0(x) = \left(\frac{m\omega}{\pi\hbar}\right)^{1/4} e^{-m\omega x^2/(2\hbar)}$（ガウス関数）。

5. 中心力場・水素原子

✓ 球面調和関数と角運動量量子数

中心力ポテンシャル $V(r)$ では $\psi_{nlm}(r,\theta,\phi) = R_{nl}(r)Y_l^m(\theta,\phi)$。

角運動量: $\hat L^2 Y_l^m = \hbar^2 l(l+1)Y_l^m$，$\hat L_z Y_l^m = \hbar m Y_l^m$（$l=0,1,2,\ldots$；$m=-l,\ldots,l$）。

✓ 水素原子のエネルギー準位

クーロンポテンシャル $V(r) = -e^2/(4\pi\varepsilon_0 r)$ のもとでの束縛状態エネルギー:

$$E_n = -\frac{m_e e^4}{2(4\pi\varepsilon_0)^2\hbar^2}\cdot\frac{1}{n^2} = -\frac{13.6\,\text{eV}}{n^2}, \quad n = 1,2,3,\ldots$$

縮退度: $n^2$（スピンを含めると $2n^2$）。主量子数 $n$，方位量子数 $l=0,\ldots,n-1$，磁気量子数 $m=-l,\ldots,l$。

6. スピンと角運動量

📖 スピン $1/2$ と Pauli 行列

スピン演算子: $\hat S_i = \frac{\hbar}{2}\sigma_i$（$i=x,y,z$）。Pauli 行列:

$$\sigma_x = \begin{pmatrix}0&1\\1&0\end{pmatrix}, \quad \sigma_y = \begin{pmatrix}0&-i\\i&0\end{pmatrix}, \quad \sigma_z = \begin{pmatrix}1&0\\0&-1\end{pmatrix}$$

固有値 $\pm\hbar/2$，$[\hat S_i,\hat S_j] = i\hbar\varepsilon_{ijk}\hat S_k$。

✓ 角運動量の合成（Clebsch-Gordan）

$j_1\otimes j_2$ の合成: $|j_1-j_2| \le j \le j_1+j_2$。$|j,m\rangle = \sum_{m_1,m_2}\langle j_1 m_1 j_2 m_2|j m\rangle|j_1 m_1\rangle|j_2 m_2\rangle$。

7. 摂動論

✓ 時間非依存摂動論

$\hat H = \hat H_0 + \lambda\hat H'$（$\lambda$ 小）。$E_n = E_n^{(0)} + \lambda E_n^{(1)} + \lambda^2 E_n^{(2)} + \cdots$

$$E_n^{(1)} = \langle n^{(0)}|\hat H'|n^{(0)}\rangle$$ $$E_n^{(2)} = \sum_{k\ne n}\frac{|\langle k^{(0)}|\hat H'|n^{(0)}\rangle|^2}{E_n^{(0)}-E_k^{(0)}}$$

一階補正波動関数: $|n^{(1)}\rangle = \sum_{k\ne n}\dfrac{\langle k^{(0)}|\hat H'|n^{(0)}\rangle}{E_n^{(0)}-E_k^{(0)}}|k^{(0)}\rangle$。

📖 時間依存摂動論と Fermi の黄金律

初期状態 $|i\rangle$ から終状態 $|f\rangle$ への遷移率:

$$\Gamma_{i\to f} = \frac{2\pi}{\hbar}|\langle f|\hat H'|i\rangle|^2 \rho(E_f)$$

$\rho(E_f)$: 終状態の状態密度。

✏️ 練習問題
  1. 無限深さ井戸型ポテンシャル（$0\le x\le a$ で $V=0$，他で $V=\infty$）の固有値・固有関数を求めよ。
  2. 調和振動子の $\langle x\rangle$，$\langle p\rangle$，$\langle x^2\rangle$，$\langle p^2\rangle$ を昇降演算子で計算せよ。
  3. $\hat H_0 = \hbar\omega(\hat a^\dagger\hat a + 1/2)$ に摂動 $\hat H' = \lambda\hat x^2$ を加えたときの一次・二次エネルギー補正を求めよ。
  4. スピン1/2系が磁場 $\mathbf{B} = B\hat z$ 中に置かれたとき，Hamiltonian を書き，時間発展を求めよ（Larmor 歳差）。
解答例を見る
  1. 井戸内部（$0\le x\le a$）では $-\dfrac{\hbar^2}{2m}\psi''=E\psi$。$k=\sqrt{2mE}/\hbar$ とおくと $\psi=A\sin kx+B\cos kx$。境界条件 $\psi(0)=0$ より $B=0$，$\psi(a)=0$ より $ka=n\pi$（$n=1,2,\ldots$）。よって $$E_n=\frac{\hbar^2 k_n^2}{2m}=\frac{n^2\pi^2\hbar^2}{2ma^2},\quad \psi_n(x)=\sqrt{\frac{2}{a}}\sin\!\frac{n\pi x}{a}.$$ 規格化定数は $\int_0^a|\psi_n|^2dx=1$ から $A=\sqrt{2/a}$。
  2. $\hat x=\sqrt{\dfrac{\hbar}{2m\omega}}(\hat a+\hat a^\dagger)$，$\hat p=i\sqrt{\dfrac{\hbar m\omega}{2}}(\hat a^\dagger-\hat a)$ を用いる。固有状態 $|n\rangle$ で $\hat a|n\rangle=\sqrt n|n-1\rangle$，$\hat a^\dagger|n\rangle=\sqrt{n+1}|n+1\rangle$ より $$\langle x\rangle=0,\qquad \langle p\rangle=0.$$ $\hat a\hat a^\dagger=\hat a^\dagger\hat a+1$ と $\langle n|\hat a^\dagger\hat a|n\rangle=n$ を使うと $$\langle x^2\rangle=\frac{\hbar}{2m\omega}\langle(\hat a+\hat a^\dagger)^2\rangle=\frac{\hbar}{2m\omega}(2n+1),$$ $$\langle p^2\rangle=\frac{\hbar m\omega}{2}\langle(\hat a^\dagger-\hat a)^2\rangle\cdot(-1)=\frac{\hbar m\omega}{2}(2n+1).$$ よって $\Delta x\,\Delta p=\dfrac{\hbar}{2}(2n+1)$，基底状態 $n=0$ で最小 $\hbar/2$。
  3. 一次補正は $\hat x^2=\dfrac{\hbar}{2m\omega}(\hat a+\hat a^\dagger)^2$ の対角成分から $$E_n^{(1)}=\lambda\langle n|\hat x^2|n\rangle=\frac{\lambda\hbar}{2m\omega}(2n+1).$$ 非対角成分は $\langle n\pm2|\hat x^2|n\rangle$ のみ非零で，$\langle n+2|\hat x^2|n\rangle=\dfrac{\hbar}{2m\omega}\sqrt{(n+1)(n+2)}$，$\langle n-2|\hat x^2|n\rangle=\dfrac{\hbar}{2m\omega}\sqrt{n(n-1)}$。エネルギー分母 $E_n^{(0)}-E_{n\pm2}^{(0)}=\mp2\hbar\omega$ を代入すると $$E_n^{(2)}=\lambda^2\!\left(\frac{\hbar}{2m\omega}\right)^{\!2}\!\left[\frac{(n+1)(n+2)}{-2\hbar\omega}+\frac{n(n-1)}{2\hbar\omega}\right]=-\frac{\lambda^2\hbar(2n+1)}{4m^2\omega^3}.$$ （厳密には $\hat H'$ は実効振動数を $\omega\to\sqrt{\omega^2+2\lambda/m}$ にずらす。）
  4. 磁気モーメント $\boldsymbol\mu=\gamma\hat{\mathbf S}$ より $\hat H=-\boldsymbol\mu\cdot\mathbf B=-\gamma B\,\hat S_z=-\dfrac{\gamma B\hbar}{2}\sigma_z$。固有状態 $|\!\uparrow\rangle,|\!\downarrow\rangle$ のエネルギーは $\mp\dfrac{\gamma B\hbar}{2}$。Larmor 振動数を $\omega_0=\gamma B$ とおき初期状態 $|\psi(0)\rangle=\cos\frac\theta2|\!\uparrow\rangle+\sin\frac\theta2|\!\downarrow\rangle$ とすると $$|\psi(t)\rangle=\cos\tfrac\theta2\,e^{i\omega_0 t/2}|\!\uparrow\rangle+\sin\tfrac\theta2\,e^{-i\omega_0 t/2}|\!\downarrow\rangle.$$ 期待値は $\langle\hat S_z\rangle=\dfrac\hbar2\cos\theta$（一定），$\langle\hat S_x\rangle=\dfrac\hbar2\sin\theta\cos\omega_0 t$，$\langle\hat S_y\rangle=\dfrac\hbar2\sin\theta\sin\omega_0 t$。すなわちスピンは $z$ 軸まわりに角振動数 $\omega_0$ で歳差運動（Larmor 歳差）する。
