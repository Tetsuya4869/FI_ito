💡 前提知識

微分積分の基礎（速度＝位置の微分）と ベクトル。この先は 古典力学 へ。

1. 運動の記述

📖 位置・速度・加速度

位置 $\mathbf{r}(t)$ を時間で微分すると速度、もう一度微分すると加速度:

$$\mathbf{v} = \frac{d\mathbf{r}}{dt}, \qquad \mathbf{a} = \frac{d\mathbf{v}}{dt} = \frac{d^2\mathbf{r}}{dt^2}$$

等加速度直線運動: $v = v_0 + at$、$x = x_0 + v_0 t + \frac12 at^2$。

2. 運動の3法則

✓ ニュートンの運動の3法則
  1. 慣性の法則: 力が働かなければ物体は静止または等速直線運動を続ける。
  2. 運動方程式: $\mathbf{F} = m\mathbf{a}$（力＝質量×加速度）。
  3. 作用・反作用: $A$ が $B$ に力 $\mathbf{F}$ を及ぼすと、$B$ は $A$ に $-\mathbf{F}$ を及ぼす。
📝 例 — 自由落下

重力のみ。$F=mg$ より $a=g\approx 9.8\,\text{m/s}^2$。$t$ 秒後の速度 $v=gt$、落下距離 $x=\frac12 gt^2$。

4. 仕事とエネルギー

📖 仕事・運動エネルギー・位置エネルギー

仕事: $W = \mathbf{F}\cdot\mathbf{d} = Fd\cos\theta$。

運動エネルギー: $K = \frac12 mv^2$。仕事＝運動エネルギーの変化（仕事とエネルギーの定理）。

位置エネルギー: 重力 $U=mgh$、ばね $U=\frac12 kx^2$。

✓ 力学的エネルギー保存則

保存力のみが働くとき、運動エネルギーと位置エネルギーの和は一定:

$$E = K + U = \frac12 mv^2 + U = \text{一定}$$

5. 運動量と力積

📖 運動量・力積

運動量: $\mathbf{p}=m\mathbf{v}$。運動方程式は $\mathbf{F}=\dfrac{d\mathbf{p}}{dt}$ とも書ける。

力積: $\displaystyle\int \mathbf{F}\,dt = \Delta\mathbf{p}$（力積＝運動量の変化）。

✓ 運動量保存則

外力が働かない系では全運動量が保存。衝突問題の基本道具。

$$m_1\mathbf{v}_1 + m_2\mathbf{v}_2 = m_1\mathbf{v}_1' + m_2\mathbf{v}_2'$$

6. 円運動と万有引力

📖 等速円運動

半径 $r$、速さ $v$ の円運動には中心向きの加速度（向心加速度）が必要:

$$a = \frac{v^2}{r} = r\omega^2, \quad F = \frac{mv^2}{r}$$

$\omega=v/r$ は角速度。

📝 例 — 惑星の軌道

万有引力が向心力になる: $\dfrac{GMm}{r^2}=\dfrac{mv^2}{r}$。これを解くと軌道速度 $v=\sqrt{GM/r}$。ケプラーの法則の出発点。

7. 次のステップ

💡 ここから先へ

同じ運動を「最小作用の原理」やエネルギー（ラグランジアン・ハミルトニアン）で捉え直すのが 古典力学。電磁気の力は 電磁気学、高速・強重力では 相対性理論、ミクロな世界は 量子力学 へ。
