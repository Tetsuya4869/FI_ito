1. ODE の基礎

✓ 定理 — 存在一意性（Picard-Lindelöf）

$f: U \subset \mathbb{R} \times \mathbb{R}^n \to \mathbb{R}^n$ が連続かつ $x$ について局所 Lipschitz，$(t_0, x_0) \in U$ のとき，IVP

$$\dot x = f(t,x), \quad x(t_0) = x_0$$

は $t_0$ のある近傍で一意の解をもつ。解は $U$ の境界まで延長できる（最大解）。

✓ 補題 — Gronwall の不等式

$u(t) \le \alpha + \int_{t_0}^t \beta(s) u(s)\,ds$ ならば，$u(t) \le \alpha \exp\!\left(\int_{t_0}^t \beta(s)\,ds\right)$。

ODE の解の一意性証明・安定性解析に頻用される。

📖 定義 — フロー

自律系 $\dot x = f(x)$ の解を $\varphi_t(x_0)$（時刻 $t$ における $x_0$ 出発の解の値）と書く。$\varphi_0 = \mathrm{id}$，$\varphi_{t+s} = \varphi_t \circ \varphi_s$（群の性質）。

2. 位相平面・平衡点

📖 定義 — 平衡点・極限集合

$f(x^*) = 0$（$\dot x = 0$）となる $x^*$ を平衡点（固定点）という。

$\omega$-極限集合 $\omega(x_0) = \{y \mid \exists t_n \to \infty,\; \varphi_{t_n}(x_0) \to y\}$。

✓ 定理 — Poincaré-Bendixson（2次元）

$\mathbb{R}^2$ のコンパクト正不変集合 $K$ に $\omega(x_0) \subset K$ で，$\omega(x_0)$ が平衡点を含まなければ，$\omega(x_0)$ は閉軌道（極限サイクル）。

系: 2次元自律系でカオスは起きない（位相平面は単純な挙動しかとれない）。

3. 線形自律系の分類

$\dot x = Ax$（$A \in M_2(\mathbb{R})$）の平衡点 $x^*=0$ の分類（固有値 $\lambda_{1,2}$）:

固有値の型挙動安定性
$\lambda_1 < \lambda_2 < 0$安定結節点漸近安定
$0 < \lambda_1 < \lambda_2$不安定結節点不安定
$\lambda_1 < 0 < \lambda_2$鞍点不安定
$\lambda = \alpha \pm \beta i$，$\alpha < 0$安定渦状点漸近安定
$\lambda = \pm\beta i$（純虚数）中心（閉軌道）Lyapunov 安定
$\lambda = \alpha \pm \beta i$，$\alpha > 0$不安定渦状点不安定
📝 行列指数関数

$\dot x = Ax$ の解は $x(t) = e^{tA} x_0$，$e^{tA} = \sum_{k=0}^\infty \frac{(tA)^k}{k!}$。

$A$ が対角化可能（$A = P\Lambda P^{-1}$）なら $e^{tA} = Pe^{t\Lambda}P^{-1}$。Jordan 標準形の場合は Jordan ブロックごとに計算。

4. Lyapunov 安定性

📖 定義 — 安定性の種類

平衡点 $x^* = 0$ が：

  • Lyapunov 安定: $\forall \varepsilon > 0,\; \exists \delta > 0,\; |x_0| < \delta \implies |\varphi_t(x_0)| < \varepsilon$（$t \ge 0$）
  • 漸近安定: Lyapunov 安定かつ $|x_0| < \delta \implies \varphi_t(x_0) \to 0$（$t \to \infty$）
  • 大域漸近安定: すべての初期値から $\to 0$
✓ 定理 — Lyapunov の直接法

連続関数 $V: U \to \mathbb{R}$（$V(0)=0$，$V(x)>0$（$x \ne 0$））が

  • $\dot V(x) = \nabla V(x) \cdot f(x) \le 0$（$\forall x \in U$）$\implies$ $x^*=0$ は Lyapunov 安定
  • $\dot V(x) < 0$（$x \ne 0$）$\implies$ 漸近安定

$V$ をLyapunov 関数という（「エネルギー」の一般化）。

📝 例 — 振り子方程式

$\ddot\theta + \sin\theta = 0$（$\theta \in (-\pi,\pi)$）。$V = \frac{1}{2}\dot\theta^2 + (1-\cos\theta)$ とおくと $\dot V = \dot\theta\ddot\theta + \dot\theta\sin\theta = 0$。Lyapunov 安定（だが漸近安定でない）。

5. 安定多様体定理

✓ 定理 — 安定多様体定理

双曲型平衡点 $x^*$（線形化 $Df(x^*)$ の固有値がすべて実部 $\ne 0$）のとき，

  • 安定多様体 $W^s(x^*) = \{x \mid \varphi_t(x) \to x^*,\; t\to+\infty\}$: $\dim = $ 負実部固有値の個数
  • 不安定多様体 $W^u(x^*) = \{x \mid \varphi_t(x) \to x^*,\; t\to-\infty\}$: $\dim = $ 正実部固有値の個数

局所的に $W^s, W^u$ は $x^*$ での線形化の固有空間に接する $C^\infty$ 部分多様体。

6. 分岐理論

📖 主要な局所分岐
分岐名標準形内容
鞍点-結節点 (saddle-node)$\dot x = \mu - x^2$$\mu=0$ で平衡点が消滅・出現
転写点 (transcritical)$\dot x = \mu x - x^2$平衡点の安定性が入れ替わる
ピッチフォーク$\dot x = \mu x - x^3$対称破れ，1→3 個の平衡点
Hopf 分岐（複素固有値）平衡点から極限サイクルが生まれる
📝 例 — Hopf 分岐

$\dot r = \mu r - r^3$，$\dot\theta = \omega$（極座標）。$\mu < 0$: 原点は安定渦状点。$\mu > 0$: 不安定渦状点 + 安定極限サイクル $r = \sqrt\mu$。$\mu = 0$ が Hopf 分岐点。

7. カオスと Lorenz 系

📖 定義 — Lorenz 方程式
$$\dot x = \sigma(y - x),\quad \dot y = x(\rho - z) - y,\quad \dot z = xy - \beta z$$

Lorenz が大気対流のモデルとして導出（$\sigma=10$，$\rho=28$，$\beta=8/3$ が典型）。決定論的だが初期値鋭敏依存（バタフライ効果）。

📖 定義 — カオスの特徴
  • 初期値鋭敏依存性: Lyapunov 指数 $\lambda_1 > 0$（隣接軌道が指数的に離れる）
  • 位相的推移性: 軌道が位相空間を稠密に走る
  • 周期軌道の稠密性: 周期軌道が稠密に存在
  • ストレンジアトラクター: 非整数次元のフラクタル構造をもつ吸引子
💡 Lyapunov 指数
$$\lambda = \lim_{t\to\infty} \frac{1}{t}\ln\frac{|\delta x(t)|}{|\delta x(0)|}$$

最大 Lyapunov 指数 $\lambda_1 > 0$ がカオスの定量的指標。Lorenz 系では $\lambda_1 \approx 0.9$。

✏️ 練習問題
  1. $\dot x = x - x^3$ の平衡点を求め，各平衡点の安定性を調べよ。
  2. $V(x,y) = x^2 + y^2$ が $\dot x = -x + xy^2$，$\dot y = -y - x^2 y$ の Lyapunov 関数であることを確認せよ。
  3. $\dot x = \mu x + x^3$ のピッチフォーク分岐を調べ，$\mu$ の各範囲での相図を描け。
  4. Lorenz 系でスカラー量 $V = x^2 + y^2 + (z-\rho-\sigma)^2$ が $\dot V = -2(\sigma x^2 + y^2 + \beta z(z-(\rho+\sigma)/2)) + \cdots$ 程度の形になることを計算し，解が有界であることを説明せよ。
