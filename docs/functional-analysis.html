1. ノルム空間とバナッハ空間

📖 定義 — ノルム空間・バナッハ空間

$\|\cdot\|: X \to [0,\infty)$ がノルムであるとは：(1) $\|x\|=0 \iff x=0$，(2) $\|\alpha x\|=|\alpha|\|x\|$，(3) $\|x+y\| \le \|x\|+\|y\|$（三角不等式）。

ノルム空間が（ノルムから誘導される距離について）完備なときバナッハ空間という。

📝 主要なバナッハ空間
  • $L^p(\Omega)$（$1 \le p \le \infty$）: $\|f\|_p = \left(\int |f|^p\right)^{1/p}$（$p < \infty$），$\|f\|_\infty = \mathrm{ess\,sup}|f|$
  • $C(K)$（$K$ コンパクト）: $\|f\|_\infty = \max_K|f|$
  • $\ell^p = \{(x_n) \mid \sum |x_n|^p < \infty\}$
  • Sobolev 空間 $W^{k,p}(\Omega)$
✓ Hölder の不等式
$$\int |fg|\,d\mu \le \|f\|_p \|g\|_q, \quad \frac{1}{p} + \frac{1}{q} = 1 \quad (1 \le p \le \infty)$$
✓ Minkowski の不等式
$$\|f + g\|_p \le \|f\|_p + \|g\|_p \quad (1 \le p \le \infty)$$

2. ヒルベルト空間

📖 定義 — ヒルベルト空間

内積 $\langle \cdot, \cdot \rangle$ をもち，$\|x\| = \sqrt{\langle x,x\rangle}$ で定まるノルムについて完備なときヒルベルト空間という。$L^2(\Omega)$ はその典型例。

✓ 定理 — Cauchy-Schwarz の不等式
$$|\langle x, y\rangle| \le \|x\|\|y\|$$
✓ 定理 — 射影定理

$H$ がヒルベルト空間，$M \subset H$ が閉部分空間のとき，任意の $x \in H$ に対して $x = Px + (x-Px)$ と一意に分解できる（$Px \in M$，$x-Px \perp M$）。$P: H \to M$ を直交射影という。

✓ 定理 — Riesz の表現定理（ヒルベルト空間版）

$H$ 上の有界線形汎関数 $f: H \to \mathbb{F}$ は，ある唯一の $y \in H$ によって $f(x) = \langle x, y\rangle$ と表せる。特に $H^* \cong H$（自己双対）。

✓ 定理 — 完全正規直交系

可分ヒルベルト空間は完全正規直交基底 $\{e_n\}$ をもち（Parseval の等式）：

$$x = \sum_{n=1}^\infty \langle x, e_n\rangle e_n, \quad \|x\|^2 = \sum_{n=1}^\infty |\langle x, e_n\rangle|^2$$

3. 有界線形作用素

📖 定義 — 有界線形作用素

線形写像 $T: X \to Y$（$X, Y$ バナッハ空間）が有界であるとは，

$$\|T\| = \sup_{\|x\|\le 1} \|Tx\| < \infty$$

有界 $\iff$ 連続（線形写像では同値）。$\mathcal{B}(X,Y)$ で有界作用素全体を表す。

📝 作用素の例
  • 微分作用素 $D = d/dx$: $C^1[0,1] \to C[0,1]$（非有界）
  • 積分作用素 $T f(x) = \int_0^1 K(x,y)f(y)dy$: $L^2 \to L^2$（核 $K \in L^2$ のとき有界）
  • 乗算作用素 $M_\phi f = \phi f$: $L^2 \to L^2$（$\phi \in L^\infty$ のとき $\|M_\phi\| = \|\phi\|_\infty$）
  • シフト作用素 $S(x_1,x_2,\ldots) = (0,x_1,x_2,\ldots)$ on $\ell^2$

4. 三大定理

✓ 定理 — 一様有界定理（Banach-Steinhaus）

$X$ がバナッハ空間，$\{T_\alpha\} \subset \mathcal{B}(X,Y)$ が各点有界（$\sup_\alpha \|T_\alpha x\| < \infty$ for all $x$）ならば，一様有界（$\sup_\alpha \|T_\alpha\| < \infty$）。

✓ 定理 — 開写像定理

$T: X \to Y$ が全射有界線形作用素でバナッハ空間間なら，$T$ は開写像（開集合の像が開集合）。

系 — 逆写像定理: $T$ が全単射なら $T^{-1}$ も有界（自動的に連続）。

✓ 定理 — 閉グラフ定理

$T: X \to Y$（バナッハ空間間）が線形で，グラフ $\{(x,Tx)\} \subset X \times Y$ が閉集合ならば，$T$ は有界。

✓ 定理 — Hahn-Banach

部分空間 $M \subset X$ 上定義された有界線形汎関数 $f$ は，ノルムを保ちながら $X$ 全体に拡張できる：

$$\exists \tilde f \in X^*,\;\; \tilde f|_M = f,\;\; \|\tilde f\|_{X^*} = \|f\|_{M^*}$$

系: ノルム空間で $x \ne 0$ ならば $f(x) = \|x\|$，$\|f\|=1$ な汎関数が存在（点の分離）。

5. 双対空間と弱収束

📖 定義 — 双対空間

$X^* = \mathcal{B}(X, \mathbb{F})$（有界線形汎関数の空間）を双対空間という。

空間 $X$双対 $X^*$
$L^p(\Omega)$（$1 \le p < \infty$）$L^q(\Omega)$（$1/p+1/q=1$）
$\ell^p$（$1 \le p < \infty$）$\ell^q$
$C(K)$正則 Borel 測度の空間（Riesz 表現）
ヒルベルト空間 $H$$H$（自己双対）
📖 定義 — 弱収束

$x_n \rightharpoonup x$（弱収束）とは，$\forall f \in X^*,\; f(x_n) \to f(x)$。強収束 $\|x_n - x\| \to 0$ より弱い。

✓ 定理 — Banach-Alaoglu

バナッハ空間 $X$ の単位球 $\{x \mid \|x\| \le 1\}$ は弱* 位相でコンパクト。

系: 反射的バナッハ空間（$X^{**} = X$）では有界列は弱収束部分列をもつ。

6. スペクトル理論

📖 定義 — スペクトルと固有値

$T \in \mathcal{B}(X)$。$\lambda \in \mathbb{C}$ がレゾルベント集合に属するとは，$(\lambda I - T)^{-1} \in \mathcal{B}(X)$ が存在すること。

スペクトル $\sigma(T) = \mathbb{C} \setminus \rho(T)$（$\rho(T)$ はレゾルベント集合）。有限次元では固有値の集合。

✓ スペクトルの基本性質
  • $\sigma(T) \subset \{|\lambda| \le \|T\|\}$（コンパクト集合）
  • スペクトル半径 $r(T) = \sup_{\lambda \in \sigma(T)}|\lambda| = \lim_{n\to\infty}\|T^n\|^{1/n}$
✓ 定理 — コンパクト自己共役作用素のスペクトル定理

$T = T^*$ がヒルベルト空間 $H$ 上のコンパクト自己共役作用素ならば：

  • 固有値は全て実数
  • 異なる固有値の固有空間は直交
  • $H = \bigoplus_n \ker(\lambda_n I - T)$（固有空間の直交和）
  • $Tx = \sum_n \lambda_n \langle x, e_n\rangle e_n$（$\{e_n\}$ は正規直交固有ベクトル系）
✏️ 練習問題
  1. $L^p[0,1]$（$1 \le p < \infty$）は可分なバナッハ空間であることを示せ。
  2. $H = L^2[0,1]$，$T f(x) = \int_0^x f(t)dt$ のノルム $\|T\|$ を求めよ。
  3. $\ell^1$ の双対が $\ell^\infty$ であることを示せ（Hölder の不等式を使う）。
  4. 閉グラフ定理を使って，$T: C^1[0,1] \to C[0,1]$，$Tf = f'$ が非有界であることを示せ（$\|T\| = \infty$ を直接示してもよい）。
