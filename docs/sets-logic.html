1. 集合

📖 定義 — 集合と要素

ものの集まりを集合、その個々を要素という。$x\in A$（$x$ は $A$ の要素）、$x\notin A$（要素でない）。

表し方: $A=\{1,2,3\}$（列挙）、$B=\{x \mid x>0\}$（条件）。代表的な数の集合:

$\mathbb{N}$（自然数）$\subset \mathbb{Z}$（整数）$\subset \mathbb{Q}$（有理数）$\subset \mathbb{R}$（実数）$\subset \mathbb{C}$（複素数）。

📖 部分集合

$A$ のすべての要素が $B$ に属するとき $A\subseteq B$（$A$ は $B$ の部分集合）。空集合 $\varnothing$ はすべての集合の部分集合。

2. 集合の演算

📖 和・共通部分・差・補
演算記号意味
和集合$A\cup B$$A$ または $B$ に属する
共通部分$A\cap B$$A$ かつ $B$ に属する
差集合$A\setminus B$$A$ に属し $B$ に属さない
補集合$A^c$全体集合のうち $A$ 以外

ド・モルガンの法則: $(A\cup B)^c = A^c\cap B^c$、$(A\cap B)^c = A^c\cup B^c$。

3. 論理（命題と結合子）

📖 命題と結合子

真偽が定まる文を命題という。結合子:

記号読み意味
$\lnot P$でない否定
$P \land Q$かつ両方真のとき真
$P \lor Q$または少なくとも一方が真
$P \Rightarrow Q$ならば$P$ が真で $Q$ が偽のときだけ偽
$P \Leftrightarrow Q$同値真偽が一致
💡 対偶

$P\Rightarrow Q$ と、その対偶 $\lnot Q\Rightarrow\lnot P$ は常に同値。証明で「直接示しにくいときは対偶を示す」のが定石。

4. 量化子 ∀ と ∃

📖 全称・存在

$\forall x$（すべての $x$ について）、$\exists x$（ある $x$ が存在して）。

例: 「$\forall \varepsilon>0,\ \exists \delta>0$ s.t. …」は解析学の ε-δ 論法の骨格。

否定のルール: $\lnot(\forall x\, P(x)) \equiv \exists x\,\lnot P(x)$、$\lnot(\exists x\,P(x))\equiv \forall x\,\lnot P(x)$。

5. 写像（関数）

📖 定義 — 写像

$f:A\to B$ は、$A$ の各要素に $B$ の要素をひとつ対応させる規則。

  • 単射: 異なる入力は異なる出力（$f(x)=f(y)\Rightarrow x=y$）
  • 全射: $B$ のどの要素も誰かの出力になっている
  • 全単射: 単射かつ全射（逆写像 $f^{-1}$ が存在）

6. 証明の型

✓ よく使う証明法
  • 直接証明: 仮定から定義・既知の事実を積んで結論へ。
  • 対偶法: $P\Rightarrow Q$ の代わりに $\lnot Q\Rightarrow\lnot P$ を示す。
  • 背理法: 結論を否定すると矛盾が出ることを示す。
  • 数学的帰納法: $n=1$ で成立、$n=k$ を仮定して $n=k+1$ を示す。
📝 例 — 背理法（$\sqrt2$ は無理数）

$\sqrt2=p/q$（既約分数）と仮定。$2q^2=p^2$ より $p$ は偶数 $p=2k$。すると $q^2=2k^2$ で $q$ も偶数。これは既約に矛盾。よって $\sqrt2$ は無理数。$\square$

7. 次のステップ

💡 ここから先へ

これらの記号と証明法は全ページの土台です。続けて 微分積分の基礎 → 解析学 へ。集合論を深めると 位相空間論 や 測度論 につながります。

✏️ 練習問題
  1. $A=\{1,2,3,4\}$、$B=\{3,4,5\}$ について $A\cup B$、$A\cap B$、$A\setminus B$ を求めよ。
  2. 「$n^2$ が偶数 $\Rightarrow$ $n$ は偶数」を対偶法で証明せよ。
  3. $\lnot(\forall x>0,\ x^2>x)$ を量化子を使って書き換え、真偽を判定せよ。
  4. $f(x)=2x+1$（$\mathbb{R}\to\mathbb{R}$）が全単射であることを示し、逆写像を求めよ。
  5. 数学的帰納法で $1+2+\cdots+n = \dfrac{n(n+1)}{2}$ を証明せよ。
解答例を見る
  1. 要素を直接比較する。$A\cup B$ は $A$ または $B$ に属する要素全体なので $A\cup B=\{1,2,3,4,5\}$。$A\cap B$ は両方に属する要素なので $A\cap B=\{3,4\}$。$A\setminus B$ は $A$ に属し $B$ に属さない要素なので $A\setminus B=\{1,2\}$。
  2. 命題「$n^2$ が偶数 $\Rightarrow n$ は偶数」の対偶は「$n$ が奇数 $\Rightarrow n^2$ は奇数」。$n$ が奇数なら $n=2k+1$（$k\in\mathbb{Z}$）と書ける。このとき $n^2=(2k+1)^2=4k^2+4k+1=2(2k^2+2k)+1$ となり、$2k^2+2k\in\mathbb{Z}$ だから $n^2$ は奇数。よって対偶が成り立ち、元の命題も成り立つ。$\square$
  3. 量化子の否定ルール $\lnot(\forall x\,P(x))\equiv\exists x\,\lnot P(x)$ を適用すると、$\lnot(\forall x>0,\ x^2>x)\equiv \exists x>0,\ x^2\le x$。真偽: たとえば $x=\tfrac12>0$ のとき $x^2=\tfrac14\le\tfrac12=x$ が成り立つので、この存在命題は真（したがって元の全称命題は偽）。
  4. $f(x)=2x+1$。
    単射: $f(x)=f(y)$ とすると $2x+1=2y+1$ より $2x=2y$、ゆえに $x=y$。よって単射。
    全射: 任意の $y\in\mathbb{R}$ に対し $x=\dfrac{y-1}{2}\in\mathbb{R}$ をとれば $f(x)=2\cdot\dfrac{y-1}{2}+1=y$。よって全射。
    以上より $f$ は全単射。逆写像は $y=2x+1$ を $x$ について解いて $x=\dfrac{y-1}{2}$、すなわち $f^{-1}(x)=\dfrac{x-1}{2}$。
  5. $S(n):\ 1+2+\cdots+n=\dfrac{n(n+1)}{2}$ とおく。
    （基底）$n=1$ のとき左辺 $=1$、右辺 $=\dfrac{1\cdot2}{2}=1$ で一致。
    （帰納）$n=k$ で $S(k)$ が成り立つと仮定すると $1+2+\cdots+k=\dfrac{k(k+1)}{2}$。両辺に $k+1$ を加えると
    $1+2+\cdots+k+(k+1)=\dfrac{k(k+1)}{2}+(k+1)=\dfrac{k(k+1)+2(k+1)}{2}=\dfrac{(k+1)(k+2)}{2}$。
    これは $S(k+1)$ そのもの。よって数学的帰納法によりすべての自然数 $n$ で $S(n)$ が成り立つ。$\square$
