1. 複素数と複素平面

📖 定義 — 複素数

$z = x + iy$（$x,y \in \mathbb{R}$，$i^2 = -1$）。$\text{Re}(z) = x$，$\text{Im}(z) = y$，$|z| = \sqrt{x^2+y^2}$，$\bar z = x - iy$。

極形式: $z = r e^{i\theta} = r(\cos\theta + i\sin\theta)$，$r = |z|$，$\theta = \arg z$。

✓ Euler の公式
$$e^{i\theta} = \cos\theta + i\sin\theta, \quad e^{i\pi} + 1 = 0$$

2. 正則関数と C-R 方程式

📖 定義 — 正則関数 (Holomorphic)

$f: \Omega \to \mathbb{C}$（$\Omega \subset \mathbb{C}$ 開集合）が $z_0 \in \Omega$ で複素微分可能であるとは，

$$f'(z_0) = \lim_{h \to 0} \frac{f(z_0+h) - f(z_0)}{h} \quad (h \in \mathbb{C})$$

が存在すること。$\Omega$ の全点で複素微分可能なとき，$f$ は $\Omega$ 上で正則（holomorphic）という。

✓ 定理 — Cauchy-Riemann 方程式

$f(x+iy) = u(x,y) + iv(x,y)$ が $z_0$ で複素微分可能 $\iff$ $u,v$ が $z_0$ で全微分可能かつ，

$$\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}$$
✓ 定理 — 正則関数は調和関数を生む

$f = u+iv$ が正則 $\implies$ $u,v$ はどちらも調和関数: $\Delta u = \Delta v = 0$。

📝 例
  • $f(z) = z^n$，$e^z$，$\sin z$，$\cos z$，$\ln z$（分枝切断を除いた領域で）は正則。
  • $f(z) = \bar z$，$|z|$ は正則でない（C-R 方程式を満たさない）。

3. Cauchy 積分定理と積分公式

✓ 定理 — Cauchy 積分定理

$f$ が単連結領域 $\Omega$ 上で正則ならば，$\Omega$ 内の任意の閉曲線 $\gamma$ に対して，

$$\oint_\gamma f(z)\,dz = 0$$
✓ 定理 — Cauchy 積分公式

$f$ が $\Omega$ 上正則，$C$ が $\Omega$ 内の（反時計回りの）単純閉曲線，$z_0$ が $C$ の内部の点ならば，

$$f(z_0) = \frac{1}{2\pi i} \oint_C \frac{f(z)}{z - z_0}\,dz$$

より一般に，$n$ 階導関数についても，

$$f^{(n)}(z_0) = \frac{n!}{2\pi i} \oint_C \frac{f(z)}{(z-z_0)^{n+1}}\,dz$$
✓ 系 — 正則関数は無限回微分可能

正則関数は（実解析とは違い）自動的に無限回微分可能であり，各 $f^{(n)}$ も正則。

4. Taylor・Laurent 展開

✓ 定理 — Taylor 展開

$f$ が $|z-z_0| < R$ で正則ならば，この円板内で収束する冪級数展開をもつ：

$$f(z) = \sum_{n=0}^\infty c_n (z-z_0)^n, \quad c_n = \frac{f^{(n)}(z_0)}{n!}$$
📖 定義 — Laurent 展開と特異点

$0 < |z-z_0| < R$ において，

$$f(z) = \sum_{n=-\infty}^\infty c_n (z-z_0)^n$$

$z_0$ が孤立特異点のとき：

  • $c_n = 0$（$n < 0$）: 除去可能特異点
  • 有限個の $c_n \ne 0$（$n < 0$）: 極（最大次数 $m$ を位数という）
  • 無限個の $c_n \ne 0$（$n < 0$）: 真性特異点

5. 留数定理と実積分への応用

📖 定義 — 留数 (Residue)

孤立特異点 $z_0$ での留数を

$$\mathrm{Res}(f, z_0) = c_{-1}$$

（Laurent 展開の $-1$ 次の係数）と定義する。$m$ 位の極では，

$$\mathrm{Res}(f, z_0) = \frac{1}{(m-1)!}\lim_{z\to z_0} \frac{d^{m-1}}{dz^{m-1}}\left[(z-z_0)^m f(z)\right]$$
✓ 定理 — 留数定理

$f$ が閉曲線 $C$ の内部で有限個の孤立特異点 $z_1,\ldots,z_k$ 以外で正則ならば，

$$\oint_C f(z)\,dz = 2\pi i \sum_{j=1}^k \mathrm{Res}(f, z_j)$$
📝 例 — 実積分への応用

$\displaystyle\int_{-\infty}^\infty \frac{1}{1+x^2}\,dx$

$f(z) = \frac{1}{1+z^2} = \frac{1}{(z-i)(z+i)}$。上半円弧 $C_R$ を使い，$R\to\infty$。上半平面の極は $z=i$。

$$\mathrm{Res}(f,i) = \frac{1}{2i} \implies \int_{-\infty}^\infty \frac{dx}{1+x^2} = 2\pi i \cdot \frac{1}{2i} = \pi$$

6. 等角写像

📖 定義 — 等角写像

正則かつ $f'(z_0) \ne 0$ な写像は $z_0$ の近傍で等角（角度と向きを保存）。

✓ 定理 — Riemann の写像定理

$\mathbb{C}$ の単連結な真部分集合 $\Omega$（$\Omega \ne \mathbb{C}$）から単位円板 $\mathbb{D} = \{|z|<1\}$ への双正則写像が存在する。

📝 重要な等角写像
  • $w = z^2$: 上半平面 → 全平面（$-x$ 軸を除く）
  • $w = e^z$: 帯領域 $\{0 < \text{Im}(z) < \pi\}$ → 上半平面
  • Möbius変換 $w = \dfrac{az+b}{cz+d}$: 円・直線を円・直線へ写す
  • Joukowski変換 $w = z + 1/z$: 翼型の設計に応用
✏️ 練習問題
  1. $f(z)=|z|^2$ が $z=0$ 以外では正則でないことを C-R 方程式を使って示せ。
  2. $\displaystyle\oint_{|z|=2} \frac{e^z}{z^2(z-1)}\,dz$ を留数定理で計算せよ。
  3. $\displaystyle\int_0^\infty \frac{\cos x}{1+x^2}\,dx$ を複素積分で求めよ。
  4. 上半平面を単位円板へ写す双正則写像を一つ構成せよ。
解答例を見る
  1. $f(z)=|z|^2=x^2+y^2$ より $u(x,y)=x^2+y^2$，$v(x,y)=0$。よって $u_x=2x,\; u_y=2y,\; v_x=v_y=0$。C-R 方程式 $u_x=v_y,\; u_y=-v_x$ は $2x=0$ かつ $2y=0$，すなわち $x=y=0$ でのみ成立する。したがって $z=0$ を除いて複素微分可能でなく，どの開集合上でも正則にならない。
  2. $\displaystyle f(z)=\frac{e^z}{z^2(z-1)}$。$|z|=2$ の内部の極は $z=0$（2位）と $z=1$（1位）。
    $z=1$（単純極）: $\displaystyle \mathrm{Res}(f,1)=\lim_{z\to1}(z-1)f(z)=\frac{e^{1}}{1^2}=e$。
    $z=0$（2位）: $\displaystyle \mathrm{Res}(f,0)=\lim_{z\to0}\frac{d}{dz}\!\left[z^2 f(z)\right]=\lim_{z\to0}\frac{d}{dz}\frac{e^z}{z-1}=\lim_{z\to0}\frac{e^z(z-1)-e^z}{(z-1)^2}=\frac{e^0(0-2)}{(-1)^2}=-2$。
    よって $\displaystyle\oint_{|z|=2}f(z)\,dz=2\pi i\,(e-2)$。
  3. 被積分関数は偶関数なので $\displaystyle\int_0^\infty\frac{\cos x}{1+x^2}\,dx=\frac12\int_{-\infty}^\infty\frac{\cos x}{1+x^2}\,dx=\frac12\,\mathrm{Re}\int_{-\infty}^\infty\frac{e^{ix}}{1+x^2}\,dx$。
    $\displaystyle g(z)=\frac{e^{iz}}{(z-i)(z+i)}$ を上半円弧で積分すると，上半平面の極は $z=i$（Jordan の補題より半円上の寄与は $R\to\infty$ で $0$）。
    $\displaystyle \mathrm{Res}(g,i)=\frac{e^{i\cdot i}}{2i}=\frac{e^{-1}}{2i}$ より $\displaystyle\int_{-\infty}^\infty\frac{e^{ix}}{1+x^2}\,dx=2\pi i\cdot\frac{e^{-1}}{2i}=\frac{\pi}{e}$（実数）。
    したがって $\displaystyle\int_0^\infty\frac{\cos x}{1+x^2}\,dx=\frac{\pi}{2e}$。
  4. Cayley 変換 $\displaystyle w=\frac{z-i}{z+i}$ が一例。実軸上の点 $x$ に対し $|w|=\dfrac{|x-i|}{|x+i|}=1$ なので実軸は単位円周へ写る。また $z=i$（上半平面の内点）は $w=0$ へ写るので，上半平面 $\{\mathrm{Im}\,z>0\}$ 全体が単位円板 $\{|w|<1\}$ へ双正則に写される。
