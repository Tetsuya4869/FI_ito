1. ホモトピーと基本群

📖 定義 — ホモトピー

連続写像 $f, g: X \to Y$ がホモトープ（$f \simeq g$）とは，連続写像 $H: X \times [0,1] \to Y$ で $H(x,0) = f(x)$，$H(x,1) = g(x)$ が存在すること。$H$ をホモトピーという。

📖 定義 — 基点付きループと基本群

基点 $x_0 \in X$ でのループ: $\gamma: [0,1] \to X$，$\gamma(0) = \gamma(1) = x_0$（基点固定ホモトープの同値類）。

基本群（第一ホモトピー群）:

$$\pi_1(X, x_0) = \{\text{基点 } x_0 \text{ でのループの同値類}\}$$

積: $[\gamma][\delta] = [\gamma \cdot \delta]$（$\gamma$ のあと $\delta$）。群演算は well-defined。

📖 定義 — 単連結

$X$ が単連結とは，弧状連結かつ $\pi_1(X) = \{1\}$（すべてのループが収縮可能）。

2. Seifert-van Kampen の定理

✓ 定理 — Seifert-van Kampen

$X = U \cup V$（$U, V, U\cap V$ が弧状連結な開集合，$x_0 \in U\cap V$）のとき，

$$\pi_1(X, x_0) \cong \pi_1(U, x_0) *_{\pi_1(U\cap V, x_0)} \pi_1(V, x_0)$$

（$\pi_1(U\cap V)$ で amalgamated free product）

📝 例 — $\pi_1(S^1) = \mathbb{Z}$ の計算

$S^1$ を上弧 $U$ と下弧 $V$（各々 $\simeq$ 開区間）に分ける。$U \cap V = $ 2点（弧状連結でないので少し工夫）。より詳しくは少しオーバーラップするように取ると: $\pi_1(U) = \pi_1(V) = 1$，$U \cap V \simeq S^0 \times [0,1]$。van Kampen から $\pi_1(S^1) \cong \mathbb{Z}$。

3. 被覆空間

📖 定義 — 被覆空間

連続全射 $p: \tilde X \to X$ が被覆写像: 各 $x \in X$ に開近傍 $U$ が存在し，$p^{-1}(U) = \bigsqcup_\alpha V_\alpha$（$V_\alpha$ は互いに素な開集合）で各 $p|_{V_\alpha}: V_\alpha \to U$ が同相。

✓ 被覆空間と基本群の対応

$X$ の被覆空間の同型類 $\leftrightarrow$ $\pi_1(X, x_0)$ の共役類の部分群（局所単連結な場合）。

  • 普遍被覆 $\tilde X$: $\pi_1(\tilde X) = 1$，$\pi_1(X) = \mathrm{Deck}(\tilde X / X)$（デック変換群）
  • $S^1$ の普遍被覆: $\mathbb{R}$（$p: \mathbb{R} \to S^1$，$p(t) = e^{2\pi it}$），$\mathrm{Deck} \cong \mathbb{Z}$

4. 単体複体とホモロジー

📖 定義 — 単体複体

$k$-単体 $\sigma = [v_0, v_1, \ldots, v_k]$（$\mathbb{R}^n$ 内の $k+1$ 個の一般位置の点の凸包）。単体複体 $K$: 単体の有限集合で，面が含まれ，交差が面になるもの。

📖 定義 — 単体ホモロジー

$k$-鎖群 $C_k(K; \mathbb{Z}) = $ $k$-単体の $\mathbb{Z}$-自由加群。境界作用素 $\partial_k: C_k \to C_{k-1}$:

$$\partial_k [v_0,\ldots,v_k] = \sum_{i=0}^k (-1)^i [v_0,\ldots,\hat v_i,\ldots,v_k]$$

$\partial_{k-1} \circ \partial_k = 0$（境界の境界 $= 0$）。

$k$-次ホモロジー群: $H_k(K) = \ker\partial_k / \mathrm{Im}\,\partial_{k+1}$（$k$-サイクル / $k$-境界）

5. Euler 標数

📖 定義 — Euler 標数

単体複体 $K$ のEuler 標数（ホモロジーによる定義）:

$$\chi(K) = \sum_{k \ge 0} (-1)^k \,\mathrm{rank}\, H_k(K; \mathbb{Z})$$

単体による定義: $\chi(K) = \sum_{k} (-1)^k c_k$（$c_k$ は $k$-単体の個数）。

$K$ が多面体の場合: $\chi = V - E + F$（頂点数 - 辺数 + 面数）。

✓ 定理 — Euler の多面体定理

凸多面体では $V - E + F = 2$。

閉曲面（種数 $g$）: $\chi = 2 - 2g$。$g=0$（球）: $\chi=2$，$g=1$（トーラス）: $\chi=0$，$g=2$（二重トーラス）: $\chi=-2$。

6. CW 複体

📖 定義 — CW 複体

$X$ を $n$-骨格 $X^0 \subset X^1 \subset \cdots$ の列として構成。$X^n$ は $X^{n-1}$ に $n$-セル（$n$-円板）を貼り付けて得る。多くの空間（多様体・代数多様体など）は CW 複体として実現できる。

📝 CW 構造の例
  • $S^n$: 0-セル 1個 + $n$-セル 1個
  • $\mathbb{CP}^n$: 各次元に 1個ずつのセル（$0, 2, 4, \ldots, 2n$）
  • $T^2$: 0-セル 1，1-セル 2，2-セル 1（$\chi = 1 - 2 + 1 = 0$ ✓）

7. 応用と計算例

✓ 定理 — Brouwer の不動点定理（ホモロジーによる証明）

連続写像 $f: D^n \to D^n$ は不動点をもつ。証明: 不動点がないと仮定すると $f$ から $D^n \to S^{n-1}$ への引き込みが構成でき，$H_{n-1}(D^n) = 0$ と $H_{n-1}(S^{n-1}) = \mathbb{Z}$ の矛盾。

✓ 定理 — Borsuk-Ulam

任意の連続写像 $f: S^n \to \mathbb{R}^n$ に対して，$f(x) = f(-x)$ となる $x \in S^n$ が存在する（「地球上には気温と気圧が対蹠点と同じ点が存在する」）。

✓ 定理 — Hairy Ball（毛球定理）

$S^{2n}$ 上には零点のない連続接ベクトル場は存在しない（「球面を覆う毛は必ず寝癖がつく」）。証明: Euler 標数 $\chi(S^{2n}) = 2 \ne 0$。

✏️ 練習問題
  1. van Kampen の定理を使って $\pi_1(T^2) = \mathbb{Z}^2$ を示せ（$T^2 = U \cup V$，$U \cap V \simeq S^1$）。
  2. 正四面体（頂点 4，辺 6，面 4）の Euler 標数を計算し，球面と同相であることを確認せよ。
  3. $H_k(\mathbb{RP}^2; \mathbb{Z})$ を境界作用素の行列を明示的に計算して求めよ。
  4. $f: S^1 \to S^1$，$f(e^{i\theta}) = e^{2i\theta}$（2倍写像）の引き戻し $f^*: H^1(S^1) \to H^1(S^1)$ を求めよ。
