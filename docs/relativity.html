1. 特殊相対論

📖 特殊相対性原理と光速不変

Einstein の 2 公準:

  1. 物理法則はすべての慣性系で同一の形をもつ（相対性原理）。
  2. 真空中の光速 $c$ はすべての慣性系で同じ（光速不変）。

これらから Galilei 変換は棄却され，Lorentz 変換が導かれる。

✓ Lorentz 変換

系 $S$ に対して速度 $v$（$x$ 方向）で動く系 $S'$ への変換 ($\beta = v/c$，$\gamma = 1/\sqrt{1-\beta^2}$):

$$ct' = \gamma(ct - \beta x), \quad x' = \gamma(x - \beta ct)$$ $$y' = y, \quad z' = z$$

逆変換: $\beta\to-\beta$。$\gamma \ge 1$，$v\to c$ で $\gamma\to\infty$。

📖 時空間隔と因果律

Minkowski 計量: $ds^2 = -c^2dt^2 + dx^2 + dy^2 + dz^2 = \eta_{\mu\nu}dx^\mu dx^\nu$。

時空間隔はすべての慣性系で不変。

$ds^2$種類因果性
$< 0$時間的 (timelike)因果的に接続可能
$= 0$光的 (null / lightlike)光信号で接続
$> 0$空間的 (spacelike)因果的に分離
📝 時間の遅れ・長さの収縮

時間の遅れ: $S'$ で静止した時計の固有時間 $\Delta\tau$，$S$ からみると $\Delta t = \gamma\Delta\tau > \Delta\tau$。

長さの収縮: $S'$ で固有長さ $L_0$ の棒，$S$ からみると $L = L_0/\gamma < L_0$（運動方向のみ）。

速度の合成: $u' = (u - v)/(1 - uv/c^2)$（$u = c$ なら $u' = c$）。

2. 4-ベクトルと相対論的力学

📖 4-ベクトルの共変性

4-位置: $x^\mu = (ct, x, y, z)$。4-速度: $u^\mu = dx^\mu/d\tau = \gamma(c, \mathbf{v})$。

4-運動量: $p^\mu = mu^\mu = (\gamma mc, \gamma m\mathbf{v}) = (E/c, \mathbf{p})$。

不変量: $p_\mu p^\mu = -(mc)^2 \Rightarrow E^2 = (pc)^2 + (mc^2)^2$。

✓ $E = mc^2$ と相対論的運動方程式

静止エネルギー: $E_0 = mc^2$。全エネルギー: $E = \gamma mc^2$。

相対論的運動方程式: $dp^\mu/d\tau = f^\mu$（4-力 $f^\mu$）。

Lorentz 力の共変形: $f^\mu = qF^{\mu\nu}u_\nu$。

3. 一般相対論の基礎

📖 等価原理と一般共変性

弱い等価原理: 慣性質量 $=$ 重力質量。重力と加速度系は局所的に区別できない。

強い等価原理: 局所慣性系では特殊相対論の物理法則が成立する。

一般共変性: 物理法則は任意の座標変換のもとで同じ形（テンソル方程式）で書ける。

📖 計量テンソルと共変微分

時空の計量: $ds^2 = g_{\mu\nu}dx^\mu dx^\nu$。$g_{\mu\nu}$（対称テンソル）が重力場を記述。

Christoffel 記号（接続係数）:

$$\Gamma^\lambda_{\mu\nu} = \frac{1}{2}g^{\lambda\sigma}(\partial_\mu g_{\nu\sigma} + \partial_\nu g_{\mu\sigma} - \partial_\sigma g_{\mu\nu})$$

測地線方程式（自由落下）: $\dfrac{d^2x^\lambda}{d\tau^2} + \Gamma^\lambda_{\mu\nu}\dfrac{dx^\mu}{d\tau}\dfrac{dx^\nu}{d\tau} = 0$。

📖 Riemann 曲率テンソル
$$R^\rho{}_{\sigma\mu\nu} = \partial_\mu\Gamma^\rho_{\nu\sigma} - \partial_\nu\Gamma^\rho_{\mu\sigma} + \Gamma^\rho_{\mu\lambda}\Gamma^\lambda_{\nu\sigma} - \Gamma^\rho_{\nu\lambda}\Gamma^\lambda_{\mu\sigma}$$

Ricci テンソル: $R_{\mu\nu} = R^\lambda{}_{\mu\lambda\nu}$。Ricci スカラー: $R = g^{\mu\nu}R_{\mu\nu}$。

Einstein テンソル: $G_{\mu\nu} = R_{\mu\nu} - \frac{1}{2}g_{\mu\nu}R$（$\nabla^\mu G_{\mu\nu}=0$ が自動的に成立）。

4. Einstein 方程式

✓ Einstein 方程式
$$G_{\mu\nu} + \Lambda g_{\mu\nu} = \frac{8\pi G}{c^4}T_{\mu\nu}$$

$G$: 重力定数，$\Lambda$: 宇宙定数，$T_{\mu\nu}$: エネルギー運動量テンソル。

弱重力場・低速極限で Newton 重力 $\nabla^2\phi = 4\pi G\rho$ に帰着する。

📖 Einstein-Hilbert 作用
$$S = \frac{c^4}{16\pi G}\int (R - 2\Lambda)\sqrt{-g}\,d^4x + S_{\rm matter}$$

$g = \det(g_{\mu\nu})$。$g_{\mu\nu}$ で変分すると Einstein 方程式が得られる。

5. Schwarzschild 解と重力波

✓ Schwarzschild 解（球対称真空解）
$$ds^2 = -\left(1 - \frac{r_s}{r}\right)c^2 dt^2 + \left(1 - \frac{r_s}{r}\right)^{-1}dr^2 + r^2 d\Omega^2$$

Schwarzschild 半径: $r_s = 2GM/c^2$（ブラックホールの事象の地平面）。

太陽: $r_s \approx 3\,\text{km}$，地球: $r_s \approx 9\,\text{mm}$。

📖 重力波

弱重力場の線形化 $g_{\mu\nu} = \eta_{\mu\nu} + h_{\mu\nu}$（$|h|\ll 1$）。Lorenz ゲージで:

$$\Box\bar h_{\mu\nu} = -\frac{16\pi G}{c^4}T_{\mu\nu}$$

真空解（TT ゲージ）は横波（2 自由度: $+$ 偏光，$\times$ 偏光）。伝播速度 $= c$。

連星ブラックホール合体からの重力波パワー: $P = -\dfrac{32G^4}{5c^5}\dfrac{m_1^2 m_2^2(m_1+m_2)}{r^5}$（四重極放射）。
