1. マルコフ決定過程

📖 定義 — マルコフ決定過程 (MDP)

マルコフ決定過程は5つ組 $(\mathcal{S}, \mathcal{A}, P, R, \gamma)$ で定義される：

  • $\mathcal{S}$：状態空間
  • $\mathcal{A}$：行動空間
  • $P(s' \mid s, a)$：遷移確率（状態 $s$ で行動 $a$ をとったとき $s'$ へ遷移する確率）
  • $R(s, a)$：報酬関数（即時報酬の期待値）
  • $\gamma \in [0, 1)$：割引率

マルコフ性: 次状態と報酬は現在の状態と行動のみに依存し，過去の履歴に依存しない。

$$\Pr(s_{t+1} \mid s_t, a_t) = \Pr(s_{t+1} \mid s_0, a_0, \ldots, s_t, a_t)$$

📖 定義 — 方策と収益

方策 $\pi(a \mid s)$ は状態 $s$ において行動 $a$ を選ぶ確率分布である（決定的方策なら $a = \pi(s)$）。

時刻 $t$ 以降の収益（割引累積報酬）は

$$G_t = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1} = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots$$

割引率 $\gamma$ は将来の報酬をどれだけ重視するかを制御し，$\gamma < 1$ は無限和の収束を保証する。

💡 強化学習の目標

環境モデル $P, R$ が未知でも，エージェントは環境との相互作用（試行錯誤）を通じて期待収益 $\mathbb{E}_\pi[G_0]$ を最大化する方策 $\pi$ を学習する。これが教師あり学習との本質的な違いである。

2. 価値関数とBellman方程式

📖 定義 — 状態価値関数と行動価値関数

方策 $\pi$ のもとでの状態価値関数:

$$V^\pi(s) = \mathbb{E}_\pi\!\left[G_t \mid S_t = s\right]$$

行動価値関数（Q関数）:

$$Q^\pi(s, a) = \mathbb{E}_\pi\!\left[G_t \mid S_t = s, A_t = a\right]$$

両者の関係: $V^\pi(s) = \sum_a \pi(a \mid s)\, Q^\pi(s, a)$。

✓ 定理 — Bellman 方程式

価値関数は再帰的関係（Bellman 期待方程式）を満たす：

$$V^\pi(s) = \sum_a \pi(a \mid s) \left[ R(s, a) + \gamma \sum_{s'} P(s' \mid s, a)\, V^\pi(s') \right]$$ $$Q^\pi(s, a) = R(s, a) + \gamma \sum_{s'} P(s' \mid s, a) \sum_{a'} \pi(a' \mid s')\, Q^\pi(s', a')$$

これは価値関数を未知数とする線形方程式系であり，一意解をもつ。

✓ 定理 — 最適方策と Bellman 最適方程式

最適価値関数 $V^*(s) = \max_\pi V^\pi(s)$，$Q^*(s,a) = \max_\pi Q^\pi(s,a)$ はBellman 最適方程式を満たす：

$$V^*(s) = \max_a \left[ R(s, a) + \gamma \sum_{s'} P(s' \mid s, a)\, V^*(s') \right]$$ $$Q^*(s, a) = R(s, a) + \gamma \sum_{s'} P(s' \mid s, a)\, \max_{a'} Q^*(s', a')$$

最適方策は貪欲に得られる: $\pi^*(s) = \arg\max_a Q^*(s, a)$。任意の MDP には決定的な最適方策が少なくとも1つ存在する。

3. 動的計画法

環境モデル $(P, R)$ が既知のとき，Bellman 方程式を用いて最適方策を計算できる。

📖 定義 — 方策評価

与えられた方策 $\pi$ の価値 $V^\pi$ を，Bellman 期待方程式を更新則として反復計算する：

$$V_{k+1}(s) \leftarrow \sum_a \pi(a \mid s) \left[ R(s, a) + \gamma \sum_{s'} P(s' \mid s, a)\, V_k(s') \right]$$

この写像は縮小写像であり，$k \to \infty$ で $V_k \to V^\pi$ に収束する。

✓ 定理 — 方策反復と価値反復

方策反復は次の2段階を交互に繰り返す：

  1. 方策評価: 現方策 $\pi$ の $V^\pi$ を計算
  2. 方策改善: $\pi'(s) = \arg\max_a \left[R(s,a) + \gamma \sum_{s'} P(s'\mid s,a) V^\pi(s')\right]$

価値反復は評価を1回で打ち切り，Bellman 最適方程式を直接反復する：

$$V_{k+1}(s) \leftarrow \max_a \left[ R(s, a) + \gamma \sum_{s'} P(s' \mid s, a)\, V_k(s') \right]$$

いずれも有限 MDP で最適方策 $\pi^*$ に収束する。

💡 縮小写像としての Bellman 作用素

Bellman 最適作用素 $T$ は上限ノルムに関して係数 $\gamma$ の縮小写像である: $\|TV - TV'\|_\infty \le \gamma \|V - V'\|_\infty$。Banach の不動点定理により，反復は唯一の不動点 $V^*$ へ幾何級数的に収束する。

4. モデルフリー学習

遷移確率 $P$ と報酬 $R$ が未知のとき，経験（サンプル）から直接学習する。

📖 定義 — モンテカルロ法とTD学習

モンテカルロ (MC) 法: エピソード完了後の実収益 $G_t$ で価値を更新する。

$$V(S_t) \leftarrow V(S_t) + \alpha \bigl[ G_t - V(S_t) \bigr]$$

TD (時間的差分) 学習: 1ステップ先のブートストラップ推定を目標に用いる。

$$V(S_t) \leftarrow V(S_t) + \alpha \underbrace{\bigl[ R_{t+1} + \gamma V(S_{t+1}) - V(S_t) \bigr]}_{\text{TD誤差 } \delta_t}$$

MC は不偏だが分散が大きく，TD は有偏だが分散が小さくオンラインで学習できる。

✓ 定理 — Q学習 (Off-policy)

Q学習は Bellman 最適方程式をサンプルで近似し，行動価値を更新する：

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \bigl[ R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t) \bigr]$$

行動選択に用いる挙動方策（例: $\varepsilon$-貪欲）と学習対象の目標方策（貪欲）が異なる方策オフ型である。適切な学習率とすべての状態行動対の無限回訪問のもとで $Q \to Q^*$ に収束する。

📝 例 — SARSA (On-policy)

SARSA は実際にとった次行動 $A_{t+1}$ を用いて更新する方策オン型手法である（名称は $(S,A,R,S',A')$ に由来）：

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \bigl[ R_{t+1} + \gamma Q(S_{t+1}, A_{t+1}) - Q(S_t, A_t) \bigr]$$

Q学習が $\max$ を用いるのに対し，SARSA は挙動方策自身の価値を学習するため，探索中のリスクをより考慮した保守的な方策を得やすい（例: 崖歩き問題）。

5. 関数近似と深層強化学習

📖 定義 — 価値関数近似

状態空間が巨大・連続の場合，価値をテーブルで表現できない。パラメータ $\theta$ をもつ関数（線形モデルやニューラルネット）で近似する：

$$Q(s, a; \theta) \approx Q^*(s, a)$$

目標 $y = R + \gamma \max_{a'} Q(s', a'; \theta)$ と予測の二乗誤差を最小化する半勾配法で $\theta$ を更新する：

$$\theta \leftarrow \theta + \alpha \bigl[ y - Q(s, a; \theta) \bigr] \nabla_\theta Q(s, a; \theta)$$
✓ 定理 — DQN (Deep Q-Network)

DQN は深層ニューラルネットで Q関数を近似し，学習を安定化する2つの工夫を導入する。損失関数は

$$L(\theta) = \mathbb{E}_{(s,a,r,s') \sim \mathcal{D}} \left[ \bigl( r + \gamma \max_{a'} Q(s', a'; \theta^-) - Q(s, a; \theta) \bigr)^2 \right]$$
  • 経験再生 (Experience Replay): 遷移 $(s,a,r,s')$ をバッファ $\mathcal{D}$ に蓄積し，ランダムサンプリングして学習。サンプル間の時間相関を断ち，データ効率を高める。
  • 目標ネットワーク (Target Network): 目標値の計算に固定パラメータ $\theta^-$ を用い，一定周期で $\theta^- \leftarrow \theta$ と同期。目標が動くことによる発散を防ぐ。
💡 「死の三徴」

関数近似・ブートストラップ・方策オフ学習の3つが揃うと学習が不安定・発散しやすい（deadly triad）。DQN の経験再生と目標ネットワークはこの不安定性を緩和する実践的な対策である。

6. 方策勾配法

📖 定義 — 方策のパラメータ化

方策自身をパラメータ $\theta$ で直接表現し $\pi_\theta(a \mid s)$，目的関数 $J(\theta) = \mathbb{E}_{\pi_\theta}[G_0]$ を勾配上昇で最大化する。連続行動空間や確率的方策を自然に扱える。

✓ 定理 — 方策勾配定理

目的関数の勾配は，遷移確率の勾配を含まない次の形で表される：

$$\nabla_\theta J(\theta) = \mathbb{E}_{\pi_\theta}\!\left[ \nabla_\theta \log \pi_\theta(a \mid s)\, Q^{\pi_\theta}(s, a) \right]$$

$Q^{\pi_\theta}$ をベースライン $b(s)$（典型的には $V(s)$）で補正したアドバンテージ $A(s,a) = Q(s,a) - V(s)$ を用いると，不偏性を保ったまま分散を低減できる。

📝 例 — REINFORCE・Actor-Critic・PPO
  • REINFORCE: モンテカルロ収益 $G_t$ を用いる基本手法。$\theta \leftarrow \theta + \alpha\, \gamma^t\, G_t\, \nabla_\theta \log \pi_\theta(A_t \mid S_t)$。分散が大きい。
  • Actor-Critic: 方策（Actor）と価値関数（Critic）を同時学習し，Critic の推定するアドバンテージで方策を更新。TD 誤差 $\delta_t$ がアドバンテージの推定値となる。
  • PPO (近接方策最適化): 重要度比 $r_t(\theta) = \dfrac{\pi_\theta(a\mid s)}{\pi_{\theta_{\rm old}}(a\mid s)}$ をクリップした代理目的を最大化し，1回の更新幅を制限して安定化する。
$$L^{\rm CLIP}(\theta) = \mathbb{E}\!\left[ \min\bigl( r_t(\theta) A_t,\ \mathrm{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon) A_t \bigr) \right]$$

7. 応用

📝 例 — 実世界への応用
  • ゲーム (AlphaGo/AlphaZero): モンテカルロ木探索と深層方策・価値ネットワークを組み合わせ，自己対戦のみで囲碁・将棋・チェスの超人的方策を獲得。
  • ロボット制御: 連続行動空間での歩行・把持・操作の学習。方策勾配法（PPO・SAC）とシミュレーションからの転移（sim-to-real）が中心。
  • 推薦システム: ユーザとの逐次的相互作用を MDP とみなし，長期的なエンゲージメントを最大化する推薦方策を最適化。
  • 資源配分: データセンターの冷却制御，通信ネットワークのルーティング，在庫・電力管理など，逐次的意思決定問題に適用。
💡 次のステップ

強化学習は最適制御と深く関連している。連続時間・連続状態での最適制御（HJB 方程式・LQR）は 制御理論 で，価値関数近似や方策の学習を支える基礎技法は 機械学習の数理 で学ぶとよい。

✏️ 練習問題
  1. 2状態・2行動の小さな MDP を1つ定め，割引率 $\gamma = 0.9$ のもとで方策評価と価値反復を数ステップ手計算し，最適方策を求めよ。
  2. Bellman 最適作用素が上限ノルムに関して係数 $\gamma$ の縮小写像であることを証明せよ。
  3. 同一の経験系列に対して Q学習と SARSA の更新式を書き下し，$\max$ の有無が更新結果に与える違いを説明せよ。崖歩き問題で両者の学習する方策が異なる理由を論ぜよ。
  4. 方策勾配定理においてベースライン $b(s)$ を導入しても勾配の期待値が変わらない（不偏性が保たれる）ことを示せ。
  5. DQN の目標ネットワークを取り除いた場合に学習が不安定化する理由を，目標値が学習中に変動する点から説明せよ。
