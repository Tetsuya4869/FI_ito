1. 最小作用の原理

📖 定義 — 作用汎関数

一般化座標 $q = (q_1,\ldots,q_n)$ に対して，Lagrangian $L(q,\dot q, t)$ を定義する。作用汎関数は

$$S[q] = \int_{t_1}^{t_2} L(q(t),\dot q(t),t)\,dt$$

通常 $L = T - V$（$T$: 運動エネルギー，$V$: ポテンシャルエネルギー）。

✓ 最小作用の原理（Hamilton の原理）

端点 $q(t_1), q(t_2)$ を固定した経路のうち，作用 $S[q]$ を停留させる経路が物理的な運動を与える：$\delta S = 0$。

2. Lagrange 力学

✓ Euler-Lagrange 方程式

$\delta S = 0$ の変分条件は，各 $k$ に対して

$$\frac{d}{dt}\frac{\partial L}{\partial \dot q_k} - \frac{\partial L}{\partial q_k} = 0 \quad (k = 1,\ldots,n)$$

これが $n$ 個の運動方程式を与える。Newton の第二法則 $F = ma$ はデカルト座標での特別な場合。

📝 例 — 単振り子

長さ $l$，質量 $m$ の単振り子。一般化座標 $q = \theta$（振れ角）。

$$T = \frac{1}{2}ml^2\dot\theta^2, \quad V = -mgl\cos\theta$$ $$L = T - V = \frac{1}{2}ml^2\dot\theta^2 + mgl\cos\theta$$

Euler-Lagrange 方程式: $ml^2\ddot\theta + mgl\sin\theta = 0 \;\Rightarrow\; \ddot\theta + \frac{g}{l}\sin\theta = 0$。

小振幅近似 $\sin\theta \approx \theta$: $\ddot\theta + \omega_0^2\theta = 0$，$\omega_0 = \sqrt{g/l}$。

💡 拘束力が自動消去される利点

Lagrange 形式では拘束条件を一般化座標に吸収するため，拘束力（糸の張力・法線反力など）は方程式に現れない。Newton 形式では拘束力も未知数として扱う必要がある。

3. Hamiltonian 力学

📖 定義 — 正準運動量と Hamiltonian

正準運動量: $p_k = \dfrac{\partial L}{\partial \dot q_k}$。

Hamiltonian (Legendre 変換): $H(q, p, t) = \displaystyle\sum_k p_k \dot q_k - L$。

$L = T - V$ かつ $T$ が $\dot q$ の二次形式のとき，$H = T + V$（全エネルギー）。

✓ 正準方程式（Hamilton の運動方程式）
$$\dot q_k = \frac{\partial H}{\partial p_k}, \quad \dot p_k = -\frac{\partial H}{\partial q_k} \quad (k=1,\ldots,n)$$

$2n$ 個の一階 ODE。Euler-Lagrange の $n$ 個の二階 ODE と等価。

📖 Poisson 括弧
$$\{f, g\} = \sum_k \left(\frac{\partial f}{\partial q_k}\frac{\partial g}{\partial p_k} - \frac{\partial f}{\partial p_k}\frac{\partial g}{\partial q_k}\right)$$

正準方程式は $\dot f = \{f, H\} + \partial_t f$。保存量 $C$ iff $\{C, H\} = 0$（かつ $\partial_t C = 0$）。

基本 Poisson 括弧: $\{q_i, p_j\} = \delta_{ij}$，$\{q_i,q_j\}=\{p_i,p_j\}=0$。

✓ Liouville の定理

位相空間 $(q,p)$ 上の流れは体積を保存する（非圧縮性）。統計力学でのアンサンブル不変性の基礎。

4. Noether の定理

✓ 定理 — Noether（連続対称性と保存量の対応）

作用 $S$ が連続変換 $q \mapsto q + \varepsilon \eta(q, t)$ のもとで不変（$\delta S = 0$）ならば，

$$J = \sum_k \frac{\partial L}{\partial \dot q_k} \eta_k$$

が運動の定数（保存量）となる。

📝 対称性と保存量の対応
対称性変換保存量
時間並進 $t \to t + \varepsilon$$L$ が $t$ に依存しないエネルギー $H$
空間並進 $x \to x + \varepsilon$$L$ が位置に依存しない運動量 $p$
回転 $\theta \to \theta + \varepsilon$$L$ が回転角に依存しない角運動量 $L_z$
位相変換 $\psi \to e^{i\varepsilon}\psi$$U(1)$ 対称性電荷（QM）

5. 剛体力学

📖 慣性テンソル

剛体の慣性テンソル $I_{ij} = \int \rho(\mathbf{r})\bigl(\delta_{ij}r^2 - r_i r_j\bigr)\,dV$。

主軸座標では $I$ は対角行列。角運動量 $\mathbf{L} = I\boldsymbol{\omega}$（主軸では各成分独立）。

✓ Euler の方程式（剛体回転）

主軸系での回転方程式（外力矩なし）:

$$I_1\dot\omega_1 = (I_2 - I_3)\omega_2\omega_3$$ $$I_2\dot\omega_2 = (I_3 - I_1)\omega_3\omega_1$$ $$I_3\dot\omega_3 = (I_1 - I_2)\omega_1\omega_2$$

6. 中心力問題

✓ 有効ポテンシャルと Kepler 則

質量 $\mu$（換算質量）の粒子が $V(r)$ 中で運動。角運動量 $l = \mu r^2 \dot\phi$ が保存。有効ポテンシャル:

$$V_{\rm eff}(r) = V(r) + \frac{l^2}{2\mu r^2}$$

重力 $V = -k/r$ の場合，軌道は楕円（$E<0$），放物線（$E=0$），双曲線（$E>0$）。

Kepler 第3則: $T^2 \propto a^3$（半長軸 $a$）。

7. 振動と正規モード

📖 連成振動・正規座標

$n$ 自由度の微小振動: $M\ddot q + Kq = 0$（$M$: 質量行列，$K$: 剛性行列）。

固有値問題 $K\mathbf{u} = \omega^2 M\mathbf{u}$ を解くことで正規モード（固有振動数 $\omega_k$，固有ベクトル $\mathbf{u}_k$）が得られる。

正規座標 $\xi_k$ では方程式が分離: $\ddot\xi_k + \omega_k^2\xi_k = 0$。

✏️ 練習問題
  1. 二重振り子の Lagrangian を求め，小振幅での正規モードを計算せよ。
  2. $H = p^2/(2m) + m\omega^2 q^2/2$ から正準方程式を導き，解を求めよ。
  3. 平面極座標での質点運動に Noether の定理を適用し，角運動量の保存を示せ。
  4. 一様球の慣性テンソルを計算し，Euler 方程式の自由歳差運動を解析せよ。
解答例を見る

  1. 質量 $m_1, m_2$，長さ $l_1, l_2$，角度 $\theta_1,\theta_2$（鉛直から）とする。各質点の位置から

    $x_1 = l_1\sin\theta_1,\; y_1 = -l_1\cos\theta_1$，$x_2 = x_1 + l_2\sin\theta_2,\; y_2 = y_1 - l_2\cos\theta_2$。運動エネルギーは

    $$T = \tfrac12 m_1 l_1^2\dot\theta_1^2 + \tfrac12 m_2\bigl[l_1^2\dot\theta_1^2 + l_2^2\dot\theta_2^2 + 2l_1 l_2\dot\theta_1\dot\theta_2\cos(\theta_1-\theta_2)\bigr]$$

    ポテンシャルは $V = -(m_1+m_2)gl_1\cos\theta_1 - m_2 g l_2\cos\theta_2$。よって $L = T - V$。

    小振幅 $\theta_i \ll 1$（$\cos(\theta_1-\theta_2)\approx1$，$\cos\theta_i\approx1-\theta_i^2/2$）では $M\ddot\theta + K\theta = 0$ となり，

    $$M = \begin{pmatrix}(m_1+m_2)l_1^2 & m_2 l_1 l_2\\ m_2 l_1 l_2 & m_2 l_2^2\end{pmatrix},\quad K = \begin{pmatrix}(m_1+m_2)gl_1 & 0\\ 0 & m_2 g l_2\end{pmatrix}$$

    固有値問題 $\det(K-\omega^2 M)=0$ を解くと2つの正規振動数 $\omega_\pm$（同位相・逆位相モード）が得られる。等質量・等長 $m_1=m_2=m,\,l_1=l_2=l$ では $\omega_\pm^2 = (2\mp\sqrt2)\,g/l$。

  2. $H = \dfrac{p^2}{2m} + \dfrac12 m\omega^2 q^2$ に正準方程式を適用すると

    $$\dot q = \frac{\partial H}{\partial p} = \frac{p}{m},\qquad \dot p = -\frac{\partial H}{\partial q} = -m\omega^2 q$$

    両式から $\ddot q = \dot p/m = -\omega^2 q$。よって単振動

    $$q(t) = A\cos(\omega t + \phi),\qquad p(t) = m\dot q = -mA\omega\sin(\omega t+\phi)$$

    $A,\phi$ は初期条件で決まり，$H = \tfrac12 m\omega^2 A^2$ が保存する（位相空間では楕円軌道）。

  3. 平面極座標で $L = \tfrac12 m(\dot r^2 + r^2\dot\phi^2) - V(r)$。$V$ が $r$ のみに依存するとき $L$ は $\phi$ に陽に依存しない（回転対称性）。無限小変換 $\phi \to \phi + \varepsilon$ すなわち $\eta_\phi = 1$（$\eta_r=0$）に Noether の定理を適用すると保存量は

    $$J = \frac{\partial L}{\partial \dot\phi}\,\eta_\phi = m r^2\dot\phi$$

    これが角運動量 $l = m r^2\dot\phi$ である。実際 $\phi$ は循環座標なので $\dfrac{d}{dt}\dfrac{\partial L}{\partial\dot\phi} = \dfrac{\partial L}{\partial\phi} = 0$ から直接 $l = \text{const}$ が従う。

  4. 半径 $R$，質量 $M$，密度 $\rho = \dfrac{M}{\frac43\pi R^3}$ の一様球。対称性より主軸は任意で $I_1=I_2=I_3\equiv I$。

    $$I = \int\rho\,(y^2+z^2)\,dV = \frac{2}{3}\rho\int r^2\,dV = \frac{2}{3}\rho\cdot 4\pi\int_0^R r^4\,dr = \frac{2}{3}\rho\cdot\frac{4\pi R^5}{5} = \frac{2}{5}MR^2$$

    したがって慣性テンソルは $I_{ij} = \tfrac25 MR^2\,\delta_{ij}$（等方）。Euler 方程式は $I_1=I_2=I_3$ より右辺の $(I_j-I_k)$ が全て 0 となり

    $$\dot\omega_1 = \dot\omega_2 = \dot\omega_3 = 0$$

    すなわち外力矩がなければ角速度ベクトル $\boldsymbol\omega$ は一定で歳差運動は生じない（球対称コマは自由歳差しない）。
