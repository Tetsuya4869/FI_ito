1. 個体群動態

📖 定義 — マルサスの指数増殖

個体数 $N(t)$ が一定の純増殖率 $r$（出生率 − 死亡率）で増えるとき，

$$\frac{dN}{dt} = rN \quad\Longrightarrow\quad N(t) = N_0 e^{rt}$$

$r>0$ なら指数的に発散，$r<0$ なら絶滅。資源無制限という非現実的な仮定に基づく。

✓ 定理 — ロジスティック方程式と収容力

資源の制限を密度依存項で取り込むと，収容力 $K$ をもつロジスティック方程式

$$\frac{dN}{dt} = rN\left(1 - \frac{N}{K}\right)$$

解は $N(t) = \dfrac{K N_0 e^{rt}}{K + N_0(e^{rt}-1)}$（S字型曲線）。平衡点 $N^*=0$（不安定）と $N^*=K$（安定）。

📝 例 — 平衡と安定性の判定

$f(N)=rN(1-N/K)$ とおくと $f'(N) = r(1 - 2N/K)$。$f'(0)=r>0$ より原点は不安定，$f'(K)=-r<0$ より $N=K$ は漸近安定。任意の正の初期値から個体数は $K$ へ単調に収束する。

2. 種間相互作用

📖 定義 — Lotka-Volterra 捕食者-被食者モデル

被食者 $x$，捕食者 $y$ について

$$\frac{dx}{dt} = \alpha x - \beta xy, \qquad \frac{dy}{dt} = \delta xy - \gamma y$$

$\alpha$: 被食者の増殖率，$\gamma$: 捕食者の死亡率，$\beta,\delta$: 相互作用係数。

✓ 定理 — 周期軌道と保存量

内部平衡点 $(x^*,y^*) = (\gamma/\delta,\ \alpha/\beta)$ のまわりで，系は保存量

$$H(x,y) = \delta x - \gamma\ln x + \beta y - \alpha\ln y$$

をもち，閉じた周期軌道を描く（中立安定な中心）。被食者と捕食者の個体数は位相をずらして振動する。

📝 例 — 競争モデルと共存条件

2種競争の Lotka-Volterra 系

$$\frac{dN_1}{dt} = r_1 N_1\!\left(1-\frac{N_1+a_{12}N_2}{K_1}\right),\quad \frac{dN_2}{dt} = r_2 N_2\!\left(1-\frac{N_2+a_{21}N_1}{K_2}\right)$$

共存平衡が安定となる条件は $a_{12}a_{21}<1$（種間競争が種内競争より弱い）。逆に $a_{12}a_{21}>1$ では初期条件依存で一方が排除される（競争排除）。

3. 感染症数理

📖 定義 — SIR モデル

総人口を感受性 $S$・感染 $I$・回復 $R$ に分け，感染率 $\beta$，回復率 $\gamma$ とすると

$$\frac{dS}{dt} = -\beta SI,\quad \frac{dI}{dt} = \beta SI - \gamma I,\quad \frac{dR}{dt} = \gamma I$$

$S+I+R = N$（一定）。$1/\gamma$ は平均感染期間。

✓ 定理 — 基本再生産数と流行の閾値

1人の感染者が全感受性集団 $S_0$ 中で生み出す二次感染者数

$$R_0 = \frac{\beta S_0}{\gamma}$$

流行開始時 $\dfrac{dI}{dt} = (\beta S_0 - \gamma)I$ より，$R_0>1$ で流行（$I$ 増大），$R_0<1$ で終息。$R_0=1$ が閾値。

📝 例 — 集団免疫の閾値

割合 $p$ の人がワクチン等で免疫をもつと有効再生産数は $R_{\rm eff}=R_0(1-p)$。流行を防ぐ条件 $R_{\rm eff}<1$ より

$$p > p_c = 1 - \frac{1}{R_0}$$

例: $R_0=4$ なら $p_c=0.75$。集団の75%以上が免疫をもてば流行は起こらない。

💡 最終規模方程式

SIR では流行終了時に感受性のまま残る割合 $s_\infty$ が $\ln s_\infty = R_0(s_\infty - 1)$ を満たす。$R_0$ が大きいほど多くが感染を経験する。

4. 反応拡散系とパターン

📖 定義 — 拡散方程式と反応拡散系

濃度 $u(\mathbf{x},t)$ の拡散は $\partial_t u = D\nabla^2 u$。局所反応 $f(u)$ を加えた一般の反応拡散系は

$$\frac{\partial \mathbf{u}}{\partial t} = \mathbf{D}\,\nabla^2 \mathbf{u} + \mathbf{f}(\mathbf{u})$$

$\mathbf{D}$ は拡散係数の対角行列。

✓ 定理 — Fisher-KPP 方程式と進行波

単一種の拡散ロジスティック

$$\frac{\partial u}{\partial t} = D\frac{\partial^2 u}{\partial x^2} + ru(1-u)$$

は速度 $c \ge c_{\min} = 2\sqrt{rD}$ の進行波解 $u(x-ct)$ をもつ。生物の侵入・拡がりの最小速度を与える。

📝 例 — Turing パターン形成

活性化因子 $u$・抑制因子 $v$ の2成分系で，拡散がないときは安定な一様平衡が，拡散により不安定化してパターンが生じる（拡散誘導不安定性）。条件は抑制因子の方が速く拡散すること $D_v \gg D_u$。波数 $k$ の摂動の成長率が正となる帯域があり，動物の体表模様などを説明する。

5. 興奮系と神経

📖 定義 — FitzHugh-Nagumo モデル

神経興奮の最小モデル。膜電位様の変数 $v$ と回復変数 $w$ について

$$\dot v = v - \frac{v^3}{3} - w + I, \qquad \dot w = \varepsilon(v + a - bw)$$

$\varepsilon\ll 1$ で時間スケールが分離する（速い $v$，遅い $w$）。

✓ 定理 — 興奮性と閾値挙動

$I$ が小さいとき唯一の安定平衡をもつが，閾値を超える刺激には大きな偏位（スパイク）で応答してから平衡へ戻る。これが興奮性である。$I$ を増すと Hopf 分岐により平衡が不安定化し，周期的なスパイク列（リミットサイクル）が生じる。

📝 例 — 興奮系の進行波

空間結合を加えた $\partial_t v = D\,\partial_x^2 v + v - v^3/3 - w + I$ では，活動電位が一定速度で軸索を伝わる進行パルスが現れる。前方の静止部を興奮させながら，後方は回復（不応期）するため，パルスは安定に伝播する。

6. 集団遺伝学

📖 定義 — Hardy-Weinberg 平衡

対立遺伝子 $A,a$ の頻度を $p,q$（$p+q=1$）とする。ランダム交配・無選択・無突然変異・大集団のもとで遺伝子型頻度は

$$AA:Aa:aa = p^2 : 2pq : q^2$$

に達し，以後世代を通じて遺伝子頻度 $p,q$ は不変に保たれる。

✓ 定理 — 選択と突然変異による頻度変化

遺伝子型 $AA,Aa,aa$ の適応度を $w_{AA},w_{Aa},w_{aa}$ とすると，1世代後の $A$ 頻度は

$$p' = \frac{p^2 w_{AA} + pq\,w_{Aa}}{\bar w},\quad \bar w = p^2 w_{AA} + 2pq\,w_{Aa} + q^2 w_{aa}$$

連続近似では $\dfrac{dp}{dt} \approx p(1-p)\dfrac{1}{2}\dfrac{\partial \ln \bar w}{\partial p}$。突然変異率 $\mu\,(A\!\to\!a)$ を加えると平衡頻度が $1$ からずれる。

📝 例 — 選択・突然変異平衡

劣性有害対立遺伝子（$aa$ の適応度低下 $s$）が突然変異率 $\mu$ で供給されるとき，平衡頻度は $\hat q \approx \sqrt{\mu/s}$。例: $\mu=10^{-6},\ s=0.01$ なら $\hat q \approx 0.01$。稀な遺伝病が集団中に維持される機構を説明する。

7. 進化動態

📖 定義 — レプリケーター方程式

戦略 $i$ の頻度 $x_i$，利得行列 $A$ とする。戦略 $i$ の適応度 $f_i=(Ax)_i$，平均適応度 $\bar f = x^\top A x$ として

$$\dot x_i = x_i\bigl(f_i - \bar f\,\bigr), \qquad \sum_i x_i = 1$$

平均より高い利得をもつ戦略が頻度を増す。

✓ 定理 — 進化的に安定な戦略 (ESS)

戦略 $x^*$ が ESS であるとは，任意の変異戦略 $y\ne x^*$ に対して

$$x^{*\top} A y > y^\top A y \quad\text{（侵入を許さない）}$$

が成り立つこと。ESS はレプリケーター動学の漸近安定平衡を与え，ナッシュ均衡の精緻化となっている。

📝 例 — タカ・ハトゲームと適応度地形

資源価値 $V$，争いの損失 $C\,(>V)$ のタカ・ハトゲームでは，混合 ESS でタカの頻度が $x^*=V/C$ となる。適応度地形では，集団は平均適応度 $\bar f$ を局所的に増す方向へ動くが，頻度依存選択のもとでは必ずしも全体最適へ到達しない。
