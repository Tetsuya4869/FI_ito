1. このサイトの使い方

💡 進め方のコツ
  • 数式は KaTeX で表示されます。読み飛ばさず、紙に書き写すと定着します。
  • 各ページの「📝 例」と「✏️ 練習問題」を必ず手を動かして確認しましょう。
  • 難しいと感じたら、ページ冒頭の前提知識に戻るのが近道です。
  • サイドバー上部の検索でトピックを素早く探せます。

4. レベル別ロードマップ

📝 STEP 0 — 基礎固め（高校〜大学1年）

微分積分の基礎 → ベクトルと行列 → 集合と論理 → 確率・統計の基礎 → ニュートン力学

📝 STEP 1 — 数学の柱（大学2〜3年）

解析学 → 線形代数 → 位相空間論 → 複素解析

📝 STEP 2 — 確率と物理（大学3〜4年）

測度論 → 確率論 → 確率微分方程式（伊藤の公式）、並行して 古典力学 → 量子力学

📝 STEP 3 — 発展（大学院）

関数解析、微分幾何学 → 相対性理論、統計力学、学問領域マップで全体を俯瞰
