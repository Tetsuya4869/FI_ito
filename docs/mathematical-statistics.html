💡 前提知識

本ページは確率分布・期待値・大数の法則・中心極限定理の知識を前提とする。未習の場合は 確率論 と 確率・統計の基礎 を先に参照されたい。確率変数の収束概念（概収束・確率収束・分布収束）が随所で現れる。

1. 統計的推測の枠組み

📖 定義 — 母集団・標本・統計モデル

関心の対象全体を母集団といい，その性質は未知パラメータ $\theta \in \Theta$ をもつ確率分布 $P_\theta$ で記述される。この $\{P_\theta : \theta \in \Theta\}$ を統計モデルという。

母集団から無作為に取り出した $X_1,\ldots,X_n$ を標本と呼ぶ。多くの場合，互いに独立で同一分布に従う（i.i.d.）と仮定する：$X_1,\ldots,X_n \overset{\text{i.i.d.}}{\sim} P_\theta$。

📖 定義 — 統計量と標本分布

標本のみの関数 $T = T(X_1,\ldots,X_n)$（未知パラメータを含まない）を統計量という。代表例は標本平均と不偏分散：

$$\bar X = \frac{1}{n}\sum_{i=1}^n X_i, \qquad S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar X)^2$$

統計量も確率変数であり，その従う分布を標本分布と呼ぶ。標本分布の解析が推測統計の核心である。

✓ 定理 — 正規母集団の標本分布

$X_1,\ldots,X_n \overset{\text{i.i.d.}}{\sim} N(\mu,\sigma^2)$ のとき，$\bar X$ と $S^2$ は独立で，

$$\bar X \sim N\!\left(\mu,\tfrac{\sigma^2}{n}\right), \qquad \frac{(n-1)S^2}{\sigma^2} \sim \chi^2_{n-1}, \qquad \frac{\bar X - \mu}{S/\sqrt{n}} \sim t_{n-1}$$

これらが正規母集団に対する区間推定・検定の基礎となる。

💡 推定と検定の全体像

統計的推測は大きく二つに分かれる。推定はパラメータ $\theta$ の値を当てる問題で，1点を答える点推定と範囲で答える区間推定がある。検定は「$\theta$ がある値・領域に属するか」という仮説の真偽をデータから判定する問題である。以下，この順に展開する。

2. 点推定

📖 定義 — 推定量と不偏性・一致性・有効性

$\theta$ を推定するための統計量 $\hat\theta = \hat\theta(X_1,\ldots,X_n)$ を推定量という。望ましい性質として：

  • 不偏性: $\mathbb{E}_\theta[\hat\theta] = \theta$（すべての $\theta$ で偏りがない）。
  • 一致性: $\hat\theta_n \xrightarrow{P} \theta$（$n\to\infty$ で真値に確率収束）。
  • 有効性: 不偏推定量の中で分散が最小（後述の Cramér-Rao 下界を達成）。

例えば $\bar X$ は母平均 $\mu$ の不偏かつ一致推定量である。$S^2$ は $\sigma^2$ の不偏推定量（$n-1$ で割る理由）。

📖 定義 — 最尤推定（MLE）

観測値を固定し，パラメータの関数とみた同時密度を尤度関数という：

$$L(\theta) = \prod_{i=1}^n f(x_i;\theta), \qquad \ell(\theta) = \log L(\theta) = \sum_{i=1}^n \log f(x_i;\theta)$$

尤度（対数尤度）を最大化する $\hat\theta_{\mathrm{MLE}} = \arg\max_\theta \ell(\theta)$ を最尤推定量という。多くの場合，尤度方程式 $\dfrac{\partial \ell}{\partial \theta} = 0$ を解いて求める。

📝 例 — 正規分布の MLE

$X_i \overset{\text{i.i.d.}}{\sim} N(\mu,\sigma^2)$ の対数尤度：

$$\ell(\mu,\sigma^2) = -\frac{n}{2}\log(2\pi\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^n (x_i-\mu)^2$$

$\partial\ell/\partial\mu = 0$ より $\hat\mu = \bar x$。$\partial\ell/\partial\sigma^2 = 0$ より

$$\hat\sigma^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \bar x)^2$$

$\hat\mu$ は不偏だが $\hat\sigma^2$ は $\mathbb{E}[\hat\sigma^2] = \frac{n-1}{n}\sigma^2$ で過小評価する（不偏でない）。

📝 例 — モーメント法

理論モーメントと標本モーメントを等置してパラメータを解く方法。$k$ 個のパラメータには $k$ 本の方程式 $\mathbb{E}_\theta[X^j] = \frac{1}{n}\sum_i X_i^j$（$j=1,\ldots,k$）を用いる。

例：指数分布 $\mathrm{Exp}(\lambda)$ では $\mathbb{E}[X] = 1/\lambda$ なので $\hat\lambda_{\text{MoM}} = 1/\bar X$。一般にモーメント法は計算が容易だが，MLE より効率が劣ることが多い。

✓ 定理 — MLE の漸近正規性

正則条件のもとで，MLE は一致性をもち，かつ漸近的に正規分布する：

$$\sqrt{n}\,(\hat\theta_{\mathrm{MLE}} - \theta) \xrightarrow{d} N\!\left(0,\; I_1(\theta)^{-1}\right)$$

ここで $I_1(\theta)$ は1標本あたりの Fisher 情報量（次節）。漸近分散が下界 $I_1(\theta)^{-1}$ に一致するため，MLE は漸近有効である。

3. 推定量の評価

📖 定義 — 平均二乗誤差（MSE）とバイアス・分散分解

推定量 $\hat\theta$ の精度を測る指標：

$$\mathrm{MSE}(\hat\theta) = \mathbb{E}_\theta\bigl[(\hat\theta - \theta)^2\bigr] = \underbrace{\mathrm{Var}_\theta(\hat\theta)}_{\text{分散}} + \underbrace{\bigl(\mathbb{E}_\theta[\hat\theta] - \theta\bigr)^2}_{\text{バイアス}^2}$$

不偏推定量ではバイアス項が消え $\mathrm{MSE} = \mathrm{Var}$ となる。バイアスを許せば分散を下げて MSE を改善できる場合もある（バイアス・分散トレードオフ）。

📖 定義 — Fisher 情報量

スコア関数 $\partial_\theta \log f(X;\theta)$ の分散をFisher 情報量という：

$$I(\theta) = \mathbb{E}_\theta\!\left[\left(\frac{\partial}{\partial\theta}\log f(X;\theta)\right)^2\right] = -\,\mathbb{E}_\theta\!\left[\frac{\partial^2}{\partial\theta^2}\log f(X;\theta)\right]$$

$n$ 個の i.i.d. 標本では $I_n(\theta) = n\,I_1(\theta)$（情報は加法的）。Fisher 情報量はデータが $\theta$ について「どれだけ情報をもつか」の尺度。

✓ 定理 — Cramér-Rao の下界

正則条件のもとで，$\theta$ の任意の不偏推定量 $\hat\theta$ の分散は次の下界をもつ：

$$\mathrm{Var}_\theta(\hat\theta) \ge \frac{1}{I_n(\theta)} = \frac{1}{n\,I_1(\theta)}$$

この下界を達成する不偏推定量を有効推定量という。$g(\theta)$ の不偏推定では右辺が $\{g'(\theta)\}^2 / I_n(\theta)$ となる。

📝 例 — 正規分布の母平均

$N(\mu,\sigma^2)$（$\sigma^2$ 既知）で $\log f = -\frac{(x-\mu)^2}{2\sigma^2} + \text{const}$。よって $\partial_\mu^2 \log f = -1/\sigma^2$ から $I_1(\mu) = 1/\sigma^2$。下界は

$$\frac{1}{n I_1(\mu)} = \frac{\sigma^2}{n} = \mathrm{Var}(\bar X)$$

標本平均 $\bar X$ は下界を達成しており，$\mu$ の有効推定量である。

📖 定義 — 十分統計量

統計量 $T$ が $\theta$ に対する十分統計量であるとは，$T$ を与えたときの標本の条件付き分布が $\theta$ に依存しないこと。直観的には「$T$ はデータ中の $\theta$ に関する情報をすべて含む」。

Fisher-Neyman の分解定理：$T$ が十分 $\iff$ $f(x;\theta) = g(T(x);\theta)\,h(x)$ と分解できる。

Rao-Blackwell の定理：不偏推定量を十分統計量で条件付けると（$\hat\theta^\ast = \mathbb{E}[\hat\theta \mid T]$）分散が減少し，より良い不偏推定量が得られる。

4. 区間推定

📖 定義 — 信頼区間

統計量 $L(X), U(X)$ が定める区間 $[L,U]$ が，信頼係数 $1-\alpha$ の信頼区間であるとは，すべての $\theta$ について

$$P_\theta\bigl(L(X) \le \theta \le U(X)\bigr) = 1-\alpha$$

解釈に注意：ランダムなのは区間 $[L,U]$ であり $\theta$ ではない。「同じ実験を多数繰り返すと，得られる区間の割合 $1-\alpha$ が真値 $\theta$ を含む」という頻度論的意味。

📖 定義 — ピボット量

ピボット量とは，データと $\theta$ の両方に依存するが，その分布が $\theta$ に依存しない量 $Q(X,\theta)$ をいう。$Q$ の分布の分位点から $\theta$ について不等式を解くことで信頼区間を構成できる。

📝 例 — 正規母集団の母平均の信頼区間

$N(\mu,\sigma^2)$（$\sigma^2$ 未知）。ピボット量 $T = \dfrac{\bar X - \mu}{S/\sqrt{n}} \sim t_{n-1}$ を用いる。$t_{n-1}$ の上側 $\alpha/2$ 点を $t_{n-1,\alpha/2}$ とすると，$P(-t_{n-1,\alpha/2} \le T \le t_{n-1,\alpha/2}) = 1-\alpha$ より：

$$\bar X - t_{n-1,\alpha/2}\frac{S}{\sqrt n} \;\le\; \mu \;\le\; \bar X + t_{n-1,\alpha/2}\frac{S}{\sqrt n}$$

$\sigma^2$ 既知なら $t$ 分位点を標準正規 $z_{\alpha/2}$ に置き換える。

📝 例 — 母分散の信頼区間

ピボット量 $\dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2_{n-1}$ を用いる。カイ二乗分布の分位点 $\chi^2_{n-1,1-\alpha/2}, \chi^2_{n-1,\alpha/2}$ より：

$$\frac{(n-1)S^2}{\chi^2_{n-1,\alpha/2}} \;\le\; \sigma^2 \;\le\; \frac{(n-1)S^2}{\chi^2_{n-1,1-\alpha/2}}$$

カイ二乗分布は非対称なので，区間も $S^2$ について非対称になる。

5. 仮説検定

📖 定義 — 帰無仮説・対立仮説と二種の誤り

判定したい命題を 帰無仮説 $H_0$，その否定を 対立仮説 $H_1$ とする。検定はデータから $H_0$ を棄却するか保留するかを決める手続き。二種の誤りがある：

$H_0$ 真$H_0$ 偽
$H_0$ 棄却第1種の誤り（確率 $\alpha$）正しい判定（検出力 $1-\beta$）
$H_0$ 保留正しい判定（$1-\alpha$）第2種の誤り（確率 $\beta$）

有意水準 $\alpha$ は第1種の誤りの許容上限。検出力（power）は $1-\beta$，すなわち偽の $H_0$ を正しく棄却する確率。

📖 定義 — 検定統計量と棄却域

検定は，検定統計量 $T(X)$ がある棄却域 $R$ に落ちたとき $H_0$ を棄却する手続きで与えられる。$\alpha = P_{H_0}(T \in R)$ を満たすよう $R$ を定める。検出力関数は $\beta(\theta) = P_\theta(T \in R)$。

✓ 定理 — Neyman-Pearson の補題

単純帰無 $H_0:\theta=\theta_0$ 対 単純対立 $H_1:\theta=\theta_1$ の検定において，与えられた水準 $\alpha$ のもとで検出力を最大にする検定は尤度比検定である：

$$\Lambda(x) = \frac{L(\theta_1)}{L(\theta_0)} = \frac{\prod_i f(x_i;\theta_1)}{\prod_i f(x_i;\theta_0)} > k \;\Rightarrow\; H_0\text{ を棄却}$$

閾値 $k$ は $P_{\theta_0}(\Lambda > k) = \alpha$ で定める。これが最強力検定を与える。複合仮説では一様最強力検定（UMP）や一般化尤度比検定へ拡張される。

6. 代表的な検定

✓ 定理 — t 検定

正規母集団の母平均に関する検定。$H_0:\mu=\mu_0$ に対し（$\sigma^2$ 未知）検定統計量：

$$T = \frac{\bar X - \mu_0}{S/\sqrt n} \underset{H_0}{\sim} t_{n-1}$$

$|T| > t_{n-1,\alpha/2}$ なら両側検定で棄却。二標本 t 検定は二群の平均差 $\mu_1-\mu_2$ を検定する（等分散仮定のもとプールした分散を用いる）。

📝 例 — カイ二乗検定

適合度・独立性の検定。観測度数 $O_i$ と期待度数 $E_i$ から：

$$\chi^2 = \sum_{i} \frac{(O_i - E_i)^2}{E_i} \underset{H_0}{\sim} \chi^2_{\text{df}}$$

適合度検定では $\text{df} = (\text{カテゴリ数}) - 1 - (\text{推定パラメータ数})$。$r\times c$ 分割表の独立性検定では $\text{df}=(r-1)(c-1)$。$\chi^2 > \chi^2_{\text{df},\alpha}$ で棄却。

📝 例 — F 検定

二つの正規母集団の分散の比 $\sigma_1^2/\sigma_2^2$ を検定。$H_0:\sigma_1^2=\sigma_2^2$ に対し：

$$F = \frac{S_1^2}{S_2^2} \underset{H_0}{\sim} F_{n_1-1,\,n_2-1}$$

分散分析（ANOVA）でも，群間変動と群内変動の比として F 統計量が用いられ，複数群の平均の同等性を一括検定する。

💡 p 値の正しい解釈

p 値とは「$H_0$ が真であると仮定したとき，観測された以上に極端な検定統計量が得られる確率」である。$p < \alpha$ なら有意水準 $\alpha$ で棄却。よくある誤解に注意：p 値は「$H_0$ が正しい確率」ではない。また有意（小さな p 値）であることと効果が大きいことは別問題で，効果量や信頼区間と併せて解釈すべきである。

7. ベイズ統計と回帰

📖 定義 — ベイズ推定（事前・事後分布）

ベイズ流ではパラメータ $\theta$ 自体を確率変数とみなし，観測前の信念を事前分布 $\pi(\theta)$ で表す。データ $x$ を得た後の信念は事後分布で，Bayes の定理から：

$$\pi(\theta \mid x) = \frac{f(x \mid \theta)\,\pi(\theta)}{\int f(x \mid \theta')\,\pi(\theta')\,d\theta'} \;\propto\; \underbrace{L(\theta)}_{\text{尤度}}\,\underbrace{\pi(\theta)}_{\text{事前}}$$

事後分布から点推定（事後平均 $\mathbb{E}[\theta\mid x]$ や MAP 推定 $\arg\max_\theta \pi(\theta\mid x)$）や信用区間（事後確率 $1-\alpha$ の区間）を得る。

📝 例 — ベータ・二項共役

$X \mid \theta \sim \mathrm{Bin}(n,\theta)$，事前 $\theta \sim \mathrm{Beta}(a,b)$ とすると事後は再びベータ分布：

$$\theta \mid x \sim \mathrm{Beta}(a + x,\; b + n - x)$$

事前と事後が同族になるこの性質を共役という。事後平均は $\dfrac{a+x}{a+b+n}$ で，事前情報（$a,b$）と尤度（$x,n$）の加重平均になっている。

✓ 定理 — 線形回帰と最小二乗

モデル $y_i = \mathbf{x}_i^\top\boldsymbol\beta + \varepsilon_i$（$\varepsilon_i \overset{\text{i.i.d.}}{\sim} N(0,\sigma^2)$）。残差二乗和 $\|\mathbf{y}-X\boldsymbol\beta\|^2$ を最小化する最小二乗推定量は，正規方程式 $X^\top X\boldsymbol\beta = X^\top\mathbf{y}$ より

$$\hat{\boldsymbol\beta} = (X^\top X)^{-1}X^\top\mathbf{y}$$

正規誤差のもとでは最小二乗解は MLE に一致する。Gauss-Markov の定理により $\hat{\boldsymbol\beta}$ は最良線形不偏推定量（BLUE）である。

💡 頻度論とベイズの対比

頻度論では $\theta$ は固定された未知定数で，確率はデータの反復に由来する。ベイズでは $\theta$ に確率分布を与え，主観的・情報的な不確実性を直接表現する。実用上は MAP 推定が正則化付き最尤（例：Ridge 回帰 = 正規事前のもとでの MAP）に対応するなど，両者は深く関連する。

✏️ 練習問題
  1. $X_1,\ldots,X_n \overset{\text{i.i.d.}}{\sim} \mathrm{Exp}(\lambda)$ について，$\lambda$ の最尤推定量を求めよ。さらに Fisher 情報量 $I_n(\lambda)$ と Cramér-Rao 下界を計算し，MLE がそれを（漸近的に）達成するか論ぜよ。
  2. $X_i \overset{\text{i.i.d.}}{\sim} N(\mu,\sigma^2)$ で $\sigma^2$ が未知のとき，母平均 $\mu$ の信頼係数 $95\%$ の信頼区間を構成する手順を，用いるピボット量とその分布を明示して説明せよ。
  3. 単純帰無 $H_0:\theta=\theta_0$ 対 単純対立 $H_1:\theta=\theta_1$ に Neyman-Pearson の補題を適用し，正規分布 $N(\theta,1)$（$\theta_1>\theta_0$）の最強力検定の棄却域を $\bar X$ で表せ。
  4. 二項モデル $X\mid\theta\sim\mathrm{Bin}(n,\theta)$ に一様事前 $\mathrm{Beta}(1,1)$ を置いたときの事後分布・事後平均・MAP 推定量を求め，最尤推定 $\hat\theta=x/n$ と比較せよ。
  5. 単回帰 $y_i=\beta_0+\beta_1 x_i+\varepsilon_i$ について正規方程式を立て，$\hat\beta_1 = \dfrac{\sum(x_i-\bar x)(y_i-\bar y)}{\sum(x_i-\bar x)^2}$ および $\hat\beta_0=\bar y-\hat\beta_1\bar x$ を導出せよ。
