💡 前提知識

本ページでは微分・積分・初等関数の取り扱いを前提とする。不安があれば 微分積分の基礎 を先に確認してほしい。線形系の節では固有値・固有ベクトルなど線形代数の知識も役立つ。

1. 基本概念と分類

📖 定義 — 常微分方程式

未知関数 $y(x)$ とその導関数の間の関係式

$$F\bigl(x, y, y', y'', \ldots, y^{(n)}\bigr) = 0$$

を 常微分方程式（ODE）という。「常」とは独立変数が1つ（ここでは $x$）であることを指す。最高階の導関数 $y^{(n)}$ の階数 $n$ をその方程式の 階数 という。

📖 定義 — 線形 ODE と分類

未知関数とその導関数について1次（線形）であるものを 線形 ODE という。$n$ 階線形 ODE は

$$a_n(x)\,y^{(n)} + \cdots + a_1(x)\,y' + a_0(x)\,y = g(x)$$

の形。$g(x)\equiv 0$ のとき 斉次，そうでないとき 非斉次。係数 $a_k$ がすべて定数なら 定数係数。これ以外は 非線形 という。また右辺が独立変数 $x$ を陽に含まない $y'=f(y)$ の形を 自励系（autonomous） と呼ぶ。

📖 定義 — 解・一般解・初期値問題

方程式を満たす関数を 解 という。$n$ 階 ODE の一般解は $n$ 個の任意定数を含む。条件

$$y(x_0) = y_0,\quad y'(x_0)=y_1,\ \ldots,\ y^{(n-1)}(x_0)=y_{n-1}$$

を与えて定数を決定する問題を 初期値問題（IVP） という。境界の2点で条件を課す場合は境界値問題と呼ぶ。

📝 例 — 分類の練習
方程式階数分類
$y' = ky$1線形・定数係数・斉次・自励
$y'' + \sin y = 0$2非線形（$\sin y$）・自励
$y'' + 3y' + 2y = e^{x}$2線形・定数係数・非斉次
$x\,y' + y = x^2$1線形・変数係数・非斉次

2. 1階ODE

📖 定義 — 変数分離形

$\dfrac{dy}{dx} = f(x)\,g(y)$ の形を 変数分離形 という。$g(y)\neq 0$ の範囲で変数を分離して積分する：

$$\int \frac{dy}{g(y)} = \int f(x)\,dx + C$$
📝 例 — 指数的増殖

$y' = ky$ を解く。$\dfrac{dy}{y} = k\,dx$ より $\ln|y| = kx + C_1$，したがって

$$y(x) = C\,e^{kx}\quad (C = \pm e^{C_1}).$$

初期条件 $y(0)=y_0$ なら $C = y_0$。

✓ 定理 — 1階線形 ODE と積分因子

$y' + p(x)\,y = q(x)$ に対し，積分因子

$$\mu(x) = e^{\int p(x)\,dx}$$

を掛けると左辺は $\bigl(\mu y\bigr)'$ にまとまる。ゆえに一般解は

$$y(x) = \frac{1}{\mu(x)}\left( \int \mu(x)\,q(x)\,dx + C \right).$$
📝 例 — 積分因子の適用

$y' + 2y = e^{-x}$。$p=2$ なので $\mu = e^{2x}$。両辺に掛けて $(e^{2x}y)' = e^{x}$。積分して $e^{2x}y = e^{x} + C$，よって

$$y(x) = e^{-x} + C\,e^{-2x}.$$
📖 定義 — 完全微分方程式

$M(x,y)\,dx + N(x,y)\,dy = 0$ が 完全 であるとは，ある $\Phi(x,y)$ で $d\Phi = M\,dx + N\,dy$ となること。判定条件は

$$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}.$$

このとき $\Phi_x = M,\ \Phi_y = N$ から $\Phi$ を復元し，解は $\Phi(x,y)=C$。条件を満たさない場合でも積分因子で完全化できることがある。

✓ 定理 — Bernoulli 型の線形化

$y' + p(x)\,y = q(x)\,y^{n}\ (n\neq 0,1)$ は $v = y^{1-n}$ と置くと

$$v' + (1-n)\,p(x)\,v = (1-n)\,q(x)$$

という1階線形 ODE になり，積分因子で解ける。

💡 解法の見分け方

1階 ODE はまず「変数分離できるか」を試し，だめなら「線形（$y$ について1次）か」を確認して積分因子へ。$M\,dx+N\,dy=0$ の形なら完全性を判定。$y^n$ が右辺にあれば Bernoulli を疑う。

3. 定数係数線形2階

📖 定義 — 定数係数線形2階 ODE

$a y'' + b y' + c y = g(x)$（$a\neq 0$，$a,b,c$ は定数）。一般解は斉次解 $y_h$ と特殊解 $y_p$ の和 $y = y_h + y_p$。

✓ 定理 — 特性方程式と斉次解

斉次方程式 $a y'' + b y' + c y = 0$ に $y=e^{\lambda x}$ を代入すると 特性方程式

$$a\lambda^2 + b\lambda + c = 0$$

を得る。その根 $\lambda_1,\lambda_2$ に応じて：

判別式根斉次解 $y_h$
$b^2-4ac>0$相異なる実根 $\lambda_1,\lambda_2$$C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$
$b^2-4ac=0$重根 $\lambda$$(C_1 + C_2 x)\,e^{\lambda x}$
$b^2-4ac<0$複素根 $\alpha\pm i\beta$$e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$
📝 例 — 減衰振動

$y'' + 2y' + 5y = 0$。特性方程式 $\lambda^2 + 2\lambda + 5 = 0$ の根は $\lambda = -1 \pm 2i$。よって

$$y(x) = e^{-x}\bigl(C_1\cos 2x + C_2\sin 2x\bigr).$$
✓ 定理 — 未定係数法と定数変化法

非斉次項 $g(x)$ が多項式・指数・$\sin/\cos$（およびその積）なら，同型の試行解を仮定して係数を決める 未定係数法 が使える（$g$ が斉次解と重なる場合は $x$ を掛けて修正）。一般の $g$ には 定数変化法：基本解 $y_1,y_2$（Wronskian $W=y_1 y_2'-y_1'y_2$）から

$$y_p = -y_1\!\int\!\frac{y_2\,g/a}{W}\,dx + y_2\!\int\!\frac{y_1\,g/a}{W}\,dx.$$
📝 例 — 強制振動と共鳴

無減衰の強制振動 $y'' + \omega_0^2 y = F\cos\omega x$。$\omega \neq \omega_0$ のとき特殊解は $y_p = \dfrac{F}{\omega_0^2-\omega^2}\cos\omega x$。

外力の振動数が固有振動数に一致する $\omega=\omega_0$（共鳴）では，試行解に $x$ を掛けて

$$y_p = \frac{F}{2\omega_0}\,x\sin\omega_0 x$$

となり，振幅が $x$ に比例して増大する。

💡 重ね合わせの原理

斉次線形 ODE では，解の線形結合もまた解になる。非斉次でも右辺が $g_1+g_2$ なら，それぞれの特殊解の和が全体の特殊解になる。

4. 連立線形系と行列指数

📖 定義 — 1階線形連立系

ベクトル $\mathbf{x}(t)\in\mathbb{R}^n$ と定数行列 $A$ に対し

$$\dot{\mathbf{x}} = A\,\mathbf{x}$$

を定数係数の 1階線形連立系 という。高階の単独 ODE も補助変数を導入してこの形に書き直せる。

✓ 定理 — 行列指数による解

初期値問題 $\dot{\mathbf{x}}=A\mathbf{x},\ \mathbf{x}(0)=\mathbf{x}_0$ の解は

$$\mathbf{x}(t) = e^{At}\,\mathbf{x}_0,\qquad e^{At} = \sum_{k=0}^{\infty}\frac{(At)^k}{k!}.$$

$\dfrac{d}{dt}e^{At} = A\,e^{At}$ かつ $e^{A\cdot 0}=I$ ゆえ確かに解。

✓ 定理 — 固有値による解の構成

$A$ が対角化可能で固有値 $\lambda_k$，固有ベクトル $\mathbf{v}_k$ をもつなら，基本解は $e^{\lambda_k t}\,\mathbf{v}_k$ であり一般解は

$$\mathbf{x}(t) = \sum_{k=1}^{n} c_k\, e^{\lambda_k t}\,\mathbf{v}_k.$$

複素固有値 $\alpha\pm i\beta$ は実部・虚部をとって $e^{\alpha t}\cos\beta t,\ e^{\alpha t}\sin\beta t$ の実解にする。重複固有値で固有ベクトルが不足する場合は一般化固有ベクトル（$x e^{\lambda t}$ 型）を補う。

📝 例 — 2次元系の対角化

$A = \begin{pmatrix} 1 & 1 \\ 4 & 1 \end{pmatrix}$。固有値は $\det(A-\lambda I)=(1-\lambda)^2-4=0$ より $\lambda = 3,\,-1$。対応する固有ベクトルは $\mathbf{v}_1=\binom{1}{2},\ \mathbf{v}_2=\binom{1}{-2}$。よって

$$\mathbf{x}(t) = c_1 e^{3t}\binom{1}{2} + c_2 e^{-t}\binom{1}{-2}.$$
💡 相平面と安定性の概観

固有値の符号で原点まわりの相平面の様相が決まる：両方負なら安定ノード，両方正なら不安定ノード，異符号なら鞍点（サドル），複素で実部負なら安定渦状点。詳しくは 力学系 を参照。

5. 級数解法と特殊関数

📖 定義 — 正則点と確定特異点

$y'' + P(x)\,y' + Q(x)\,y = 0$ について，$x_0$ で $P,Q$ がともに解析的なら $x_0$ は 正則点（通常点）。そうでなくても $(x-x_0)P$ と $(x-x_0)^2 Q$ がともに解析的なら $x_0$ は 確定特異点 という。

✓ 定理 — べき級数解（正則点）

$x_0$ が正則点なら，解は

$$y(x) = \sum_{n=0}^{\infty} a_n (x-x_0)^n$$

と展開でき，方程式に代入して係数の 漸化式 を得る。収束半径は最も近い特異点までの距離以上。

✓ 定理 — Frobenius 法（確定特異点）

確定特異点 $x_0$ のまわりでは，指数 $r$ を含む

$$y(x) = (x-x_0)^{r}\sum_{n=0}^{\infty} a_n (x-x_0)^n,\quad a_0\neq 0$$

を仮定する。最低次の項から 指数方程式（indicial equation） が出て $r$ が定まる。2つの指数の差が整数のときは第2解に対数項が現れることがある。

📝 例 — Bessel・Legendre 方程式

Bessel 方程式（$x=0$ が確定特異点，円筒対称な問題に頻出）：

$$x^2 y'' + x y' + (x^2 - \nu^2)\,y = 0,\quad \text{解は } J_\nu(x),\ Y_\nu(x).$$

Legendre 方程式（球対称な問題に頻出，$\ell$ が非負整数のとき多項式解）：

$$(1-x^2)\,y'' - 2x\,y' + \ell(\ell+1)\,y = 0,\quad \text{解は } P_\ell(x).$$
💡 物理での登場場所

これらの特殊関数は，偏微分方程式 を球座標・円筒座標で変数分離したときに必ず現れる。級数解法は「閉じた初等関数で書けない解」を扱う標準手段である。

6. Laplace変換

📖 定義 — Laplace 変換

$t\geq 0$ で定義された関数 $f(t)$ に対し

$$\mathcal{L}\{f\}(s) = F(s) = \int_0^{\infty} e^{-st} f(t)\,dt$$

を Laplace 変換 という（積分が収束する $s$ について定義）。

✓ 定理 — 線形性と微分の変換

線形性 $\mathcal{L}\{af+bg\}=aF+bG$ に加え，微分は

$$\mathcal{L}\{f'\} = sF(s) - f(0),\qquad \mathcal{L}\{f''\} = s^2 F(s) - s f(0) - f'(0).$$

初期値が変換式に自然に組み込まれる点が鍵。

📝 例 — 基本的な変換表
$f(t)$$F(s)$
$1$$1/s$
$t^n$$n!/s^{n+1}$
$e^{at}$$1/(s-a)$
$\sin\omega t$$\omega/(s^2+\omega^2)$
$\cos\omega t$$s/(s^2+\omega^2)$
📝 例 — 初期値問題への応用

$y'' + y = 0,\ y(0)=1,\ y'(0)=0$ を変換する。$s^2 Y - s + Y = 0$ より

$$Y(s) = \frac{s}{s^2+1}\ \xrightarrow{\ \mathcal{L}^{-1}\ }\ y(t) = \cos t.$$

微分方程式が代数方程式に化け，逆変換（部分分数分解＋変換表）で解が得られる。

✓ 定理 — たたみ込み定理

たたみ込み $(f*g)(t) = \displaystyle\int_0^t f(\tau)\,g(t-\tau)\,d\tau$ に対し

$$\mathcal{L}\{f*g\} = F(s)\,G(s).$$

積の逆変換がたたみ込みになるため，外力 $g(t)$ に対する応答を系のインパルス応答との畳み込みで表せる。

7. 存在と一意性

📖 定義 — Lipschitz 条件

$f(x,y)$ が領域内で $y$ について Lipschitz 連続 とは，ある定数 $L>0$ が存在して

$$|f(x,y_1) - f(x,y_2)| \le L\,|y_1 - y_2|$$

が成り立つこと。$\partial f/\partial y$ が有界なら十分条件として成立する。

✓ 定理 — Picard–Lindelöf（存在と一意性）

$f(x,y)$ が $(x_0,y_0)$ の近傍で連続，かつ $y$ について Lipschitz 連続ならば，初期値問題

$$y' = f(x,y),\quad y(x_0)=y_0$$

は $x_0$ のある区間で 解が一意に存在する。

📖 定義 — Picard の逐次近似

IVP を積分方程式 $y(x) = y_0 + \displaystyle\int_{x_0}^{x} f(t,y(t))\,dt$ に書き換え，

$$y_0(x)=y_0,\qquad y_{k+1}(x) = y_0 + \int_{x_0}^{x} f\bigl(t, y_k(t)\bigr)\,dt$$

と反復する。Lipschitz 条件のもとでこの列は縮小写像の不動点（＝真の解）に一様収束する。

📝 例 — 一意性が崩れる場合

$y' = \sqrt{|y|},\ y(0)=0$ では $f=\sqrt{|y|}$ が $y=0$ で Lipschitz でない。実際 $y\equiv 0$ と $y=\tfrac{1}{4}x^2\,(x\ge 0)$ の両方が解となり一意性が破れる。

💡 解の延長

局所解は，解が領域の境界に近づくか発散するまで一意に延長できる。たとえば $y'=y^2,\ y(0)=1$ の解 $y=1/(1-x)$ は $x\to 1^-$ で有限時間爆発し，区間 $(-\infty,1)$ までしか延長できない。

💡 次のステップ

独立変数が複数になると偏微分方程式 偏微分方程式 へ，解の長時間挙動・安定性・分岐を調べると 力学系 へとつながる。本ページの線形系・存在一意性はそのどちらの土台にもなる。

✏️ 練習問題
  1. 変数分離形 $y' = x\,(1+y^2)$ を初期条件 $y(0)=0$ のもとで解け。
  2. 1階線形 ODE $x y' + y = \cos x\ (x>0)$ を積分因子を用いて解け（ヒント：$\mu=x$）。
  3. 定数係数 ODE $y'' - 4y' + 4y = e^{2x}$ の一般解を，未定係数法（重根との重なりに注意）で求めよ。
  4. 連立系 $\dot{\mathbf{x}} = \begin{pmatrix} 0 & 1 \\ -2 & -3 \end{pmatrix}\mathbf{x}$ の一般解を固有値・固有ベクトルから構成し，相平面での型（ノード／サドル等）を答えよ。
  5. Laplace 変換を用いて初期値問題 $y'' + 3y' + 2y = 0,\ y(0)=1,\ y'(0)=0$ を解け。
