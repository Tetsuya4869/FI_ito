1. 凸集合と凸関数

📖 定義 — 凸集合・凸関数

$C \subset \mathbb{R}^n$ が凸集合: $x,y \in C,\; \lambda \in [0,1] \implies \lambda x + (1-\lambda)y \in C$。

$f: C \to \mathbb{R}$（$C$ 凸）が凸関数:

$$f(\lambda x + (1-\lambda)y) \le \lambda f(x) + (1-\lambda)f(y) \quad \forall x,y \in C,\; \lambda \in [0,1]$$

$\ge$ は凹関数。$f$ が $C^2$ ならば，凸 $\iff$ Hessian $\nabla^2 f \succeq 0$（半正定値）。

✓ 凸関数の基本性質
  • 局所最小値 = 大域最小値
  • 最小値集合は凸集合
  • 劣レベル集合 $\{x \mid f(x) \le c\}$ は凸
  • $\nabla f(x^*) = 0$（一階条件）が最小値の必要十分条件（凸の場合）
📝 凸関数の例
  • $f(x) = \|x\|_p$（$p \ge 1$ の $L^p$ ノルム）
  • $f(x) = e^{ax}$，$f(x) = x^2$，$f(x) = -\ln x$（$x>0$）
  • $f(X) = -\ln\det X$（正定値行列の空間上；行列値凸関数）
  • $f(x) = \max_i x_i$，$f(x) = \|Ax-b\|^2$

2. 無制約最適化

✓ 定理 — 一次・二次の最適性条件

$f: \mathbb{R}^n \to \mathbb{R}$ が $C^2$。$x^*$ が局所最小値ならば：

  • 一次必要条件: $\nabla f(x^*) = 0$
  • 二次必要条件: $\nabla^2 f(x^*) \succeq 0$（半正定値）

十分条件: $\nabla f(x^*)=0$ かつ $\nabla^2 f(x^*) \succ 0$（正定値） $\implies$ 厳密局所最小値。

📝 最急降下法・Newton 法

最急降下法: $x_{k+1} = x_k - \alpha_k \nabla f(x_k)$（$\alpha_k > 0$: ステップサイズ）。収束率: $\mu$-強凸・$L$-滑らかで $O((1-\mu/L)^k)$。

Newton 法: $x_{k+1} = x_k - [\nabla^2 f(x_k)]^{-1}\nabla f(x_k)$。局所 2 次収束（速い）。

3. Lagrange 乗数法

✓ 定理 — Lagrange の乗数定理

等式制約最適化: $\min f(x)$ s.t. $h_i(x) = 0$（$i = 1,\ldots,m$）。$x^*$ が正則な局所最小値（制約の勾配が線形独立）ならば，

$$\nabla f(x^*) + \sum_{i=1}^m \lambda_i \nabla h_i(x^*) = 0$$

を満たす $\lambda = (\lambda_1,\ldots,\lambda_m) \in \mathbb{R}^m$（Lagrange 乗数）が存在する。

Lagrangian: $\mathcal{L}(x,\lambda) = f(x) + \sum_i \lambda_i h_i(x)$ の停留点。

📝 例 — 球面上の最大・最小

$\min f(x) = x_1 + 2x_2$ s.t. $x_1^2 + x_2^2 = 1$。

$\nabla f = \lambda \nabla h$: $(1, 2) = \lambda (2x_1, 2x_2)$ → $x_1 = 1/(2\lambda)$，$x_2 = 1/\lambda$。制約に代入: $\lambda = \pm\sqrt{5}/2$。最小値 $-\sqrt{5}$，最大値 $\sqrt{5}$。

4. KKT 条件

✓ 定理 — KKT 条件（Karush-Kuhn-Tucker）

$\min f(x)$ s.t. $g_j(x) \le 0$（$j=1,\ldots,p$），$h_i(x)=0$（$i=1,\ldots,m$）。正則性（LICQ など）のもとで，局所最小値 $x^*$ は：

  1. 停留性: $\nabla f(x^*) + \sum_j \mu_j \nabla g_j(x^*) + \sum_i \lambda_i \nabla h_i(x^*) = 0$
  2. 主実行可能性: $g_j(x^*) \le 0$，$h_i(x^*) = 0$
  3. 双対実行可能性: $\mu_j \ge 0$
  4. 相補スラック条件: $\mu_j g_j(x^*) = 0$（$\forall j$）

$f, g_j$ が凸，$h_i$ が線形のとき，KKT条件は最適性の必要十分条件。

5. 双対性理論

📖 定義 — Lagrange 双対問題

主問題 $p^* = \inf_x \sup_{\lambda \ge 0, \mu} \mathcal{L}(x,\lambda,\mu)$ に対して，双対問題:

$$d^* = \sup_{\lambda \ge 0, \mu} \inf_x \mathcal{L}(x,\lambda,\mu) = \sup_{\lambda \ge 0,\mu} q(\lambda,\mu)$$

$q(\lambda,\mu) = \inf_x \mathcal{L}(x,\lambda,\mu)$ を双対関数（常に凹）という。

✓ 定理 — 弱双対性・強双対性

弱双対性: $d^* \le p^*$（常に成立）。

強双対性: Slater 条件（$\exists \hat x,\; g_j(\hat x) < 0$ かつ $h_i(\hat x) = 0$）のもとで $d^* = p^*$。

📝 線形計画の双対

主問題: $\min c^T x$ s.t. $Ax = b$，$x \ge 0$。
双対問題: $\max b^T y$ s.t. $A^T y \le c$。強双対性により最適値は一致（$c^Tx^* = b^Ty^*$）。

📖 定義 — Fenchel 共役

$f: \mathbb{R}^n \to (-\infty,+\infty]$ のFenchel 共役（凸共役）:

$$f^*(y) = \sup_{x} (\langle y, x\rangle - f(x))$$

$f^{**} = f$（$f$ が閉凸関数のとき）。Young の不等式: $\langle x,y\rangle \le f(x) + f^*(y)$。

6. 変分法

📖 定義 — 汎関数と変分

汎関数 $J[y] = \int_a^b F(x, y, y')\,dx$ の最小化。$y^* + \varepsilon\eta$（$\eta(a)=\eta(b)=0$）を代入し $\varepsilon=0$ で微分して 0 とおく（第一変分 $= 0$）。

✓ 定理 — Euler-Lagrange 方程式

$J[y] = \int_a^b F(x,y,y')\,dx$ が $y^*$ で極値をとるならば，

$$\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'} = 0$$
📝 重要な例
  • 最短路: $F = \sqrt{1+y'^2}$ → $y'' = 0$（直線）
  • 最速降下線（Brachistochrone）: サイクロイドが解
  • 古典力学: $F = \frac{1}{2}m\dot q^2 - V(q)$（ラグランジアン）→ Newton の運動方程式 $m\ddot q = -\nabla V$
  • 測地線: Riemann 多様体上で $J[c] = \int \sqrt{g_{ij}\dot c^i \dot c^j}\,dt$ の Euler-Lagrange 方程式が測地線方程式

7. 劣微分と近接点法

📖 定義 — 劣微分

凸関数 $f$ の点 $x$ での劣微分:

$$\partial f(x) = \{g \in \mathbb{R}^n \mid f(y) \ge f(x) + \langle g, y-x\rangle \;\; \forall y\}$$

$f$ が $x$ で微分可能なら $\partial f(x) = \{\nabla f(x)\}$。$f(x) = |x|$ では $\partial f(0) = [-1,1]$。

✓ 最適性条件
$$x^* \text{ が最小値} \iff 0 \in \partial f(x^*)$$
📖 定義 — 近接点作用素
$$\mathrm{prox}_{\alpha f}(x) = \arg\min_y \left(f(y) + \frac{1}{2\alpha}\|y-x\|^2\right)$$

LASSO など疎性促進正則化の最適化に使われる近接勾配法（ISTA/FISTA）の中核。

✏️ 練習問題
  1. $f(x) = \|x\|_1 = \sum |x_i|$ の劣微分 $\partial f(x)$ を求めよ（各成分別に考えよ）。
  2. Lagrange 乗数法で，$\sum x_i = 1$，$x_i \ge 0$ のもとで $-\sum x_i \ln x_i$（エントロピー）を最大化せよ。
  3. $F(x,y,y') = y'^2$ の Euler-Lagrange 方程式を解き，境界条件 $y(0)=0$，$y(1)=1$ を満たす解を求めよ。
  4. $\min \|x\|_1$ s.t. $Ax=b$ の双対問題を導け。
