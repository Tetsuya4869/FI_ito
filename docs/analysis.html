1. 実数の完備性

📖 定義 — 上界・下界・上限・下限

$S \subset \mathbb{R}$ が空でないとする。

  • $\forall x \in S,\; x \le M$ を満たす $M$ を 上界 (upper bound) という。
  • 上界の集合が下に有界なとき，その最小値を 上限 $\sup S$ という。
  • 下界・下限 $\inf S$ も同様に定義する。
✓ 定理 — 実数の完備性公理 (Dedekind)

上に有界な空でない実数の部分集合は必ず上限をもつ。

$$\emptyset \ne S \subset \mathbb{R},\;\; S \text{ は上に有界} \implies \sup S \in \mathbb{R}$$
💡 注意

有理数 $\mathbb{Q}$ は完備でない。例: $S = \{x \in \mathbb{Q} \mid x^2 < 2\}$ の上限 $\sqrt{2} \notin \mathbb{Q}$。

2. 数列と極限

📖 定義 — 数列の収束 (ε-N論法)

数列 $\{a_n\}$ が $\alpha$ に収束するとは，

$$\forall \varepsilon > 0,\;\; \exists N \in \mathbb{N},\;\; n > N \implies |a_n - \alpha| < \varepsilon$$

が成り立つことをいい，$\lim_{n\to\infty} a_n = \alpha$ と書く。

📖 定義 — Cauchy列

数列 $\{a_n\}$ が Cauchy列 であるとは，

$$\forall \varepsilon > 0,\;\; \exists N,\;\; m,n > N \implies |a_m - a_n| < \varepsilon$$
✓ 定理 — Cauchy の収束判定

実数列が収束する $\iff$ Cauchy列である。

✓ 定理 — Bolzano-Weierstrass

有界な実数列は収束する部分列をもつ。

3. 連続関数

📖 定義 — 連続性 (ε-δ論法)

$f: D \to \mathbb{R}$，$a \in D$。$f$ が $a$ で連続であるとは，

$$\forall \varepsilon > 0,\;\; \exists \delta > 0,\;\; |x - a| < \delta \implies |f(x) - f(a)| < \varepsilon$$
✓ 定理 — 中間値定理 (IVT)

$f: [a,b] \to \mathbb{R}$ が連続で $f(a) \ne f(b)$ ならば，$f(a)$ と $f(b)$ の間の任意の値 $c$ に対して，$f(\xi) = c$ となる $\xi \in (a,b)$ が存在する。

✓ 定理 — 最大値・最小値定理

閉区間 $[a,b]$ 上の連続関数は最大値と最小値をとる。

📖 定義 — 一様連続性

$f$ が $D$ 上で一様連続であるとは，

$$\forall \varepsilon > 0,\;\; \exists \delta > 0,\;\; \forall x,y \in D,\;\; |x-y| < \delta \implies |f(x)-f(y)| < \varepsilon$$

（$\delta$ が $x$ の選び方によらない点が連続性との違い。）

4. 微分法

📖 定義 — 導関数
$$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

この極限が存在するとき，$f$ は $a$ で微分可能という。

✓ 定理 — Rolle の定理

$f: [a,b] \to \mathbb{R}$ が連続，$(a,b)$ で微分可能，$f(a)=f(b)$ ならば，$f'(\xi)=0$ となる $\xi \in (a,b)$ が存在する。

✓ 定理 — 平均値定理 (MVT)

$f: [a,b] \to \mathbb{R}$ が連続，$(a,b)$ で微分可能ならば，

$$f(b) - f(a) = f'(\xi)(b-a)$$

を満たす $\xi \in (a,b)$ が存在する。

証明（平均値定理）

$g(x) = f(x) - \dfrac{f(b)-f(a)}{b-a}(x-a)$ と定義する。$g(a)=g(b)=f(a)$ なので，Rolleの定理より $g'(\xi)=0$ となる $\xi \in (a,b)$ が存在する。$g'(\xi)=f'(\xi)-\dfrac{f(b)-f(a)}{b-a}=0$ を解くと主張が得られる。

✓ 定理 — L'Hôpital の定理

$\lim_{x\to a} f(x) = \lim_{x\to a} g(x) = 0$（または $\pm\infty$）で $g'(x)\ne 0$，$\lim_{x\to a}\frac{f'(x)}{g'(x)} = L$ が存在するとき，

$$\lim_{x\to a} \frac{f(x)}{g(x)} = L$$

5. 積分法

📖 定義 — Riemann積分

区間 $[a,b]$ の分割 $\Delta: a=x_0 < x_1 < \cdots < x_n = b$，代表点 $\xi_i \in [x_{i-1},x_i]$ に対して，

$$S(\Delta,\xi) = \sum_{i=1}^n f(\xi_i)(x_i - x_{i-1})$$

$|\Delta| = \max_i(x_i - x_{i-1}) \to 0$ のとき $S(\Delta,\xi)$ が一定値 $I$ に収束するとき，$f$ は Riemann 積分可能といい，$I = \int_a^b f(x)\,dx$ と書く。

✓ 定理 — 微分積分学の基本定理 (FTC)

$f: [a,b] \to \mathbb{R}$ が連続のとき，$F(x) = \int_a^x f(t)\,dt$ とおけば，

$$F'(x) = f(x)$$

また $F$ が $f$ の原始関数（$F'=f$）ならば，

$$\int_a^b f(x)\,dx = F(b) - F(a)$$
📝 例 — Gauss積分
$$\int_{-\infty}^{\infty} e^{-x^2}\,dx = \sqrt{\pi}$$

証明のアイデア: $I^2 = \int_{-\infty}^\infty\int_{-\infty}^\infty e^{-(x^2+y^2)}dx\,dy$ を極座標変換 $x=r\cos\theta,\,y=r\sin\theta$ で計算する。

6. 級数とテイラー展開

✓ 定理 — テイラーの定理

$f$ が $[a,a+h]$ 上で $n+1$ 回微分可能ならば，

$$f(a+h) = \sum_{k=0}^n \frac{f^{(k)}(a)}{k!} h^k + R_n$$

剰余項（Lagrange 形）: $R_n = \dfrac{f^{(n+1)}(a+\theta h)}{(n+1)!}h^{n+1}$，$\theta \in (0,1)$

📝 主要なマクローリン展開
$$e^x = \sum_{n=0}^\infty \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \cdots \quad (|x|<\infty)$$ $$\sin x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{6} + \cdots$$ $$\cos x = \sum_{n=0}^\infty \frac{(-1)^n x^{2n}}{(2n)!} = 1 - \frac{x^2}{2} + \cdots$$ $$\ln(1+x) = \sum_{n=1}^\infty \frac{(-1)^{n-1}x^n}{n} = x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots \quad (|x|\le 1,\,x\ne -1)$$
✓ 主な収束判定法
  • 比率判定: $L = \lim_{n\to\infty}|a_{n+1}/a_n|$。$L<1$ なら絶対収束，$L>1$ なら発散。
  • 根判定: $L = \limsup_{n\to\infty}|a_n|^{1/n}$。$L<1$ なら絶対収束，$L>1$ なら発散。
  • Dirichlet判定: $\{a_n\}$ が単調減少で $0$ に収束し $\{b_n\}$ の部分和が有界 $\implies \sum a_n b_n$ 収束。
✏️ 練習問題
  1. $\varepsilon$-$\delta$ 論法を用いて $\lim_{x\to 2}(3x-1)=5$ を証明せよ。
  2. $f(x)=x^3$ が $\mathbb{R}$ 上で一様連続でないことを示せ。
  3. $\displaystyle\int_0^1 x^n \ln x\,dx$ を計算せよ。
  4. $e = \sum_{n=0}^\infty 1/n!$ が無理数であることを証明せよ。（ヒント: $e$ が有理数と仮定して矛盾を導く。）
