💡 前提知識

結び目理論は 位相空間論 の言葉（同相・ホモトピー・連続写像）を基礎とし，不変量の多くは 代数的位相幾何（基本群・ホモロジー）を用いて構成される。これらに目を通しておくと理解が深まる。

1. 結び目と絡み目

📖 定義 — 結び目

結び目とは円周 $S^1$ から3次元空間 $\mathbb{R}^3$（または $S^3$）への滑らかな埋め込み $K: S^1 \hookrightarrow \mathbb{R}^3$ の像である。2つの結び目 $K_1, K_2$ は，一方を切らずに連続変形（アンビエント・イソトピー）で他方に移せるとき同値とみなす。

複数の円周を同時に埋め込んだものを絡み目（link）といい，各成分を成分（component）と呼ぶ。結び目は成分が1つの絡み目である。

📖 定義 — 結び目図式

結び目を平面 $\mathbb{R}^2$ に射影し，各交差点でどちらのひもが上を通るか（上下情報）を切れ目で示した図を結び目図式（diagram）という。一般の位置への射影では，交差は二重点のみで，三重点や接点は現れない。

図式は結び目を平面上で扱う基本的な道具であり，不変量の多くは図式を通じて定義・計算される。

📝 例 — 基本的な結び目

自明な結び目（unknot）$\bigcirc$: 交差のない単純閉曲線。ほどけてただの円になる結び目。

三葉結び目（trefoil）$3_1$: 交差数3の最も単純な非自明結び目。右手型と左手型があり，互いに鏡像だが同値ではない（キラル）。

八の字結び目（figure-eight）$4_1$: 交差数4。自身の鏡像と同値（アンキラル）。

📝 例 — Hopf 絡み目

2つの円が一度だけ絡んだ最も単純な非自明な絡み目を Hopf 絡み目という。2成分・交差数2で，2つの成分は分離できない。絡み数（後述）が $\pm 1$ である。

2. ライデマイスター移動

✓ 定理 — ライデマイスターの定理

同じ結び目（または絡み目）の任意の2つの図式は，平面イソトピーと次の3種類の局所変形の有限列で互いに移り合う。

  • R1（ねじれ）: ひもにできた小さなループ（カール）を加える／除く。
  • R2（重なり）: 2本のひもの重なりをずらして2交差を加える／除く。
  • R3（三角形）: 1本のひもを他の交差点をまたいで滑らせる。
💡 図式の同値性

2つの図式が同じ結び目を表すための必要十分条件は，R1・R2・R3 の有限列で移り合うことである。これにより，3次元での連続変形という扱いにくい問題が，平面図式上の組合せ的操作に翻訳される。

💡 不変量を作る指針

図式に量 $Q$ を割り当て，R1・R2・R3 のすべてで $Q$ が変わらないことを示せば，$Q$ は結び目の不変量になる。多項式不変量や三色彩色可能性は，まさにこの方針で構成される。R1 のみで変化する量（例: 巻きつき数 writhe）は正則イソトピー不変量であり，フレーム付き結び目の不変量として意味をもつ。

3. 結び目不変量

📖 定義 — 三色彩色可能性

図式の各弧（交差で途切れずに続く部分）に3色 $\{0,1,2\}$ を割り当てる。各交差で，3本の弧（上の弧1本と下の弧2本）がすべて同色かすべて異色となり，かつ2色以上が使われている彩色が存在するとき，その結び目は三色彩色可能であるという。

三色彩色可能性はライデマイスター移動で保たれる不変量である。三葉結び目は三色彩色可能だが，自明な結び目は不可能なので，両者は異なると示せる。

📖 定義 — 交差数

結び目 $K$ の交差数 $c(K)$ は，$K$ のすべての図式における交差点の個数の最小値である。$c(\text{自明}) = 0$，$c(3_1) = 3$，$c(4_1) = 4$。交差数は結び目を分類・命名する基本指標だが，与えられた図式から最小性を判定するのは一般に難しい。

📖 定義 — 絡み数

2成分の有向絡み目 $L = K_1 \cup K_2$ に対し，各交差に符号 $\varepsilon = \pm 1$ を右手系で割り当て，$K_1$ と $K_2$ の間の交差について和をとった半分を絡み数という。

$$\mathrm{lk}(K_1, K_2) = \frac{1}{2}\sum_{p \in K_1 \cap K_2} \varepsilon(p)$$

絡み数は整数値の不変量で，2成分がどれだけ絡んでいるかを測る。Hopf 絡み目では $\mathrm{lk} = \pm 1$，分離可能な2成分では $0$ である。

📝 例 — 交差符号

有向図式の交差で，下のひもを上のひもへ反時計回りに回して向きが一致するなら $\varepsilon = +1$（正交差），時計回りなら $\varepsilon = -1$（負交差）。Gauss の積分公式 $\mathrm{lk}(K_1,K_2) = \frac{1}{4\pi}\oint\!\oint \frac{(\mathbf{r}_1-\mathbf{r}_2)\cdot(d\mathbf{r}_1 \times d\mathbf{r}_2)}{|\mathbf{r}_1-\mathbf{r}_2|^3}$ は絡み数を線積分で与える。

4. 多項式不変量

📖 定義 — Jones 多項式とスケイン関係式

有向絡み目 $L$ に変数 $t$ の Laurent 多項式 $V_L(t)$ を割り当てる Jones 多項式は，次のスケイン関係式で帰納的に定まる。

$$t^{-1} V_{L_+} - t\, V_{L_-} + (t^{-1/2} - t^{1/2})\, V_{L_0} = 0$$

ここで $L_+, L_-, L_0$ は1つの交差だけが異なる（正交差・負交差・交差解消）図式である。規格化は $V_{\bigcirc} = 1$。

✓ 定理 — Jones 多項式は不変量

$V_L(t)$ はライデマイスター移動 R1・R2・R3 で不変であり，有向絡み目の同値類の不変量である。三葉結び目では右手型と左手型で異なる値をとり，これらが互いに異なることを示せる（キラリティの検出）。

📖 定義 — Alexander 多項式

Alexander 多項式 $\Delta_L(t)$ は，Jones 多項式より古典的な不変量で，同じくスケイン関係式

$$\Delta_{L_+}(t) - \Delta_{L_-}(t) = (t^{1/2} - t^{-1/2})\, \Delta_{L_0}(t)$$

を満たす。結び目群（後述）の Alexander 行列の行列式や，無限巡回被覆のホモロジーからも定義され，代数的位相幾何と直結する。ただし鏡像を区別できない弱点がある。

✓ HOMFLY 多項式

Jones と Alexander を共通に含む2変数の多項式不変量が HOMFLY 多項式 $P_L(a, z)$ である。スケイン関係式は

$$a\, P_{L_+} - a^{-1} P_{L_-} = z\, P_{L_0}, \qquad P_{\bigcirc} = 1.$$

適当な変数代入で Jones（$a = t^{-1},\, z = t^{1/2} - t^{-1/2}$）と Alexander（$a = 1$）が得られる。さらに一般化として Kauffman 多項式がある。

5. 結び目群

📖 定義 — 結び目群

結び目 $K \subset S^3$ に対し，補空間 $S^3 \setminus K$ の基本群 $\pi_1(S^3 \setminus K)$ を $K$ の結び目群という。これはアンビエント・イソトピー不変量であり，補空間の位相情報を群として捉える強力な不変量である。

例: 自明な結び目の結び目群は $\mathbb{Z}$。三葉結び目の結び目群は $\langle a, b \mid aba = bab \rangle$（3次ブレイド群と同型）で，非可換である。

✓ 定理 — Wirtinger 表示

結び目図式から結び目群の表示が機械的に得られる。各弧に生成元 $x_i$（その弧を回るメリディアン）を割り当て，各交差で関係子を作る。上の弧を $x_k$，下の弧の入口・出口を $x_i, x_j$ とすると，正交差では

$$x_j = x_k\, x_i\, x_k^{-1}$$

という形の関係が得られる（負交差では $x_j = x_k^{-1} x_i x_k$）。これらを集めた群表示を Wirtinger 表示という。$n$ 交差の図式から生成元 $n$ 個・関係子 $n$ 個（うち1個は冗長）が得られる。

💡 補空間が結び目を決める

Gordon–Luecke の定理により，$S^3$ 内の結び目はその補空間 $S^3 \setminus K$ の同相型で（向きを込めて）完全に決定される。したがって補空間の位相不変量は原理的に結び目の完全不変量を与えるが，結び目群だけでは（鏡像など）一部の区別ができない場合もある。

6. Seifert 曲面と種数

📖 定義 — Seifert 曲面

有向結び目 $K$ に対し，$K$ を境界にもつ $\mathbb{R}^3$ 内の連結・向き付け可能なコンパクト曲面 $\Sigma$（$\partial\Sigma = K$）を Seifert 曲面という。任意の有向結び目は Seifert 曲面をもつ。

図式から Seifert 曲面を構成する手続きが Seifert のアルゴリズムである: 各交差を向きに沿って解消して得られる Seifert 円に円板を貼り，元の交差にねじれたバンドを付ける。

📖 定義 — 結び目の種数

結び目 $K$ の 種数 $g(K)$ は，$K$ の Seifert 曲面の種数の最小値である。$g(\text{自明}) = 0$ であり，逆に $g(K) = 0$ なら $K$ は自明な結び目である。種数は加法的で，連結和に対し $g(K_1 \# K_2) = g(K_1) + g(K_2)$ が成り立つ。

📝 例 — 三葉結び目の種数

三葉結び目 $3_1$ の Seifert 曲面はオイラー標数 $\chi = -1$ の曲面（1つの境界をもつトーラスの一部）となり，種数 $g(3_1) = 1$ である。これは自明な結び目（$g=0$）と区別する別の証明を与える。八の字結び目も $g(4_1)=1$。

💡 有向性の必要性

Seifert 曲面は向き付け可能曲面なので，結び目に向きを与えて初めて Seifert のアルゴリズムが適用できる。曲面の種数 $g$ と境界数 $b=1$，オイラー標数 $\chi$ の間には $\chi = 1 - 2g$ の関係があり，種数の計算は $\chi$ の計算に帰着される。Seifert 行列からは Alexander 多項式や符号数（signature）といった不変量も得られる。

7. 応用

📝 例 — DNA とタンパク質の結び目

環状 DNA は複製や組換えの過程で結び目や絡み目を作る。トポイソメラーゼはストランドを切断・再結合してこれを解く酵素であり，その作用を絡み数・ねじれ・巻きつき（$Lk = Tw + Wr$）で定量化する。タンパク質の主鎖にも結び目構造が見つかり，折りたたみや安定性との関連が研究されている。

📝 例 — Chern–Simons 理論

Witten は3次元の Chern–Simons ゲージ理論の経路積分から Wilson ループの期待値として Jones 多項式が得られることを示した。これは結び目不変量を 場の量子論 と量子群の枠組みで再解釈するもので，トポロジカル場の理論（TQFT）の出発点となった。

💡 統計力学との関係

Jones 多項式は2次元統計力学の Yang–Baxter 方程式を満たす $R$ 行列や，可解格子模型（Potts 模型・頂点模型）の分配関数と密接に結びつく。Temperley–Lieb 代数やブレイド群の表現が，これらを横断する共通の代数的構造を与える。

💡 次のステップ

結び目不変量の代数的・ホモロジー的な構成（Khovanov ホモロジーなど）へ進むには 代数的位相幾何 が，Chern–Simons 理論や TQFT を通じた物理的解釈へ進むには 量子場理論 が次の道標となる。

✏️ 練習問題
  1. 三葉結び目の標準図式を三色彩色し，2色以上を使う彩色が存在することを確かめよ。また自明な結び目には非自明な三色彩色が存在しないことを示せ。
  2. ライデマイスター移動 R1・R2・R3 のそれぞれについて，絡み数 $\mathrm{lk}(K_1,K_2)$ が不変であることを交差符号の変化から確かめよ。
  3. スケイン関係式と $V_{\bigcirc}=1$ を用いて，Hopf 絡み目の Jones 多項式を計算せよ。
  4. 三葉結び目の標準図式から Wirtinger 表示を書き下し，得られた表示が $\langle a,b \mid aba = bab\rangle$ と同型であることを確かめよ。
  5. 三葉結び目に Seifert のアルゴリズムを適用して Seifert 曲面を構成し，オイラー標数から種数が $1$ であることを示せ。
