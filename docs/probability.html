1. 確率空間

📖 定義 — 確率空間 (Kolmogorov, 1933)

三つ組 $(\Omega, \mathcal{F}, P)$ を確率空間という：

  • $\Omega$: 標本空間（全試行結果の集合）
  • $\mathcal{F}$: $\Omega$ 上のσ加法族（事象の族）
  • $P: \mathcal{F} \to [0,1]$: 確率測度（$P(\Omega)=1$）
✓ 確率の基本性質
  • $P(A^c) = 1 - P(A)$
  • $P(A \cup B) = P(A) + P(B) - P(A \cap B)$（加法公式）
  • $A_n \nearrow A \implies P(A_n) \nearrow P(A)$（確率の連続性）
  • Boole の不等式: $P\!\left(\bigcup_n A_n\right) \le \sum_n P(A_n)$

2. 確率変数と分布

📖 定義 — 確率変数

可測写像 $X: (\Omega, \mathcal{F}) \to (\mathbb{R}, \mathcal{B}(\mathbb{R}))$ を確率変数という。

分布関数: $F_X(x) = P(X \le x)$。単調増加，右連続，$F(-\infty)=0$，$F(\infty)=1$。

📖 定義 — 独立性

事象 $A_1, \ldots, A_n$ が独立とは，任意の部分集合 $\{i_1,\ldots,i_k\}$ に対して

$$P(A_{i_1} \cap \cdots \cap A_{i_k}) = P(A_{i_1})\cdots P(A_{i_k})$$

確率変数 $X_1,\ldots,X_n$ が独立とは，生成するσ加法族が独立であること。

3. 期待値・分散・モーメント

📖 定義
$$E[X] = \int_\Omega X\,dP = \int_{-\infty}^\infty x\,dF_X(x)$$ $$\mathrm{Var}(X) = E[(X-E[X])^2] = E[X^2] - (E[X])^2$$ $$\mathrm{Cov}(X,Y) = E[(X-EX)(Y-EY)]$$
✓ Jensen の不等式

$\varphi$ が凸関数で $E[|\varphi(X)|] < \infty$ ならば，$\varphi(E[X]) \le E[\varphi(X)]$。

例: $E[X]^2 \le E[X^2]$，$e^{E[X]} \le E[e^X]$

✓ Chebyshev の不等式
$$P(|X - \mu| \ge \varepsilon) \le \frac{\sigma^2}{\varepsilon^2} \quad (\mu = E[X],\; \sigma^2 = \mathrm{Var}(X))$$

5. 特性関数

📖 定義 — 特性関数
$$\varphi_X(t) = E[e^{itX}] = \int_{-\infty}^\infty e^{itx}\,dF_X(x), \quad t \in \mathbb{R}$$
✓ 主な性質
  • $|\varphi_X(t)| \le 1$，$\varphi_X(0) = 1$
  • $X$ と $Y$ が独立 $\implies$ $\varphi_{X+Y}(t) = \varphi_X(t)\varphi_Y(t)$
  • 一対一対応: $\varphi_X = \varphi_Y \iff F_X = F_Y$（分布を一意に定める）
  • モーメント: $E[X^n] = \dfrac{\varphi_X^{(n)}(0)}{i^n}$（$n$ 次モーメントが存在する場合）

6. 大数の法則

✓ 定理 — 強大数の法則 (SLLN)

i.i.d.（独立同分布）確率変数列 $X_1, X_2, \ldots$ で $E[|X_1|] < \infty$ ならば，

$$\frac{X_1 + \cdots + X_n}{n} \xrightarrow{a.s.} E[X_1] \quad (n \to \infty)$$

「概率1で」（almost surely）収束する。

💡 弱大数の法則 vs. 強大数の法則

弱大数の法則は「確率収束」，強大数は「概率1での収束」（より強い命題）。前者から後者は一般には導けない。

7. 中心極限定理 (CLT)

✓ 定理 — 中心極限定理 (Lindeberg-Lévy)

i.i.d. 確率変数列 $\{X_n\}$，$E[X_1] = \mu$，$\mathrm{Var}(X_1) = \sigma^2 < \infty$ とするとき，

$$\frac{\sum_{k=1}^n X_k - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} \mathcal{N}(0,1) \quad (n \to \infty)$$

（$\xrightarrow{d}$ は分布収束）

証明のアイデア（特性関数を使う）

$Z_n = \dfrac{\sum X_k - n\mu}{\sigma\sqrt{n}}$ の特性関数を計算する。$Y_k = (X_k-\mu)/\sigma$ とおくと，

$$\varphi_{Z_n}(t) = \varphi_Y(t/\sqrt{n})^n = \left(1 - \frac{t^2}{2n} + O(n^{-3/2})\right)^n \to e^{-t^2/2}$$

$e^{-t^2/2}$ は $\mathcal{N}(0,1)$ の特性関数。Lévy の連続性定理により分布収束が従う。

8. 条件付き確率と条件付き期待値

📖 定義 — 条件付き確率

$P(B) > 0$ のとき，$P(A \mid B) = \dfrac{P(A \cap B)}{P(B)}$。

Bayes の定理: $P(A_i \mid B) = \dfrac{P(B \mid A_i)P(A_i)}{\sum_j P(B \mid A_j)P(A_j)}$

📖 定義 — 条件付き期待値 $E[X \mid \mathcal{G}]$

σ加法族 $\mathcal{G} \subset \mathcal{F}$ に対して，$E[X \mid \mathcal{G}]$ は次を満たす $\mathcal{G}$-可測関数：

$$\int_G E[X \mid \mathcal{G}]\,dP = \int_G X\,dP \quad \forall G \in \mathcal{G}$$

（Radon-Nikodym 定理で存在と一意性が保証される。）

✓ 条件付き期待値の性質
  • $E[E[X \mid \mathcal{G}]] = E[X]$（tower property）
  • $\mathcal{H} \subset \mathcal{G}$ ならば $E[E[X \mid \mathcal{G}] \mid \mathcal{H}] = E[X \mid \mathcal{H}]$（smooth property）
  • $X$ が $\mathcal{G}$-可測 $\implies$ $E[XY \mid \mathcal{G}] = X \cdot E[Y \mid \mathcal{G}]$（taking out known）
  • $X$ が $\mathcal{G}$ と独立 $\implies$ $E[X \mid \mathcal{G}] = E[X]$
✏️ 練習問題
  1. コインを $n$ 回投げるとき，表の回数 $S_n$ が $\mathcal{N}(n/2, n/4)$ に近似されることを CLT で示せ。
  2. $X \sim \mathrm{Exp}(\lambda)$ の無記憶性 $P(X > s+t \mid X > s) = P(X > t)$ を証明せよ。
  3. $X,Y$ が独立標準正規のとき $E[X \mid X+Y=c]$ を計算せよ。
  4. Markov の不等式 $P(X \ge a) \le E[X]/a$（$X \ge 0$，$a > 0$）を証明し，これから Chebyshev の不等式を導け。
