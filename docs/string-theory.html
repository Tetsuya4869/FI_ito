💡 前提知識

弦理論は量子論・相対論・場の理論の総合の上に成り立つ。あらかじめ 量子場理論，相対性理論，共形場理論 を学んでおくと理解が深まる。

1. 弦の作用

📖 定義 — 点粒子から弦へ

相対論的な点粒子の作用は世界線 $x^\mu(\tau)$ の固有長に比例する:

$$S_{\text{particle}} = -mc\int ds = -mc\int\sqrt{-\eta_{\mu\nu}\dot x^\mu\dot x^\nu}\,d\tau$$

弦理論では基本対象を1次元的に拡がった弦とし，その運動は時空内に2次元の面（世界面 world-sheet）を描く。弦は世界面の座標 $(\tau,\sigma)$ から時空 $X^\mu(\tau,\sigma)$ への埋め込みとして記述される。

📖 定義 — Nambu-Goto 作用

作用を世界面の面積に比例させる:

$$S_{\text{NG}} = -\frac{1}{2\pi\alpha'}\int d\tau\,d\sigma\,\sqrt{-\det h_{ab}}, \quad h_{ab} = \partial_a X^\mu\,\partial_b X_\mu$$

$h_{ab}$ は世界面上に誘導される計量，$\alpha'$（Regge スロープ）は弦の張力 $T = 1/(2\pi\alpha')$ を定め，弦長 $\ell_s = \sqrt{\alpha'}$ に対応する。

✓ Polyakov 作用

平方根を避け量子化しやすくするため，独立な世界面計量 $\gamma_{ab}$ を補助場として導入する:

$$S_{\text{P}} = -\frac{1}{4\pi\alpha'}\int d\tau\,d\sigma\,\sqrt{-\gamma}\,\gamma^{ab}\,\partial_a X^\mu\,\partial_b X_\nu\,\eta_{\mu\nu}$$

$\gamma_{ab}$ の運動方程式（エネルギー・運動量テンソル $T_{ab}=0$）を代入すると Nambu-Goto 作用に一致する。古典レベルで両者は等価。

💡 世界面の対称性

Polyakov 作用は世界面の再パラメータ化不変性（微分同相）に加え，Weyl 変換 $\gamma_{ab}\to e^{2\omega}\gamma_{ab}$ に対する不変性をもつ。これらの局所対称性を用いて計量を平坦な形 $\gamma_{ab}=\eta_{ab}$（共形ゲージ）に固定でき，理論は世界面上の共形場理論に帰着する。

2. 量子化とスペクトル

📖 定義 — モード展開

共形ゲージでは $X^\mu$ は2次元の波動方程式 $(\partial_\tau^2-\partial_\sigma^2)X^\mu=0$ を満たし，振動モードに展開される。閉弦（周期境界条件）では右向き・左向きの独立なモードに分かれる:

$$X^\mu(\tau,\sigma) = x^\mu + \alpha' p^\mu \tau + i\sqrt{\frac{\alpha'}{2}}\sum_{n\ne 0}\frac{1}{n}\left(\alpha_n^\mu e^{-in(\tau-\sigma)} + \tilde\alpha_n^\mu e^{-in(\tau+\sigma)}\right)$$
📖 定義 — 開弦と閉弦

開弦は端点をもち，Neumann または Dirichlet 境界条件を課す。Dirichlet 条件は端点が固定される超曲面 D-ブレーンの存在を意味する。閉弦は端点のない輪であり，重力を含むセクターを与える。振動子 $\alpha_n^\mu$ は正準交換関係 $[\alpha_m^\mu,\alpha_n^\nu]=m\,\delta_{m+n,0}\,\eta^{\mu\nu}$ を満たす生成・消滅演算子となる。

✓ 質量スペクトル

Virasoro 条件（$L_0$ 制約）から質量の2乗が数演算子で与えられる。開弦（ボゾン弦）では:

$$\alpha' M^2 = N - a, \quad N = \sum_{n=1}^{\infty}\alpha_{-n}\!\cdot\!\alpha_n, \quad a = 1$$

閉弦ではレベル整合条件 $N=\tilde N$ のもとで $\alpha' M^2 = 4(N-1)$。定数 $a$ は零点エネルギーの正則化から生じる。

📝 例 — タキオンと重力子

ボゾン弦の最低状態 $N=0$ は $\alpha'M^2=-a<0$，すなわち質量2乗が負のタキオンとなり真空の不安定性を示す（後に超対称性で除去される）。

閉弦の第一励起状態 $N=\tilde N=1$，$\alpha_{-1}^\mu\tilde\alpha_{-1}^\nu|0;p\rangle$ は $M^2=0$ の無質量状態で，対称・トレースレス部分がスピン2の重力子，反対称部分が反対称テンソル場 $B_{\mu\nu}$，トレースがディラトン $\phi$ に対応する。弦理論が重力を自然に含むことを示す決定的事実である。

3. 臨界次元

✓ 共形不変性の要請

Polyakov 作用の Weyl 対称性は古典的には成り立つが，量子化すると一般に量子異常（Weyl アノマリー）として破れる。世界面の共形場理論の中心電荷を $c$ とすると，時空次元 $D$ 個のボゾン場 $X^\mu$ は $c=D$ を寄与し，ゲージ固定に伴う $bc$ ゴースト系が $c_{\text{gh}}=-26$ を寄与する。

✓ ボゾン弦の 26 次元

Weyl アノマリーが相殺し理論が無矛盾（ユニタリでゴーストなし）となるには全中心電荷が消えねばならない:

$$c_{\text{total}} = D - 26 = 0 \;\Longrightarrow\; D = 26$$

同時に零点定数が $a=1$ に固定される。すなわちボゾン弦は 26 次元時空でのみ無矛盾に定義される。この時空次元が理論自身によって決定される点が弦理論の際立った特徴である。

💡 光円錐量子化との整合

共変量子化のほか，光円錐ゲージでの量子化からも同じ結論が得られる。ローレンツ対称性 $SO(1,D-1)$ が量子論で保たれる条件が $D=26$，$a=1$ を要求し，複数の量子化法が一致する。

4. 超対称性と超弦理論

📖 定義 — 世界面の超対称性

ボゾン弦はタキオンを含みフェルミオンを記述できない。世界面にフェルミオン場 $\psi^\mu(\tau,\sigma)$ を加え，作用に世界面超対称性をもたせる（RNS 形式）:

$$S = -\frac{1}{4\pi\alpha'}\int d^2\sigma\left(\partial_a X^\mu\partial^a X_\mu + i\bar\psi^\mu\rho^a\partial_a\psi_\mu\right)$$

GSO 射影によりタキオンが除去され，時空にフェルミオンとボゾンの超対称性が実現される。

✓ 10 次元

フェルミオン場の追加により中心電荷の勘定が変わり，アノマリー相殺の条件は

$$D = 10$$

となる。超弦理論は 10 次元時空でのみ無矛盾に定義される。

📝 例 — 5 つの超弦理論

10 次元で無矛盾な超弦理論は次の5種類に分類される:

  1. Type I（開弦と閉弦，ゲージ群 $SO(32)$，$\mathcal N=1$ 超対称性）
  2. Type IIA（閉弦，非カイラル，$\mathcal N=2$）
  3. Type IIB（閉弦，カイラル，$\mathcal N=2$）
  4. ヘテロ型 $SO(32)$（左右で異なる弦，ゲージ群 $SO(32)$）
  5. ヘテロ型 $E_8\times E_8$（ゲージ群 $E_8\times E_8$）

これら5つは後に単一の理論（M理論）の異なる極限として理解される。

5. コンパクト化

📖 定義 — 余剰次元

超弦理論は 10 次元を要求するが，観測される時空は4次元である。この不整合は，余分な6次元が非常に小さくコンパクトに丸め込まれている（コンパクト化）と考えて解消する。時空を $M_4\times K_6$ と分解し，$K_6$ を微小な内部空間とする。

✓ Calabi-Yau 多様体

4次元に $\mathcal N=1$ 超対称性を残す（現実的な低エネルギー理論を得る）には，内部6次元空間がリッチ平坦でホロノミー群 $SU(3)$ をもつ複素3次元 Kähler 多様体，すなわち Calabi-Yau 多様体でなければならない:

$$R_{i\bar\jmath} = 0, \quad c_1(K_6) = 0$$

Calabi-Yau のトポロジー（Hodge 数）が4次元の粒子世代数やゲージ群を決める。

📝 例 — Kaluza-Klein 分解

コンパクト化された次元の場は，内部空間のサイズ $R$ で量子化された運動量（KK モード）をもつ。円 $S^1$ 上の場を Fourier 展開すると

$$\phi(x,y) = \sum_{n}\phi_n(x)\,e^{iny/R}, \quad M_n^2 = \frac{n^2}{R^2}$$

となり，4次元の観測者には質量 $|n|/R$ をもつ無限個の粒子（KK タワー）として見える。$R$ が小さいほど励起は重く低エネルギーで見えない。これが高次元場を4次元の粒子スペクトルに翻訳する Kaluza-Klein 機構である。

6. 双対性とM理論

📖 定義 — T 双対と S 双対

T 双対: 半径 $R$ の円にコンパクト化した弦理論は，半径 $\alpha'/R$ の理論と物理的に等価である。運動量モード $n$ と巻き付きモード $w$ が入れ替わる:

$$R \leftrightarrow \frac{\alpha'}{R}, \quad n \leftrightarrow w$$

S 双対: 弦結合定数 $g_s$ の理論と $1/g_s$ の理論が等価となる強弱結合双対性。摂動的に扱えない強結合領域を弱結合の別理論で記述できる。

✓ 11 次元 M 理論

これらの双対性により，5つの超弦理論と11次元超重力理論はすべて，唯一の基礎理論 M 理論の異なる極限であることが示された。例えば Type IIA 理論を強結合にすると11番目の次元（半径 $\sim g_s$）が現れ，11次元の理論に至る:

$$D = 11$$

M 理論の基本自由度は弦ではなく2次元膜（M2 ブレーン）と5次元膜（M5 ブレーン）と考えられている。

📝 例 — AdS/CFT 対応

Maldacena の予想により，$(d+1)$ 次元の反ド・ジッター時空 $\text{AdS}_{d+1}$ 上の重力理論（弦理論）が，その境界上の $d$ 次元共形場理論と等価である:

$$\text{Type IIB on }\text{AdS}_5\times S^5 \;\Longleftrightarrow\; \mathcal N=4\ \text{SU}(N)\ \text{超対称ゲージ理論}$$

これは重力と場の理論を結ぶホログラフィー原理の具体例で，強結合ゲージ理論を弱結合重力で解析する強力な手法を与える。

7. 意義と課題

✓ 量子重力の候補

弦理論は重力子を自然に含み，点粒子理論で生じる紫外発散が弦の拡がりによって和らげられる。無矛盾に量子化された重力を含む数少ない枠組みであり，一般相対論と量子論を統一する最有力候補とされる。

💡 統一理論としての魅力

単一の弦の異なる振動状態がすべての粒子（ゲージボゾン，フェルミオン，重力子）に対応するという描像は，物質と相互作用を統一する究極理論への希望を与える。ゲージ対称性や物質内容がコンパクト化の幾何から導かれる点も特徴的である。

💡 ランドスケープ問題と検証可能性

コンパクト化の選び方（Calabi-Yau の種類やフラックス）は膨大で，$10^{500}$ にも及ぶ多数の準安定真空（ストリング・ランドスケープ）が存在しうる。このため4次元物理を一意に予言することが難しい。またプランクスケールに直接到達できないため，超対称性粒子・余剰次元・宇宙論的痕跡などの間接的検証が現在の課題となっている。

✏️ 練習問題
  1. Polyakov 作用について $\gamma_{ab}$ の変分をとり，エネルギー・運動量テンソル $T_{ab}=0$ の条件を導け。さらにそれを代入して Nambu-Goto 作用が再現されることを示せ。
  2. 閉弦のモード展開において，レベル整合条件 $N=\tilde N$ が世界面の並進不変性（$\sigma$ の原点の任意性）から生じることを説明せよ。
  3. ボゾン弦の零点エネルギーを $\sum_{n=1}^\infty n \to \zeta(-1) = -1/12$ を用いて正則化し，$D=26$ のとき $a=1$ となることを確かめよ。
  4. 半径 $R$ の円にコンパクト化した閉弦の質量公式を運動量モード $n$ と巻き付きモード $w$ で書き，$R\leftrightarrow\alpha'/R$，$n\leftrightarrow w$ の下で不変（T 双対）であることを示せ。
  5. 円 $S^1$ 上のスカラー場を Kaluza-Klein 展開し，4次元での質量スペクトル $M_n^2=n^2/R^2$ を導け。$R\to 0$ の極限で余剰次元が観測にかからない理由を述べよ。
