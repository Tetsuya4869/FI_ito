💡 前提知識

場合の数（順列・組合せ）と 微分積分の基礎（連続分布の積分）。この先は 確率論 へ。

1. 確率の基本

📖 定義 — 確率

すべての結果が同様に確からしいとき、事象 $A$ の確率は

$$P(A) = \frac{A \text{が起こる場合の数}}{\text{全体の場合の数}}, \quad 0\le P(A)\le 1$$

性質: $P(\text{全体})=1$、$P(A^c)=1-P(A)$、互いに排反なら $P(A\cup B)=P(A)+P(B)$。

一般には $P(A\cup B)=P(A)+P(B)-P(A\cap B)$（加法定理）。

📝 例 — サイコロ

1個のサイコロで偶数が出る確率は $P=3/6=1/2$。

2. 条件付き確率とベイズ

📖 条件付き確率と独立

$B$ が起きたもとでの $A$ の確率: $P(A\mid B)=\dfrac{P(A\cap B)}{P(B)}$。

$A,B$ が独立とは $P(A\cap B)=P(A)P(B)$（互いに影響しない）。

✓ ベイズの定理
$$P(A\mid B) = \frac{P(B\mid A)\,P(A)}{P(B)}$$

「原因 $A$ → 結果 $B$」の確率から「結果 $B$ を見て原因 $A$ を推定」する。検査・機械学習の基礎。

3. 確率変数

📖 離散と連続

結果に数を対応させたものが確率変数 $X$。

離散: 確率質量関数 $P(X=k)$（例: サイコロの目）。

連続: 確率密度関数 $f(x)$ で $P(a\le X\le b)=\displaystyle\int_a^b f(x)\,dx$、全体で $\int_{-\infty}^\infty f(x)\,dx=1$。

4. 期待値と分散

📖 期待値（平均）
$$E[X] = \sum_k k\,P(X=k) \quad\text{(離散)}, \qquad E[X]=\int_{-\infty}^\infty x f(x)\,dx \quad\text{(連続)}$$

線形性: $E[aX+bY]=aE[X]+bE[Y]$（独立でなくても成立）。

📖 分散と標準偏差
$$V[X] = E[(X-E[X])^2] = E[X^2] - (E[X])^2, \quad \sigma = \sqrt{V[X]}$$

分散はばらつきの大きさ。独立なら $V[X+Y]=V[X]+V[Y]$。

6. 大数の法則と中心極限定理

✓ 大数の法則 (LLN)

独立同分布な $X_1,\ldots,X_n$ の標本平均は、$n$ を大きくすると真の平均に近づく:

$$\bar X_n = \frac{1}{n}\sum_{i=1}^n X_i \to E[X] \quad (n\to\infty)$$
✓ 中心極限定理 (CLT)

独立同分布の和は、分布の形によらず正規分布に近づく:

$$\frac{\bar X_n - \mu}{\sigma/\sqrt{n}} \xrightarrow{\;d\;} N(0,1)$$

「世の中に正規分布が多い」理由。統計的推定の根拠でもある。

7. 次のステップ

💡 ここから先へ

確率を厳密な測度として定式化するなら 測度論 → 確率論 へ。時間とともに変化するランダム現象は 確率微分方程式（伊藤の公式）、多数の粒子の統計は 統計力学 へつながります。
