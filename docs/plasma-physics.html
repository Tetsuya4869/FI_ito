1. プラズマの基本

📖 定義 — プラズマ

プラズマとは，正イオンと電子（および中性粒子）が混在する電離気体で，集団的な振る舞いを示し，全体として電気的に中性（準中性）であるものをいう。固体・液体・気体に次ぐ「第4の物質状態」と呼ばれる。

準中性条件: 巨視的スケールでは正電荷密度と負電荷密度がほぼ等しく，

$$n_e \approx \sum_i Z_i n_i$$

ここで $n_e$ は電子数密度，$n_i$ はイオン数密度，$Z_i$ はイオンの電荷数。

✓ Debye 遮蔽と Debye 長

プラズマ中に挿入された試験電荷の電場は，反対符号の荷電粒子の集まりによって遮蔽され，距離 $\lambda_D$ 程度で指数関数的に減衰する。点電荷まわりのポテンシャルは:

$$\phi(r) = \frac{q}{4\pi\varepsilon_0 r}\,e^{-r/\lambda_D}$$

Debye 長は（電子の寄与を主とすると）:

$$\lambda_D = \sqrt{\frac{\varepsilon_0 k_B T_e}{n_e e^2}}$$

$\lambda_D$ より十分大きいスケールで準中性が成り立つ。

📖 定義 — プラズマ振動数

準中性からのずれが生じると静電復元力が働き，電子は集団的に振動する。その固有角振動数が電子プラズマ振動数:

$$\omega_{pe} = \sqrt{\frac{n_e e^2}{\varepsilon_0 m_e}}$$

これは外部から加えられた電磁波がプラズマ中を伝搬できるか否かを決める指標でもある（$\omega < \omega_{pe}$ の電磁波は反射される）。

📖 定義 — プラズマパラメータ

Debye 球（半径 $\lambda_D$ の球）内に含まれる粒子数:

$$N_D = n_e \cdot \frac{4}{3}\pi\lambda_D^3, \qquad \Lambda \equiv n_e \lambda_D^3 \gg 1$$

$\Lambda \gg 1$（多数の粒子が遮蔽に関与）であることが，集団的・気体的な振る舞いを示す「理想プラズマ」の条件。物質がプラズマであるための3条件は，(i) $\lambda_D \ll L$（系のサイズ），(ii) $N_D \gg 1$，(iii) $\omega_{pe}\tau \gg 1$（$\tau$: 衝突時間）。

📝 例 — 核融合プラズマと電離層

トカマク炉心では $n_e \sim 10^{20}\,\text{m}^{-3}$，$T_e \sim 10\,\text{keV}$ 程度で，$\lambda_D \sim 10^{-5}\,\text{m}$，$\omega_{pe} \sim 10^{12}\,\text{rad/s}$。地球電離層では $n_e \sim 10^{12}\,\text{m}^{-3}$，$T_e \sim 0.1\,\text{eV}$ 程度。いずれも $N_D \gg 1$ を満たす理想プラズマである。

2. 単一粒子運動

📖 定義 — Lorentz 力と運動方程式

電場 $\mathbf{E}$，磁場 $\mathbf{B}$ 中の荷電粒子（電荷 $q$，質量 $m$）の運動方程式:

$$m\frac{d\mathbf{v}}{dt} = q(\mathbf{E} + \mathbf{v}\times\mathbf{B})$$
✓ ジャイロ運動（サイクロトロン運動）

一様磁場 $\mathbf{B}=B\hat z$，$\mathbf{E}=0$ のとき，粒子は磁力線まわりを円運動する。サイクロトロン角振動数とラーモア半径:

$$\omega_c = \frac{|q|B}{m}, \qquad r_L = \frac{mv_\perp}{|q|B} = \frac{v_\perp}{\omega_c}$$

$v_\perp$ は磁場に垂直な速度成分。磁場に平行な運動は等速直線運動となり，全体としてらせん軌道を描く。

✓ 各種ドリフト

外力や場の非一様性があると，ジャイロ運動の中心（案内中心）がゆっくり横方向に移動する。一様な $\mathbf{E}$ がある場合の $\mathbf{E}\times\mathbf{B}$ ドリフト:

$$\mathbf{v}_{E} = \frac{\mathbf{E}\times\mathbf{B}}{B^2}$$

これは電荷の符号・質量によらず共通であることが特徴。一般の力 $\mathbf{F}$ に対しては:

$$\mathbf{v}_F = \frac{\mathbf{F}\times\mathbf{B}}{qB^2}$$

磁場勾配による $\nabla B$ ドリフトと磁力線の曲率による曲率ドリフト:

$$\mathbf{v}_{\nabla B} = \frac{m v_\perp^2}{2qB^3}\,\mathbf{B}\times\nabla B, \qquad \mathbf{v}_R = \frac{m v_\parallel^2}{qB^2}\,\frac{\mathbf{R}_c\times\mathbf{B}}{R_c^2}$$
✓ 断熱不変量（磁気モーメント）

場が粒子のジャイロ周期に比べてゆっくり変化するとき，第一断熱不変量である磁気モーメントが保存する:

$$\mu = \frac{m v_\perp^2}{2B} = \text{一定}$$

$B$ が増す領域へ進むと $v_\perp$ が増え，エネルギー保存より $v_\parallel$ が減る。$v_\parallel=0$ で粒子は跳ね返される（磁気ミラー）。これが磁気閉じ込めの基礎となる。

💡 ドリフトと電流

イオンと電子のドリフト方向が異なる場合（$\nabla B$・曲率ドリフトなど），正味の電流が流れ，電荷分離や不安定性の起源となる。一方 $\mathbf{E}\times\mathbf{B}$ ドリフトは両種で同方向のため電流を生まない。

3. 運動論的記述

📖 定義 — 分布関数

粒子種 $s$ の分布関数 $f_s(\mathbf{r},\mathbf{v},t)$ は，位相空間の体積要素 $d^3r\,d^3v$ 内に存在する粒子数を表す:

$$dN_s = f_s(\mathbf{r},\mathbf{v},t)\,d^3r\,d^3v$$

巨視量はモーメントとして得られる: 数密度 $n_s = \int f_s\,d^3v$，平均速度 $\mathbf{u}_s = \dfrac{1}{n_s}\int \mathbf{v} f_s\,d^3v$。熱平衡では Maxwell 分布。

✓ Vlasov 方程式

衝突が無視できる無衝突プラズマでは，位相空間における分布関数の保存（Liouville の定理）より:

$$\frac{\partial f_s}{\partial t} + \mathbf{v}\cdot\nabla_{\mathbf{r}} f_s + \frac{q_s}{m_s}(\mathbf{E} + \mathbf{v}\times\mathbf{B})\cdot\nabla_{\mathbf{v}} f_s = 0$$

ここで $\mathbf{E},\mathbf{B}$ は外部場と全粒子が作る自己無撞着な場（平均場）の和であり，Maxwell 方程式と連立する。

✓ 衝突項と Boltzmann 方程式

衝突を考慮すると右辺に衝突項が加わり，Boltzmann 方程式となる:

$$\frac{\partial f_s}{\partial t} + \mathbf{v}\cdot\nabla_{\mathbf{r}} f_s + \frac{q_s}{m_s}(\mathbf{E} + \mathbf{v}\times\mathbf{B})\cdot\nabla_{\mathbf{v}} f_s = \left(\frac{\partial f_s}{\partial t}\right)_{\text{coll}}$$

プラズマでは Coulomb 衝突は小角散乱の多数回累積が支配的であり，衝突項として Fokker–Planck 形式（Landau 衝突項）が用いられる。Vlasov 方程式は衝突項を $0$ とした極限に対応する。

💡 記述の階層

運動論（分布関数）は最も詳細な記述で，速度空間の情報（Landau 減衰・不安定性など）を保持する。これを速度モーメントで積分し閉じることで，より粗視化された流体（MHD）記述が得られる。粒子描像・運動論・流体の使い分けが重要。

4. 電磁流体力学（MHD）

📖 定義 — 電磁流体力学（MHD）

プラズマを単一の導電性流体として扱う巨視的記述。各種のモーメント方程式を一流体に統合し，Maxwell 方程式と結合する。低周波・長波長（$\omega \ll \omega_c$，スケール $\gg r_L$）の現象に有効。

✓ MHD 方程式

質量密度 $\rho$，流速 $\mathbf{u}$，圧力 $p$，電流密度 $\mathbf{J}$ について:

$$\frac{\partial\rho}{\partial t} + \nabla\cdot(\rho\mathbf{u}) = 0 \quad\text{（連続の方程式）}$$ $$\rho\frac{d\mathbf{u}}{dt} = \mathbf{J}\times\mathbf{B} - \nabla p \quad\text{（運動方程式）}$$ $$\mathbf{E} + \mathbf{u}\times\mathbf{B} = \eta\mathbf{J} \quad\text{（一般化 Ohm の法則）}$$

これに Maxwell 方程式 $\nabla\times\mathbf{B} = \mu_0\mathbf{J}$，$\nabla\times\mathbf{E} = -\partial\mathbf{B}/\partial t$ を連立する。$\eta$ は電気抵抗率。

✓ 理想 MHD と磁場の凍結

抵抗を無視できる理想 MHD（$\eta\to 0$）では Ohm の法則は $\mathbf{E} = -\mathbf{u}\times\mathbf{B}$ となり，磁場の発展は:

$$\frac{\partial\mathbf{B}}{\partial t} = \nabla\times(\mathbf{u}\times\mathbf{B})$$

このとき任意の流体面を貫く磁束が保存され（Alfvén の凍結定理），磁力線は流体とともに運動する。一般には磁場拡散項 $\dfrac{\eta}{\mu_0}\nabla^2\mathbf{B}$ が加わり，両者の比が磁気 Reynolds 数 $R_m$ を与える。

📝 例 — 磁気圧と磁気張力

Lorentz 力は $\mathbf{J}\times\mathbf{B} = -\nabla\left(\dfrac{B^2}{2\mu_0}\right) + \dfrac{1}{\mu_0}(\mathbf{B}\cdot\nabla)\mathbf{B}$ と分解でき，第1項は等方的な磁気圧 $B^2/(2\mu_0)$，第2項は磁力線に沿う張力を表す。プラズマ圧と磁気圧の比 $\beta = \dfrac{p}{B^2/(2\mu_0)}$ は閉じ込め性能の重要指標である。

5. 波動と不安定性

✓ プラズマ波（Langmuir 波・Alfvén 波）

静電的な電子振動（Langmuir 波，電子プラズマ波）の分散関係（Bohm–Gross）:

$$\omega^2 = \omega_{pe}^2 + 3k^2 v_{th,e}^2$$

磁化プラズマでは磁力線の張力を復元力とする Alfvén 波が伝わる。Alfvén 速度と分散関係:

$$v_A = \frac{B}{\sqrt{\mu_0\rho}}, \qquad \omega = k_\parallel v_A$$
✓ Landau 減衰

無衝突プラズマでも，波の位相速度近傍の速度をもつ粒子と波とのエネルギー交換により波が減衰する純運動論的効果。減衰率は分布関数の速度勾配で決まる:

$$\gamma \propto \left.\frac{\partial f_0}{\partial v}\right|_{v=\omega/k}$$

Maxwell 分布では $\partial f_0/\partial v < 0$（位相速度より遅い粒子が多い）のため波はエネルギーを失い減衰する。逆に勾配が正なら波が成長する（速度空間不安定性）。

📖 定義 — 各種不安定性

プラズマには多数の不安定性が存在する。代表例:

  • レイリー・テイラー型（交換不安定性）: 重い流体が軽い流体に支えられる配置に類似し，圧力勾配と磁場曲率が逆向きのとき発生。
  • キンク不安定性・ソーセージ不安定性: 電流柱の折れ曲がり・くびれによるマクロな MHD 不安定性。
  • 2 流体（ストリーミング）不安定性: 相対速度をもつビームとプラズマの間で波が成長する運動論的不安定性。
  • 抵抗性不安定性（テアリングモード）: 有限抵抗により磁力線がつなぎ替わり磁気島を形成する。
💡 不安定性と閉じ込め

不安定性は閉じ込めを破壊し輸送を増大させる主因であり，核融合研究の中心課題。一方，磁気リコネクションのように天体現象（太陽フレアなど）のエネルギー解放を担う重要過程でもある。

6. 平衡と閉じ込め

✓ 圧力平衡（MHD 平衡）

定常状態（$\partial/\partial t = 0$，$\mathbf{u}=0$）では運動方程式より力のつり合い:

$$\nabla p = \mathbf{J}\times\mathbf{B}$$

これより $\mathbf{B}\cdot\nabla p = 0$，$\mathbf{J}\cdot\nabla p = 0$ が成り立ち，磁力線と電流は等圧面（磁気面）上にある。軸対称トロイダル平衡は Grad–Shafranov 方程式で記述される。

📖 定義 — 磁気閉じ込め

荷電粒子が磁力線を横切りにくい性質を利用し，磁場でプラズマを保持する方式。磁力線を閉じたトーラス状にし端からの損失を防ぐ。トロイダル磁場だけでは粒子がドリフトで逃げるため，ポロイダル磁場を加えて磁力線をらせん状にねじる（回転変換）。

📝 例 — トカマクとピンチ

トカマク: 外部コイルによる強いトロイダル磁場と，プラズマ電流が作るポロイダル磁場を組み合わせる代表的方式（ITER など）。ピンチ: プラズマ電流自身が作る磁場でプラズマを圧縮・閉じ込める方式。$\mathbf{J}\times\mathbf{B}$ による収縮を利用するが（Z ピンチ・$\theta$ ピンチ），キンク・ソーセージ不安定性に弱い。

💡 慣性閉じ込めとの対比

磁気閉じ込めに対し，強力なレーザーやイオンビームで燃料ペレットを急速に圧縮・加熱し，慣性で飛散する前に核融合を起こす慣性閉じ込め方式もある。閉じ込め時間と密度の積（Lawson 条件）が点火の鍵。

7. 応用

📖 定義 — 核融合（Lawson 条件）

軽い原子核を融合させエネルギーを得る。代表反応は D–T 反応 $\mathrm{D}+\mathrm{T}\to{}^4\mathrm{He}+n+17.6\,\text{MeV}$。自己点火に必要な条件は，密度 $n$・閉じ込め時間 $\tau_E$・温度 $T$ の三重積で表される:

$$n\,T\,\tau_E \gtrsim 3\times 10^{21}\ \text{keV·s·m}^{-3}$$
📝 例 — 宇宙・天体プラズマ

宇宙の可視物質の大半はプラズマ状態にある。恒星内部・星間物質・降着円盤・パルサー磁気圏など，磁場とプラズマの相互作用（MHD・リコネクション・ダイナモ）が天体物理の根幹を成す。銀河磁場の生成や宇宙線加速もプラズマ過程による。

📝 例 — 太陽風と宇宙天気

太陽コロナから噴き出す高温プラズマの流れが太陽風で，惑星間空間に磁場を運ぶ。地球磁気圏との相互作用が磁気嵐・オーロラを生む。コロナ質量放出（CME）は衛星・送電網に影響し，宇宙天気予報の対象となる。

📝 例 — プラズマ加工

低温プラズマは半導体製造で広く利用される。プラズマエッチング（反応性イオンエッチング）や薄膜堆積（プラズマ CVD・スパッタリング），表面改質などに不可欠。蛍光灯・プラズマディスプレイ・プラズマトーチも応用例である。

✏️ 練習問題
  1. $n_e = 10^{19}\,\text{m}^{-3}$，$T_e = 1\,\text{keV}$ のプラズマについて Debye 長 $\lambda_D$，電子プラズマ振動数 $\omega_{pe}$，プラズマパラメータ $N_D$ を概算し，理想プラズマの3条件を確認せよ。
  2. 一様磁場 $\mathbf{B}=B\hat z$ と一様電場 $\mathbf{E}=E\hat x$ 中の荷電粒子の運動方程式を解き，$\mathbf{E}\times\mathbf{B}$ ドリフトが電荷の符号によらないことを示せ。
  3. 磁気モーメント $\mu = m v_\perp^2/(2B)$ の保存とエネルギー保存を用い，磁気ミラーの反射条件（ロスコーン）を $B$ の比で表せ。
  4. 理想 MHD の誘導方程式 $\partial\mathbf{B}/\partial t = \nabla\times(\mathbf{u}\times\mathbf{B})$ から，任意の流体ループを貫く磁束 $\Phi=\int\mathbf{B}\cdot d\mathbf{S}$ が保存される（凍結定理）ことを示せ。
  5. Maxwell 分布 $f_0(v)\propto e^{-v^2/v_{th}^2}$ について $\partial f_0/\partial v < 0$（$v>0$）を確かめ，Langmuir 波が Landau 減衰する理由を物理的に説明せよ。
💡 次のステップ

プラズマを流体として扱う MHD の基礎は 流体力学 にある。また天体・宇宙スケールのプラズマと磁場の役割は 宇宙論 へと続く。
