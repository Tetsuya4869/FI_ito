1. 場の量子化への動機

💡 なぜ場の理論が必要か

非相対論的量子力学（Schrödinger 方程式）は粒子数を保存し，単一粒子の波動関数を扱う。しかし特殊相対論ではエネルギーと質量が等価（$E=mc^2$）であり，十分高エネルギーでは粒子の生成・消滅が起こる。粒子数が変化する系を矛盾なく記述するには，粒子の数を可変にする枠組み——すなわち場の量子化が要請される。

量子場理論（QFT）は，相対論的不変性・量子力学・場の概念を統合した理論であり，素粒子物理学の基本言語である。

📖 定義 — 自然単位系

QFT では計算を簡潔にするため自然単位系 $\hbar = c = 1$ を採用する。これにより質量・エネルギー・運動量・（長さや時間の逆数）はすべて同一の次元（エネルギー次元）で測られる。

$$[\text{長さ}] = [\text{時間}] = [\text{質量}]^{-1} = [\text{エネルギー}]^{-1}$$

四元運動量 $p^\mu = (E,\mathbf{p})$，計量 $\eta_{\mu\nu} = \mathrm{diag}(+,-,-,-)$ を用い，相対論的エネルギー関係は $p^\mu p_\mu = E^2 - |\mathbf{p}|^2 = m^2$ となる。

💡 「場」とは何か

場 $\phi(x)$ は時空の各点 $x = (t,\mathbf{x})$ に値をもつ力学変数である。古典場（電磁場など）を量子化すると，場の励起が粒子として現れる。光子は電磁場の量子であり，電子は電子場の量子である。すべての粒子は，対応する場の量子的励起として理解される。

2. Klein-Gordon 場

📖 定義 — Klein-Gordon 方程式

相対論的エネルギー関係 $E^2 = |\mathbf{p}|^2 + m^2$ に対応演算子 $E \to i\partial_t$，$\mathbf{p}\to -i\nabla$ を代入すると，質量 $m$ をもつ相対論的スカラー場 $\phi(x)$ に対する方程式が得られる:

$$(\Box + m^2)\phi = 0, \qquad \Box \equiv \partial^\mu\partial_\mu = \partial_t^2 - \nabla^2$$

$\Box$ は d'Alembert 演算子。$\phi$ は Lorentz スカラー（スピン 0 の粒子に対応）。

📖 定義 — Lagrangian 密度と作用

実スカラー場の Lagrangian 密度は

$$\mathcal{L} = \frac{1}{2}(\partial_\mu\phi)(\partial^\mu\phi) - \frac{1}{2}m^2\phi^2 = \frac{1}{2}(\partial\phi)^2 - \frac{1}{2}m^2\phi^2$$

作用は $S = \int d^4x\,\mathcal{L}$。場の Euler-Lagrange 方程式

$$\partial_\mu\frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi)} - \frac{\partial\mathcal{L}}{\partial\phi} = 0$$

に代入すると，Klein-Gordon 方程式 $(\Box+m^2)\phi=0$ が再現される。

💡 一粒子解釈の破綻

Klein-Gordon 方程式を一粒子波動関数の方程式と見なすと，負エネルギー解 $E = -\sqrt{|\mathbf{p}|^2+m^2}$ や負の確率密度が現れる。これらの困難は，$\phi$ を場の演算子として量子化することで解決され，負エネルギー解は反粒子として正しく解釈される。

3. 正準量子化

📖 定義 — 共役運動量と正準交換関係

場 $\phi$ に共役な運動量密度を $\pi(x) = \partial\mathcal{L}/\partial\dot\phi = \dot\phi$ で定義する。量子力学の正準交換関係 $[\hat x,\hat p]=i$ を場に拡張し，同時刻交換関係を課す:

$$[\hat\phi(\mathbf{x},t),\hat\pi(\mathbf{y},t)] = i\delta^3(\mathbf{x}-\mathbf{y})$$ $$[\hat\phi(\mathbf{x},t),\hat\phi(\mathbf{y},t)] = [\hat\pi(\mathbf{x},t),\hat\pi(\mathbf{y},t)] = 0$$
✓ モード展開と生成・消滅演算子

場をフーリエ（平面波）モードに展開すると，各モードは調和振動子と等価になる:

$$\hat\phi(x) = \int\frac{d^3p}{(2\pi)^3}\frac{1}{\sqrt{2E_\mathbf{p}}}\left(\hat a_\mathbf{p}\,e^{-ip\cdot x} + \hat a_\mathbf{p}^\dagger\,e^{+ip\cdot x}\right)$$

ここで $E_\mathbf{p} = \sqrt{|\mathbf{p}|^2+m^2}$。生成・消滅演算子は

$$[\hat a_\mathbf{p},\hat a_\mathbf{q}^\dagger] = (2\pi)^3\delta^3(\mathbf{p}-\mathbf{q}), \quad [\hat a_\mathbf{p},\hat a_\mathbf{q}] = [\hat a_\mathbf{p}^\dagger,\hat a_\mathbf{q}^\dagger] = 0$$
📖 定義 — 真空と Fock 空間

真空状態 $|0\rangle$ はすべての消滅演算子で消える状態: $\hat a_\mathbf{p}|0\rangle = 0$。$\hat a_\mathbf{p}^\dagger$ を作用させると運動量 $\mathbf{p}$ の一粒子状態 $|\mathbf{p}\rangle = \sqrt{2E_\mathbf{p}}\,\hat a_\mathbf{p}^\dagger|0\rangle$ が得られる。複数の生成演算子を作用させて得られる多粒子状態全体が張る空間をFock 空間という。

Hamiltonian は $\hat H = \int\frac{d^3p}{(2\pi)^3}\,E_\mathbf{p}\,\hat a_\mathbf{p}^\dagger\hat a_\mathbf{p}$（零点エネルギーを正規順序で除去後）。粒子数演算子 $\hat N = \int\frac{d^3p}{(2\pi)^3}\hat a_\mathbf{p}^\dagger\hat a_\mathbf{p}$ は粒子数を数える。

💡 場 = 無限個の調和振動子

各運動量モードが独立な調和振動子に対応するため，量子場は「無限自由度の調和振動子の集まり」と見なせる。これが 量子力学の昇降演算子の手法が QFT で中心的役割を果たす理由である。

4. Dirac 方程式とスピノル

📖 定義 — Dirac 方程式

スピン $1/2$ の粒子（電子など）を相対論的に記述するため，Dirac は時間・空間に関して一階の方程式を求めた:

$$(i\gamma^\mu\partial_\mu - m)\psi = 0$$

$\psi$ は 4 成分のスピノル場，$\gamma^\mu$（$\mu=0,1,2,3$）は $4\times4$ のガンマ行列である。

📖 定義 — ガンマ行列と Clifford 代数

ガンマ行列は反交換関係（Clifford 代数）を満たす:

$$\{\gamma^\mu,\gamma^\nu\} = \gamma^\mu\gamma^\nu + \gamma^\nu\gamma^\mu = 2\eta^{\mu\nu}\mathbb{1}$$

この関係から $(i\gamma^\mu\partial_\mu - m)(i\gamma^\nu\partial_\nu + m) = -(\Box + m^2)$ が従い，Dirac 場の各成分は Klein-Gordon 方程式を満たす。Dirac Lagrangian は $\mathcal{L} = \bar\psi(i\gamma^\mu\partial_\mu - m)\psi$（$\bar\psi = \psi^\dagger\gamma^0$）。

✓ 反粒子の予言

Dirac 方程式の負エネルギー解は，場の量子化において反粒子として解釈される。電子場の量子化では，電子（$b$）と陽電子（$d$）の生成・消滅演算子が現れる:

$$\hat\psi(x) = \int\frac{d^3p}{(2\pi)^3}\frac{1}{\sqrt{2E_\mathbf{p}}}\sum_s\left(\hat b_\mathbf{p}^s u^s(p)e^{-ip\cdot x} + \hat d_\mathbf{p}^{s\dagger}v^s(p)e^{+ip\cdot x}\right)$$

フェルミ統計（Pauli 排他律）を満たすため，これらは反交換関係 $\{\hat b_\mathbf{p}^r,\hat b_\mathbf{q}^{s\dagger}\} = (2\pi)^3\delta^3(\mathbf{p}-\mathbf{q})\delta^{rs}$ に従う。陽電子は 1932 年に実験的に発見された。

💡 スピン統計定理

整数スピンの場（ボソン）は交換関係で，半整数スピンの場（フェルミオン）は反交換関係で量子化される。この対応はスピン統計定理として相対論的場の理論から導かれる。

5. 相互作用と Feynman 図

📖 定義 — 相互作用項

自由場の Lagrangian に相互作用項を加えると粒子間の相互作用が記述される。代表例:

$$\mathcal{L}_{\phi^4} = \frac{1}{2}(\partial\phi)^2 - \frac{1}{2}m^2\phi^2 - \frac{\lambda}{4!}\phi^4$$

$\lambda$ は結合定数。量子電磁力学（QED）では，電子場 $\psi$ と光子場 $A_\mu$ の結合

$$\mathcal{L}_{\mathrm{QED}} = \bar\psi(i\gamma^\mu D_\mu - m)\psi - \frac{1}{4}F_{\mu\nu}F^{\mu\nu}, \quad D_\mu = \partial_\mu + ieA_\mu$$

が相互作用 $-e\bar\psi\gamma^\mu\psi A_\mu$ を与える。

✓ 摂動展開と Feynman 図

結合定数が小さいとき，散乱過程の振幅を結合定数のべき級数で展開する（摂動論）。各項はFeynman 図で図示される:

  • 外線——入射・出射する実粒子
  • 内線（プロパゲーター）——仮想粒子の伝播
  • 頂点（バーテックス）——相互作用，結合定数の因子を与える

各図はFeynman 則に従って数式（振幅への寄与）に翻訳される。例えば QED では各頂点が因子 $-ie\gamma^\mu$ を，スカラー場のプロパゲーターが $\dfrac{i}{p^2-m^2+i\epsilon}$ を与える。

📝 例 — 散乱振幅と断面積

2 粒子散乱 $a+b\to c+d$ の遷移確率は不変振幅 $\mathcal{M}$ で表される。微分断面積は

$$\frac{d\sigma}{d\Omega} = \frac{1}{64\pi^2 s}\frac{|\mathbf{p}_f|}{|\mathbf{p}_i|}|\mathcal{M}|^2$$

（重心系，$s = (p_a+p_b)^2$ は重心エネルギーの 2 乗）。$|\mathcal{M}|^2$ を Feynman 図から計算し，実験で測定される断面積と比較する。QED の予言（例: 電子の異常磁気モーメント $g-2$）は実験と $10$ 桁以上の精度で一致する。

6. くりこみ

💡 発散の出現

Feynman 図にループ（閉じた内線）が含まれると，仮想粒子の運動量について積分 $\int d^4k$ を行う。この積分は高運動量（紫外）領域でしばしば発散する。素朴に計算すると物理量が無限大になってしまう。

📖 定義 — 正則化とくりこみ

正則化: 発散を一時的に有限化する操作（カットオフ $\Lambda$ の導入，次元正則化 $d = 4-\epsilon$ など）。

くりこみ: Lagrangian 中の質量・結合定数・場の規格化を「裸の量」から「物理的（観測される）量」へ再定義し，発散を吸収する手続き。有限個のパラメータの再定義ですべての発散を除去できる理論をくりこみ可能という。QED・標準模型はくりこみ可能である。

✓ 走る結合定数と繰り込み群

くりこみの結果，結合定数はエネルギースケール $\mu$ に依存する「走る結合定数」$g(\mu)$ となる。その変化は繰り込み群方程式（ベータ関数）で支配される:

$$\mu\frac{dg}{d\mu} = \beta(g)$$

QED では高エネルギーで結合が増大する。一方 QCD では $\beta(g)<0$ となり高エネルギーで結合が減少する——これが漸近的自由性であり，クォークが高エネルギーでほぼ自由に振る舞う理由である。

💡 有効場理論の視点

現代的には，くりこみは「低エネルギーの物理は高エネルギーの詳細に鈍感である」という有効場理論の考え方として理解される。発散は理論の欠陥ではなく，未知の高エネルギー物理を低エネルギーパラメータへ繰り込む自然な手続きと見なされる。

7. ゲージ理論と標準模型

📖 定義 — 局所ゲージ対称性

場の Lagrangian が，時空各点で独立な内部対称性変換に対して不変であることを局所ゲージ対称性という。例えば QED は $U(1)$ ゲージ対称性 $\psi\to e^{i\alpha(x)}\psi$ をもつ。この不変性を保つには通常の微分を共変微分 $D_\mu = \partial_\mu + ieA_\mu$ に置き換える必要があり，こうしてゲージ場（光子 $A_\mu$）の存在と相互作用の形が必然的に導かれる。

✓ 標準模型のゲージ群

素粒子の標準模型は，次の非可換（Yang-Mills）ゲージ群に基づく場の理論である:

$$SU(3)_C \times SU(2)_L \times U(1)_Y$$
  • $SU(3)_C$——量子色力学（QCD），グルーオン 8 個，クォークの「色」を扱う強い相互作用
  • $SU(2)_L \times U(1)_Y$——電弱統一理論，$W^\pm,Z,\gamma$ を与える

ゲージ群の構造は Lie 群・Lie 代数の言語で記述される。

📖 定義 — Higgs 機構（概観）

ゲージ不変性はゲージ場の質量項を禁じるため，素朴には $W,Z$ ボソンや粒子は質量をもてない。しかし Higgs 場 $H$ が真空期待値 $\langle H\rangle \ne 0$ をもつ（自発的対称性の破れ）と，ゲージ対称性 $SU(2)_L\times U(1)_Y$ が $U(1)_{\mathrm{EM}}$ へ破れ，$W^\pm,Z$ ボソンとフェルミオンが質量を獲得する。これがHiggs 機構である。Higgs ボソンは 2012 年に LHC で発見された。

💡 次のステップ

ゲージ理論の数学的基盤である Lie 群・Lie 代数については 抽象代数学を参照。物理と数学の各分野の関係を俯瞰するには 学問領域マップが役立つ。

✏️ 練習問題
  1. Klein-Gordon Lagrangian $\mathcal{L} = \frac12(\partial\phi)^2 - \frac12 m^2\phi^2$ に Euler-Lagrange 方程式を適用し，$(\Box+m^2)\phi=0$ を導け。
  2. 正準交換関係 $[\hat\phi(\mathbf{x},t),\hat\pi(\mathbf{y},t)] = i\delta^3(\mathbf{x}-\mathbf{y})$ から，生成・消滅演算子の交換関係 $[\hat a_\mathbf{p},\hat a_\mathbf{q}^\dagger]=(2\pi)^3\delta^3(\mathbf{p}-\mathbf{q})$ を導け。
  3. ガンマ行列の反交換関係 $\{\gamma^\mu,\gamma^\nu\}=2\eta^{\mu\nu}$ を用いて，Dirac 演算子の積 $(i\gamma^\mu\partial_\mu - m)(i\gamma^\nu\partial_\nu + m)$ が $-(\Box+m^2)$ に等しいことを示せ。
  4. $\phi^4$ 理論における最低次（ツリーレベル）の $2\to2$ 散乱振幅 $\mathcal{M}$ を Feynman 則から求めよ。
  5. $U(1)$ ゲージ変換 $\psi\to e^{i\alpha(x)}\psi$ のもとで $\bar\psi(i\gamma^\mu\partial_\mu)\psi$ は不変でないことを確認し，共変微分 $D_\mu=\partial_\mu+ieA_\mu$ と $A_\mu$ の変換則を用いれば不変になることを示せ。
